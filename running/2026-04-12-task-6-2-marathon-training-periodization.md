## 一、马拉松训练周期概述

马拉松训练周期是系统性地规划和实施训练，以在比赛日达到最佳表现的过程。科学合理的训练周期是成功跑完马拉松并取得好成绩的关键。

### 1.1 马拉松训练周期结构

**典型的训练周期**:
- 准备期: 4-8周
- 发展期: 8-12周
- 竞赛期: 4-6周
- 减量期: 1-2周

**总时间**: 通常16-20周

### 1.2 马拉松训练金字塔

**训练要素**:
| 层级 | 内容 | 重要性 |
|-----|------|-------|
| 基础 | 有氧能力、E配速 | 最重要 |
| 中间 | 阈值、间歇 | 重要 |
| 巅峰 | 比赛配速训练 | 最终目标 |
| 支撑 | 力量、柔韧、技术 | 辅助 |

### 1.3 训练原则

**核心原则**:
- 循序渐进: 逐步增加训练量
- 10%规则: 每周跑量增加不超过10%
- 专项化: 训练要与比赛目标相符
- 个性化: 根据个人情况调整

## 二、基础训练期

### 2.1 基础期目标

**生理适应**:
- 建立有氧基础
- 提高脂肪氧化能力
- 强化结缔组织
- 建立跑步技术模式

**时间**: 通常4-8周

### 2.2 基础期训练构成

**训练类型分布**:
| 训练类型 | 比例 | 强度 |
|---------|------|------|
| 轻松跑 | 70-80% | E区 |
| 长跑 | 15-20% | E区 |
| 恢复跑 | 5-10% | R区 |

**周跑量建议**:
| 阶段 | 周跑量 |
|-----|--------|
| 第1-2周 | 基准量×80% |
| 第3-4周 | 基准量×90% |
| 第5-6周 | 基准量×100% |

### 2.3 基础期训练示例

**典型基础期周训练**:
```
周一: 休息
周二: 5-6km E配速
周三: 6-8km 含冲刺练习
周四: 休息或交叉训练
周五: 6-8km E配速
周六: 长跑 12-16km E配速
周日: 恢复跑 4-6km
```

---

## 三、训练发展期

### 3.1 发展期目标

**生理适应**:
- 提高乳酸阈值
- 增强VO2max
- 发展专项耐力
- 提高跑步效率

**时间**: 通常8-12周

### 3.2 发展期训练构成

**训练类型分布**:
| 训练类型 | 比例 | 强度 |
|---------|------|------|
| 轻松跑 | 50-60% | E区 |
| 阈值跑 | 15-20% | T区 |
| 间歇跑 | 10-15% | I区 |
| 长跑 | 15-20% | E-M区 |

### 3.3 关键训练类型

**阈值跑 (Tempo Run)**:
| 类型 | 时长 | 强度 |
|-----|------|------|
| 持续阈值 | 20-40分钟 | T区 |
| 巡航间歇 | 3-4×10分钟 | T区 |

**间歇训练**:
| 类型 | 距离 | 重复 | 强度 |
|-----|------|------|------|
| VO2max训练 | 800-1600m | 4-6次 | I区 |
| 速度训练 | 400-600m | 6-8次 | R区 |

### 3.4 发展期训练示例

**典型发展期周训练**:
```
周一: 休息
周二: 8km 含阈值段落 (20min T区)
周三: 6km 轻松跑
周四: 间歇训练 (6×800m I区)
周五: 休息
周六: 长跑 20-25km (含最后段落T区)
周日: 恢复跑 6km
```

---

## 四、竞赛期训练

### 4.1 竞赛期目标

**生理适应**:
- 达到巅峰状态
- 验证比赛配速
- 最大化糖原储存
- 心理准备

**时间**: 通常4-6周

### 4.2 竞赛期训练构成

**训练类型分布**:
| 训练类型 | 比例 | 说明 |
|---------|------|------|
| 轻松跑 | 40-50% | 恢复为主 |
| 马拉松配速 | 20-30% | 专项训练 |
| 阈值/间歇 | 10-15% | 维持状态 |
| 长跑 | 15-20% | 最后大运动量 |

### 4.3 马拉松配速训练

**MP训练 (Marathon Pace)**:
| 类型 | 时长/距离 | 强度 |
|-----|----------|------|
| MP持续 | 20-30分钟 | M区 |
| MP长跑 | 2-3小时 | M区 |
| 负分割 | 前半E区/后半M区 | 渐进 |

### 4.4 竞赛期训练示例

**竞赛期关键周**:
```
周一: 休息
周二: 10km 含8km MP
周三: 6km 轻松
周四: 阈值 (3×5min T区)
周五: 休息
周六: 长跑 28-32km (后半段M配速)
周日: 恢复跑 8km
```

## 五、长跑训练

### 5.1 长跑的重要性

**生理适应**:
- 增强脂肪氧化能力
- 提高糖原储存容量
- 强化心理韧性
- 适应长时间运动

**建议频率**: 每周1次

### 5.2 长跑距离规划

**渐进原则**:
| 周数 | 最大长跑距离 |
|-----|------------|
| 基础期末 | 16-20km |
| 发展期中期 | 25-30km |
| 发展期末 | 32-35km |
| 竞赛期 | 35-38km |
| 减量期 | 20-25km |

### 5.3 长跑配速

**配速建议**:
| 训练目的 | 配速 |
|---------|------|
| 基础期 | E配速 |
| 发展期前半 | E-M配速 |
| 发展期后半 | M配速 |
| 竞赛期 | M配速（后期可含T区） |

### 5.4 长跑补给

**训练中补给**:
- 超过90分钟的训练进行补给
- 目标: 30-60g碳水/小时
- 训练使用与比赛相同的补给品

## 七、恢复与减量

### 7.1 恢复周

**频率**: 每3-4周安排一个恢复周

**恢复周特点**:
- 跑量减少30-50%
- 强度不变或略降低
- 以轻松跑为主

### 7.2 减量期 (Tapering)

**目的**:
- 消除疲劳
- 恢复糖原储备
- 达到巅峰状态

**减量策略**:
| 赛前周 | 跑量 | 强度 |
|-------|------|------|
| 3周 | 80% | 正常 |
| 2周 | 60% | 略降 |
| 1周 | 40% | 降低 |

---

## 八、训练监控

### 8.1 监控指标

**生理指标**:
- 静息心率
- HRV (心率变异性)
- 睡眠质量
- 体重变化

**训练指标**:
- RPE (训练感觉)
- 运动表现测试
- 主观疲劳感

### 8.2 调整原则

**调整信号**:
| 信号 | 建议 |
|-----|------|
| 持续疲劳 | 减少训练量 |
| 静息心率升高 | 增加恢复 |
| 表现下降 | 减少强度 |
| 伤痛 | 休息或就医 |

---

## 九、结论

马拉松训练周期是一个系统性的过程，需要科学规划、严格执行和灵活调整。基础期建立有氧基础，发展期提升专项能力，竞赛期达到巅峰状态。合理的训练量分布、强度安排和恢复策略是成功的关键。每个跑者应根据自身情况调整训练计划，并在训练中进行持续监控。
