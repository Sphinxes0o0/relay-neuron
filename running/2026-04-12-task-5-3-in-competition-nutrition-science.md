## 一、赛中补给概述

赛中补给是马拉松等耐力运动表现的关键因素。正确的补给策略可以延缓疲劳、维持血糖水平、预防脱水，并最终提高运动表现。

### 1.1 赛中补给的重要性

**能量消耗**:
- 马拉松可消耗2000-3000 kcal
- 糖原储备通常只能维持约2小时
- 补给不足导致"撞墙"现象

**研究证据** (Cox et al., Sports Medicine 2020):
- 科学补给的运动员表现显著优于自由选择饮食的运动员
- 碳水化合物摄入可改善30-60分钟以上的运动表现
- 液体和电解质补充对维持体内平衡至关重要

### 1.2 赛中补给研究历程

| 时期 | 研究重点 |
|-----|---------|
| 1980s | 碳水化合物作为燃料 |
| 1990s | 液体与电解质补充 |
| 2000s | 补给时机优化 |
| 2010s | 碳水化合物类型研究 |
| 2020s | 个性化补给策略 |

## 二、碳水化合物补给科学

### 2.1 碳水化合物氧化率

**关键发现**:
- 单一碳水化合物最大氧化率: 约60 g/h
- 混合碳水化合物（葡萄糖+果糖）最大氧化率: 约90 g/h
- 高强度运动时氧化率降低

**最新研究** (Jeukendrup, 2017; Journal of Sports Sciences):
- 葡萄糖和果糖的协同作用使碳水化合物氧化率提高
- 建议长时间运动使用混合碳水化合物来源
- 超过吸收能力的摄入会导致胃肠问题

### 2.2 碳水化合物摄入建议

**按运动时长建议** (基于ACSM和A立场声明):

| 运动时长 | 碳水化合物摄入 |
|---------|--------------|
| <45分钟 | 通常不需要 |
| 45-75分钟 | 少量，或仅在最后阶段 |
| 1-2.5小时 | 30-60 g/h |
| >2.5小时 | 60-90 g/h |

**马拉松补给目标**:
| 项目 | 建议量 |
|-----|--------|
| 总碳水 | 60-90 g/h |
| 葡萄糖 | 30-60 g/h |
| 果糖 | 15-30 g/h |
| 总液体 | 400-800 ml/h |

### 2.3 碳水化合物来源

**能量胶 (Energy Gels)**:
| 类型 | 碳水含量 | 特点 |
|-----|---------|------|
| 标准胶 | 20-25g | 需要水配合 |
| 高浓度胶 | 30-40g | 更黏稠 |
| 双重碳水 | 25-30g | 葡萄糖+果糖 |
| 含咖啡因 | 25-30g | 额外刺激 |

**能量饮料**:
| 浓度 | 碳水含量 | 适用情况 |
|-----|---------|---------|
| 低渗 | <5% | 快速吸收 |
| 等渗 | 5-7% | 最佳选择 |
| 高渗 | >8% | 需配合大量水 |

**固体食物**:
| 食物 | 碳水含量 | 优缺点 |
|-----|---------|--------|
| 能量棒 | 20-30g | 需咀嚼，可能难消化 |
| 香蕉 | 25-30g | 天然，但需提前准备 |
| 葡萄干 | 60g/100g | 简单碳水，但需水 |

## 三、水分与电解质补给

### 3.1 脱水风险

**脱水对运动表现的影响**:
| 脱水程度 | 影响 |
|---------|------|
| 1%体重 | 运动表现开始下降 |
| 2%体重 | 明显影响，体温调节受损 |
| 3-5%体重 | 严重下降，可能需要退出 |
| >5%体重 | 危险，可能危及生命 |

**研究证据** (Sawka et al., 2007; Medicine & Science in Sports & Exercise):
- 脱水2%即可显著降低运动表现
- 高温环境下脱水风险更高
- 个体间出汗率差异显著

### 3.2 出汗率评估

**个人出汗率测量**:
```
出汗率 (L/h) = (运动前体重 - 运动后体重 + 摄入液体 - 排尿量) / 运动时间
```

**出汗率分类**:
| 类型 | 出汗率 | 建议补液量 |
|-----|--------|----------|
| 低 | <0.5 L/h | 400-600 ml/h |
| 中 | 0.5-1.0 L/h | 600-900 ml/h |
| 高 | 1.0-1.5 L/h | 900-1200 ml/h |
| 极高 | >1.5 L/h | 1200+ ml/h |

### 3.3 电解质补充

**钠的重要性**:
- 钠是汗液中主要电解质
- 平均出汗钠浓度: 40-80 mEq/L
- 高钠流失者需要更多补充

**电解质补充建议**:
| 条件 | 钠摄入 | 液体量 |
|-----|-------|--------|
| 正常条件 | 300-600 mg/h | 400-800 ml/h |
| 高温/大量出汗 | 600-1000+ mg/h | 800-1200 ml/h |

**其他电解质**:
| 电解质 | 功能 | 流失量 |
|-------|------|--------|
| 钾 | 肌肉收缩 | 50-150 mg/h |
| 镁 | 肌肉放松 | 30-50 mg/h |
| 氯 | 消化液 | 300-600 mg/h |

### 3.4 低钠血症风险

**运动相关低钠血症**:
- 过度饮水导致血钠稀释
- 严重情况可危及生命
- 女性、体重轻者风险较高

**预防措施**:
- 按需饮水，不要强制过量
- 摄入含钠电解质饮料
- 了解个人出汗率

## 四、补给时机策略

### 4.1 赛前补给

**赛前饮食原则**:
- 赛前3-4小时: 主餐（碳水化合物为主，低脂低纤维）
- 赛前1-2小时: 轻度碳水化合物补充
- 赛前30分钟: 能量胶或碳水饮料（可选）

**赛前饮食示例** (假设7:00出发):
| 时间 | 食物 | 碳水含量 |
|-----|------|---------|
| 前晚(20:00) | 大餐-米饭+鱼肉 | 150-200g |
| 睡前(22:00) | 面包+蜂蜜 | 40-50g |
| 早5:30 | 燕麦+香蕉+果汁 | 80-100g |
| 早6:30 | 能量胶 | 25-30g |

### 4.2 赛中补给计划

**补给时机原则**:
- 设定固定时间提醒
- 不要等到口渴才喝
- 碳水补给每30-45分钟

**马拉松补给时间表** (以3.5小时完赛为例):
| 距离 | 时间 | 动作 |
|-----|------|------|
| 5km | ~25min | 喝水 |
| 10km | ~50min | 能量胶 |
| 15km | ~1:15 | 喝水 |
| 半程 | ~1:30 | 能量胶+水 |
| 25km | ~2:00 | 喝水 |
| 30km | ~2:30 | 能量胶 |
| 35km | ~2:50 | 水 |
| 40km | ~3:15 | 最后的胶 |

### 4.3 补给策略调整

**根据条件调整**:

| 条件 | 调整策略 |
|-----|---------|
| 高温 | 增加液体摄入，减少碳水比例 |
| 高湿度 | 注意电解质补偿 |
| 高海拔 | 增加碳水化合物比例 |
| 下坡多 | 减少冲击，注意膝关节 |

## 五、胃肠问题管理

### 5.1 赛中胃肠问题原因

**常见原因**:
| 因素 | 机制 |
|-----|------|
| 高渗饮料 | 液体进入肠道 |
| 脂肪/纤维 | 延迟胃排空 |
| 脱水 | 肠道血流减少 |
| 高强度 | 血液重新分配远离肠道 |

### 5.2 预防策略

**补给品选择**:
- 选择熟悉的产品
- 避免高渗溶液
- 配合足量清水
- 避免在训练中尝试新品

**补给技术**:
- 每次少量，多次摄入
- 选择温和口味
- 注意包装便携性
- 考虑温度对产品的影响

### 5.3 常见错误

| 错误 | 后果 | 避免方法 |
|-----|------|---------|
| 摄入过多 | 胃部不适、恶心 | 按计划摄入 |
| 摄入不足 | 能量耗尽 | 提前计划 |
| 突然换产品 | 胃肠不适 | 训练中测试 |
| 只喝水 | 低钠血症 | 补充电解质 |

## 六、特殊赛事补给

### 6.1 越野跑补给

**挑战**:
- 地形复杂，补给困难
- 长时间持续运动
- 环境条件多变

**特殊考虑**:
| 因素 | 策略 |
|-----|------|
| 补给站间隔 | 自带部分补给 |
| 温度变化 | 调整液体和碳水比例 |
| 技术路段 | 快速吸收的凝胶 |
| 长CP段 | 高热量密度食物 |

### 6.2 超马补给

**超马特点**:
- 运动时间远超4小时
- 能量需求极大
- 胃肠问题更常见

**超马补给策略**:
- 使用多种碳水化合物来源
- 包含固体和液体食物
- 注意蛋白质和脂肪
- 频繁少量摄入

### 6.3 团队赛/接力赛

**注意事项**:
- 交接区补给规划
- 运动员个人补给包
- 交接区外额外补给
- 简单快速的补给品

## 七、2020-2026研究进展

### 7.1 最新研究

**碳水化合物氧化研究**:
- 2023年研究证实葡萄糖-果糖联合的协同效应
- 新型碳水化合物来源（如异麦芽酮糖）的吸收特性得到研究

- 2020-2024年研究重新评估过度饮水风险

**补给产品发展**:

### 8.2 比赛补给原则

1. **一致性**: 按计划执行，不要等到渴了或饿了
2. **熟悉性**: 仅使用训练中测试过的产品
3. **渐进性**: 从少量开始，逐步调整
4. **灵活性**: 根据情况适时调整

### 8.3 常见问题处理

| 问题 | 可能原因 | 解决方案 |
|-----|---------|---------|
| 胃部不适 | 摄入过多或高渗 | 减少量或用水稀释 |
| 恶心 | 产品不耐受或过凉 | 换产品或室温使用 |
| 抽筋 | 电解质不足或脱水 | 补充盐分和液体 |
| 能量不足 | 碳水摄入不足 | 增加能量胶频率 |

## 九、结论

赛中补给是耐力运动表现的关键因素。通过科学规划碳水化合物、水分和电解质的摄入时机和数量，可以显著提高运动表现。关键是个体化策略、在训练中充分测试、并根据比赛条件灵活调整。预防胃肠问题和低钠血症同样重要，需要在补给计划中予以考虑。
