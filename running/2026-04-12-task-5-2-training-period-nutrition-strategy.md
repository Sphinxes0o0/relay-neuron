## 一、训练期营养概述

训练期营养是跑步训练成功的基础保障。正确的营养策略能够支持训练适应、加速恢复、维持免疫功能，并优化运动表现。

### 1.1 训练期营养的重要性

**核心功能**:
- 支持训练适应和超量恢复
- 补充训练中消耗的能量储备
- 维持免疫系统功能
- 促进肌肉修复和增长
- 预防过度训练综合征

### 1.2 训练期 vs 比赛期营养差异

| 方面 | 训练期 | 比赛期 |
|-----|--------|--------|
| 能量需求 | 取决于训练强度 | 高度集中 |
| 碳水比例 | 中等 (50-55%) | 较高 (60-70%) |
| 蛋白质需求 | 较高 (修复需要) | 维持性 |
| 进餐时机 | 规律饮食 | 灵活调整 |
| 试验新食物 | 可以 | 避免 |

## 二、能量需求与宏量营养素

### 2.1 能量需求计算

**基础能量消耗 (BMR)**:
- 男性: 约 24 kcal/kg 体重/天
- 女性: 约 22 kcal/kg 体重/天

**跑步训练能量消耗**:
| 运动类型 | 能量消耗 (kcal/km) |
|---------|-------------------|
| 轻松跑 | 50-60 |
| 马拉松配速 | 60-70 |
| 节奏跑 | 70-85 |
| 间歇训练 | 80-100 |
| 越野跑 | 80-120+ |

**体重70kg跑者示例**:
- 轻松跑10公里: 约500-600 kcal
- 间歇训练: 约800-1000 kcal

### 2.2 碳水化合物需求

**训练期碳水化合物摄入建议**:

| 训练强度 | 碳水化合物摄入 | 70kg跑者示例 |
|---------|--------------|-------------|
| 低强度/休息日 | 5-6 g/kg | 350-420g |
| 中等强度训练 | 6-7 g/kg | 420-490g |
| 高强度训练日 | 7-10 g/kg | 490-700g |
| 超长训练日 | 10-12 g/kg | 700-840g |

**研究证据**:
- 研究表明碳水化合物储备对训练质量有直接影响
- 训练期间维持适当的糖原水平可提高训练强度承受能力
- 低碳水化合物训练可能增强脂肪氧化能力，但可能限制高强度训练表现

**碳水化合物来源**:
| 食物 | 碳水含量 (每100g) |
|-----|------------------|
| 米饭 | 28g |
| 面食 | 25g |
| 燕麦 | 60g |
| 土豆 | 17g |
| 香蕉 | 23g |
| 全麦面包 | 40g |
| 运动饮料 | 6g/100ml |
| 能量胶 | 25-30g/条 |

### 2.3 蛋白质需求

**蛋白质摄入建议**:

| 跑者类型 | 蛋白质需求 | 70kg跑者 |
|---------|-----------|---------|
| 普通跑者 | 1.2-1.4 g/kg | 84-98g |
| 高强度训练 | 1.4-1.6 g/kg | 98-112g |
| 力量训练为主 | 1.6-2.0 g/kg | 112-140g |

**研究证据** (Kreider et al., 2017; Journal of the International Society of Sports Nutrition):
- 蛋白质摄入1.4-2.0 g/kg体重对耐力运动员是最优范围
- 更高的蛋白质摄入（>2.0 g/kg）对瘦体重增加有益
- 蛋白质分次摄入（每3-4小时）可优化肌肉蛋白质合成

**最佳蛋白质摄入时机**:
- 训练后30-120分钟内摄入20-40g蛋白质
- 睡前摄入缓释蛋白质可支持夜间恢复
- 每餐摄入15-25g蛋白质

**蛋白质来源**:
| 食物 | 蛋白质含量 (每100g) |
|-----|-------------------|
| 鸡胸肉 | 31g |
| 三文鱼 | 20g |
| 鸡蛋 | 13g |
| 希腊酸奶 | 10g |
| 豆腐 | 8g |
| 扁豆 | 9g |
| 蛋白粉 | 80g |

### 2.4 脂肪需求

**脂肪摄入建议**:
- 脂肪应提供20-30%的总能量摄入
- 重点摄入单不饱和脂肪和多不饱和脂肪
- 限制饱和脂肪摄入（<10%总能量）

**脂肪类型重要性**:
| 脂肪类型 | 来源 | 建议 |
|---------|------|------|
| Omega-3 | 深海鱼、亚麻籽 | 增加摄入 |
| Omega-6 | 坚果、植物油 | 平衡摄入 |
| 单不饱和 | 橄榄油、牛油果 | 优先选择 |
| 饱和脂肪 | 红肉、奶制品 | 限制摄入 |

## 三、训练周期营养策略

### 3.1 基础训练期营养

**特点**:
- 训练强度相对较低
- 跑量逐渐增加
- 重点是建立有氧基础

**营养策略**:
| 营养素 | 建议 | 目的 |
|-------|------|------|
| 碳水 | 5-7 g/kg | 维持糖原储备 |
| 蛋白质 | 1.4-1.6 g/kg | 肌肉修复 |
| 脂肪 | 20-30%总热量 | 激素健康 |
| 纤维 | 中等 | 肠道健康 |

### 3.2 强化训练期营养

**特点**:
- 高强度训练增加
- 训练量达到峰值
- 身体承受最大应激

**营养策略**:
| 营养素 | 建议 | 目的 |
|-------|------|------|
| 碳水 | 7-10 g/kg | 补充糖原消耗 |
| 蛋白质 | 1.6-2.0 g/kg | 支持恢复 |
| 脂肪 | 20-25%总热量 | 维持能量 |
| 电解质 | 增加 | 补充汗液流失 |

**强化期饮食示例** (70kg跑者，高强度训练日):
```
早餐: 燕麦+香蕉+牛奶 (100g碳水，30g蛋白)
午餐: 米饭+鸡胸肉+蔬菜 (120g碳水，40g蛋白)
下午训练前: 能量棒+水果 (60g碳水)
训练后: 蛋白质奶昔+碳水 (40g蛋白，80g碳水)
晚餐: 面食+鱼肉+蔬菜 (150g碳水，45g蛋白)
睡前: 希腊酸奶+坚果 (20g蛋白，15g脂肪)
```

### 3.3 减量训练期营养

**特点**:
- 训练量显著减少
- 为比赛做最后准备
- 碳水化合物负荷开始

**营养策略**:
| 阶段 | 碳水摄入 | 调整 |
|-----|---------|------|
| 减量第1周 | 6-7 g/kg | 正常饮食 |
| 减量第2周 | 7-8 g/kg | 增加碳水 |
| 赛前3天 | 8-10 g/kg | 逐步增加 |
| 赛前1天 | 10-12 g/kg | 最大化糖原 |

### 3.4 碳水化合物周期化

**Periodized Carbohydrate Approach** (基于2020-2023年研究):

**研究证据**:
- 2022年Systematic Review (Degn Gejl & Nybo) 发现碳水化合物周期化可能对训练适应有益
- 个体化碳水化合物摄入策略比固定方案更有效
- 训练强度高时日碳水摄入可达10-12 g/kg

**周期化方案示例**:

| 训练日类型 | 碳水摄入 | 策略 |
|-----------|---------|------|
| 高强度训练日 | 高碳水 (8-10 g/kg) | 优化训练质量 |
| 低强度训练日 | 中等碳水 (5-6 g/kg) | 增强脂肪氧化 |
| 休息日 | 较低碳水 (3-5 g/kg) | 代谢适应 |
| 比赛日 | 极高碳水 (10-12 g/kg) | 最大化糖原 |

## 四、训练营养实践

### 4.1 训练前营养

**训练前饮食原则**:
- 训练前2-4小时完成主餐
- 以碳水化合物为主
- 适量蛋白质
- 低脂低纤维

**训练前饮食示例** (训练在2小时后):
| 餐次 | 食物 | 碳水含量 |
|-----|------|---------|
| 早餐 | 燕麦+香蕉+果汁 | 80-100g |
| 训练前 | 能量胶 (30分钟前) | 25-30g |

### 4.2 训练中营养

**训练中补给原则**:

| 训练时长 | 补给策略 |
|---------|---------|
| <60分钟 | 通常不需要 |
| 60-90分钟 | 每小时30-60g碳水 |
| >90分钟 | 每小时60-90g碳水 + 电解质 |

**研究更新** (2024 ACSM指南):
- 长时间运动中摄入碳水可延缓疲劳
- 碳水化合物与电解质协同补充效果更佳
- 训练中蛋白质摄入可能有益于适应

### 4.3 训练后营养

**训练后营养时机**:
- 训练后30分钟内开始补充
- 持续2-4小时分次摄入
- 碳水:蛋白质比例 3:1 至 4:1

**训练后饮食示例**:
| 时机 | 食物 | 营养含量 |
|-----|------|---------|
| 即刻 (0-30分钟) | 蛋白质奶昔+水果 | 20-25g蛋白，30-40g碳水 |
| 主餐 (1-2小时) | 米饭+鸡胸肉+蔬菜 | 80-100g碳水，35-40g蛋白 |

### 4.4 蛋白质-碳水化合物协同

**研究证据** (European Journal of Applied Physiology, 2020):
- 碳水化合物-蛋白质联合补充可增强耐力训练适应
- 与单纯碳水相比，联合补充可能更好地促进糖原再合成
- 蛋白质摄入时机可能影响肌肉蛋白质合成

## 五、特殊训练情况营养

### 5.1 连续高强度训练

**连续训练日营养策略**:
- 确保充足的夜间恢复营养
- 增加早间碳水化合物摄入
- 训练间期适度补充
- 注意免疫营养（维生素C、D、锌）

### 5.2 空腹训练

**空腹训练利弊**:
| 潜在好处 | 潜在风险 |
|---------|---------|
| 增强脂肪氧化能力 | 可能降低训练强度 |
| 提高胰岛素敏感性 | 增加肌肉分解风险 |
| 适应酮症代谢 | 不适合高强度训练 |

**建议**:
- 仅适用于低强度训练（轻松跑）
- 训练时间限制在60分钟内
- 训练后尽快补充营养

### 5.3 夜间训练

**夜间训练营养考虑**:
- 训练前1-2小时适度进食
- 选择易消化的碳水化合物
- 训练后补充蛋白质和碳水
- 注意睡前营养支持恢复

## 六、营养与恢复

### 6.1 恢复期营养原则

**恢复营养四要素** (RICE原则的演变):
- **Rehydrate**: 补充水分和电解质
- **Refuel**: 补充糖原储备
- **Repair**: 提供蛋白质支持肌肉修复
- **Rest**: 避免过度刺激消化系统

### 6.2 水分与电解质

**出汗率估算**:
- 测量运动前后体重差
- 每减少1kg体重需补充1.2-1.5L液体

**电解质补充**:
| 电解质 | 功能 | 来源 |
|-------|------|------|
| 钠 | 维持水分平衡 | 运动饮料、盐片 |
| 钾 | 肌肉功能 | 香蕉、土豆 |
| 镁 | 肌肉放松 | 坚果、绿叶菜 |
| 氯 | 消化液成分 | 盐分 |

### 6.3 免疫营养

**支持免疫的关键营养素**:
| 营养素 | 功能 | 食物来源 |
|-------|------|---------|
| 维生素C | 免疫细胞功能 | 柑橘、浆果 |
| 维生素D | 免疫调节 | 阳光、鱼油 |
| 锌 | 免疫细胞发育 | 瘦肉、海鲜 |
| 谷氨酰胺 | 免疫细胞能量 | 肉类、豆类 |

## 七、2020-2026研究进展

### 7.1 关键研究

**碳水化合物周期化研究**:
- 2022年Degn Gejl等人的系统综述分析了碳水化合物周期化对耐力运动员的影响
- 发现个体化策略优于固定方案
- 训练强度匹配是关键因素

**蛋白质时机研究**:
- 训练后蛋白质摄入时机窗口（30-120分钟）的重要性得到进一步验证
- 睡前蛋白质补充支持恢复的概念得到扩展
- 亮氨酸在触发肌肉蛋白质合成中的作用被深入研究

**营养 periodization 新概念**:
- 训练期营养与训练周期同步调整
- 根据训练周期调整宏量营养素比例
- 个体化营养策略的发展

### 7.2 新发现

**代谢灵活性**:
- 训练良好者更能灵活利用不同燃料
- 碳水化合物周期化可能增强代谢灵活性
- 脂肪适应训练需与高强度训练平衡

## 八、实践建议

### 8.1 训练营养检查清单

- [ ] 计算个人能量需求
- [ ] 确定碳水化合物目标 (5-10 g/kg)
- [ ] 确保蛋白质摄入 (1.4-2.0 g/kg)
- [ ] 选择健康脂肪来源
- [ ] 制定训练前/中/后营养计划
- [ ] 准备训练补给品
- [ ] 记录训练后恢复情况
- [ ] 调整营养策略

### 8.2 常见错误

| 错误 | 风险 | 解决方案 |
|-----|------|---------|
| 训练期摄入不足 | 疲劳、表现下降 | 增加总热量 |
| 蛋白质不足 | 肌肉分解、恢复慢 | 增加蛋白质摄入 |
| 脂肪过低 | 激素失衡 | 确保20-30%脂肪 |
| 补水不足 | 脱水、抽筋 | 建立补水习惯 |

## 九、结论

训练期营养是跑步训练成功的基础。通过合理的宏量营养素分配、正确的补给时机、个体化的营养策略，跑者可以最大化训练适应、优化恢复、预防损伤。关键是根据训练强度和周期调整营养策略，保持能量平衡，并确保所有关键营养素的充足摄入。
