# 转接神经元 (Relay Neuron)

健身/运动科学文献研究存档，涵盖 2024-2026 年最新研究。

## 项目结构

```
relay-neuron/
├── README.md
├── RESEARCH_MASTER_LIST.md      # 研究任务清单
└── research/
    ├── YYYY-MM-DD-research-round-N.md   # 综合研究轮次
    └── YYYY-MM-DD-task-X-X-*.md         # 专题研究文件
```

## 研究领域

| 分类 | 主题 |
|------|------|
| 肌肥大与肌肉增长 | 阻力训练机制、mTOR通路、卫星细胞 |
| 营养与补剂 | 蛋白质、肌酸、亮氨酸、Omega-3 |
| 阻力训练科学 | 训练频率、训练量、BFR、离心/向心 |
| 心肺与代谢 | HIIT、VO2max、代谢综合征 |
| 睡眠与恢复 | 运动睡眠关系、恢复期蛋白合成 |
| 健康与疾病 | 衰老肌少症、糖尿病、心血管健康 |
| 碳水化合物研究 | 肌糖原、碳水时机、血糖控制 |
| 鱼油与Omega-3 | 抗炎、心血管、认知功能 |
