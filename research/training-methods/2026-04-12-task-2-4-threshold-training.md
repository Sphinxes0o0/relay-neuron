## 一、阈值训练概述

阈值训练（Threshold Training）是跑步训练中最重要的训练方法之一，通过在乳酸阈值强度或附近进行训练来提高有氧能力。

### 1.1 定义

**阈值训练原理**:
- 在乳酸阈值(LT)强度进行训练
- 该强度接近但略低于无氧阈
- 训练时间通常为20-40分钟
- 也称为"T-pace"或"阈值配速"

### 1.2 Runner's World 定义

**Threshold/ T-pace running**:
- 最高效的训练类型之一
- 帮助跑者避免过度训练
- 产生更满意的训练效果
- 平衡有氧和无氧挑战

## 二、阈值的生理机制

### 2.1 乳酸阈值

| 概念 | 定义 |
|-----|------|
| 乳酸阈值 (LT) | 血乳酸开始急剧累积的点 |
| 无氧阈 (AT) | 乳酸产生>清除 |
| 最大乳酸稳态 (MLSS) | 可维持的最高有氧强度 |

### 2.2 训练适应

| 适应类型 | 变化 |
|---------|------|
| 乳酸脱氢酶 | 活性提高 |
| 乳酸转运蛋白 | 清除效率增强 |
| 线粒体氧化能力 | 增强 |
| 肌纤维耐疲劳性 | 改善 |

## 四、阈值训练的实践应用

### 4.1 训练强度

**确定阈值配速的方法**:

| 方法 | 描述 |
|-----|------|
| 心率 | 85-90% 最大心率 |
| 配速 | 10K配速 + 15-20秒/公里 |
| 谈话测试 | 勉强对话 |

### 4.2 训练处方

**基本参数**:
| 参数 | 建议值 |
|-----|-------|
| 强度 | LT配速或T-pace |
| 时长 | 20-40分钟 |
| 频率 | 每周1-2次 |
| 间歇 | 无需完全休息 |

### 4.3 训练形式

**持续阈值跑**:
- 20-40分钟不间断
- 稳定配速
- 适合有基础的跑者

**阈值间歇**:
- 3-5组 × 10-15分钟
- 组间1-2分钟休息
- 可控制强度

**Tempo跑** (已在2.2中详述):
- 包含热身的完整训练
- 实际阈值时间20-30分钟

## 六、阈值训练的注意事项

### 6.1 常见错误

| 错误 | 正确做法 |
|-----|---------|
| 配速过快 | 以谈话测试为准 |
| 过于频繁 | 每周1-2次足够 |
| 热身不足 | 至少5公里热身 |
| 恢复不足 | 确保间隔充足 |

### 6.2 过度训练信号

- 表现持续下降
- 晨起静息心率升高
- HRV异常
- 持续疲劳感

## 九、实践建议

### 9.1 训练计划

**中级跑者阈值训练周**:
```
周一: 休息
周二: 间歇 - 6×1000m
周三: 阈值 - 25分钟 @ T-pace
周四: 轻松跑
周五: 休息
周六: 长距离 - 20km
周日: 休息
```

### 9.2 阈值识别

**谈话测试标准**:
- 你可以说话但稍微费力
- 不能唱歌
- 句子长度受限

## 十、结论

阈值训练是提高跑者有氧能力和乳酸阈值的关键方法。通过在适当的强度进行训练，可以有效改善代谢能力和比赛表现。关键在于准确的配速控制和充分的恢复。
