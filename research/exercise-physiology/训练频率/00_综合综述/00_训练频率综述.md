## 训练频率定义

### 基本定义

**训练频率（Training Frequency）** 指在特定时间段内（通常为一周）进行训练的次数，反映了刺激施加的时间密度。

### 操作性分类

| 类别 | 定义 | 典型周期 |
|-----|------|---------|
| 低频 | < 2次/周/肌群 | 隔周训练 |
| 中频 | 2-3次/周/肌群 | 每周2-3次 |
| 高频 | 4-6次/周/肌群 | 每天训练 |
| 分化训练 | 每次训练不同肌群 | 每周轮换 |

### 关键指标

- **频率（Frequency）**: 单位时间内训练次数
- **间期（Interval）**: 两次训练同一肌群的间隔时间
- **周期化（Periodization）**: 训练计划的系统性变化

> **来源**: Kraemer & Ratamess (2004) [^1]

## 肌肉恢复时间

### 生理机制

肌肉恢复涉及多个生理系统的协同恢复：

```
训练刺激 → 肌纤维损伤 → 炎症反应 → 蛋白质合成 → 超量恢复
    0-24h          24-72h           48-96h          72-120h
```

### 恢复时间框架

| 组织/系统 | 恢复时间 | 影响因素 |
|----------|---------|---------|
| 肌糖原储备 | 24-48小时 | 碳水摄入、运动强度 |
| ATP-PCr系统 | 3-5分钟 | 训练强度 |
| 肌纤维超微结构 | 24-96小时 | 训练量、遗传因素 |
| 神经-肌肉接头 | 24-48小时 | 训练经验 |
| 胶原蛋白合成 | 48-72小时 | 营养状态 |
| 激素环境 | 24-72小时 | 睡眠、压力 |

> **来源**:居胡远等 (2021)[^2]，Kraemer et al. (2020)[^3]

### 延迟性肌肉酸痛（DOMS）

DOMS通常在训练后24-72小时达到峰值，与以下因素相关：

- 离心收缩比例
- 训练量增加幅度
- 新训练动作的引入
- 个体恢复能力差异

### 影响恢复速度的因素

| 加速恢复 | 延缓恢复 |
|---------|---------|
| 充足睡眠（7-9小时） | 睡眠不足 |
| 蛋白质摄入（1.6-2.2g/kg/天） | 营养不良 |
| 主动恢复 | 过度训练 |
| 年龄 < 30岁 | 年龄 > 40岁 |
| 训练经验丰富 | 训练经验不足 |

> **来源**: Kerksick et al. (2017)[^4]

## 频率与肌肉增长

### 剂量-反应关系概述

肌肉增长（肥大）为非线性剂量-反应关系，存在收益递减点：

```
肌肉蛋白质合成 (MPS) 响应
        ↑
        │    ╭───────────── 高频区间（递减）
        │   ╱
        │  ╱
        │ ╱ ← 中频区间（最优效率比）
        │╱
        └──────────────────────────→
              训练频率 (次/周/肌群)
```

### 元分析研究证据

**Schoenfeld et al. (2016) Meta分析[^5]**:
- 比较低频（≤2次/周）与高频（≥3次/周）训练
- 发现高频组肌肉增长优势显著（p < 0.05）
- 组间差异约2-3% (可能无临床意义)
- **结论**: 频率本身并非决定因素，总训练量是关键

**Schoenfeld et al. (2021) RCT[^6]**:
- 8周干预
- 低频组: 3次/周全身训练
- 高频组: 6次/周（每次1/3肌群）
- 结果: 无组间差异
- **结论**: 等总训练量下，频率对肥大影响有限

### 每肌群每周训练组数与增长关系

| 每周组数 | 预期增长 | 适用人群 |
|---------|---------|---------|
| 0-4组 | 最小/无 | 久坐人群入门 |
| 5-9组 | 轻度增长 | 初级阶段 |
| 10-15组 | 中等增长 | 中级训练者 |
| 16-20组 | 显著增长 | 中高级训练者 |
| 20-30组 | 最大增长（ plateau风险） | 高级训练者 |
| > 30组 | 过度训练风险 | 仅限特殊阶段 |

> **来源**: Brad et al. (2021)[^7]

### 频率对不同肌群的影响差异

大肌群（背、腿）可能受益于较高频率：

| 肌群 | 推荐最低频率 | 原因 |
|-----|------------|-----|
| 股四头肌 | 2次/周 | 体积大，恢复慢 |
| 胸肌 | 1-2次/周 | 恢复较快 |
| 背阔肌 | 2次/周 | 结构复杂 |
| 三角肌 | 2-3次/周 | 小肌群，易疲劳 |
| 肱二头肌 | 2次/周 | 小肌群 |
| 核心肌群 | 3-5次/周 | 参与多动作，可高频 |

## 力量训练频率

### 最大力量发展

力量提升对频率的响应具有高度个体差异性：

| 训练阶段 | 推荐频率 | 每次训练组数 | 强度 |
|---------|---------|-------------|-----|
| 力量基础期 | 2-3次/周 | 3-5组 | 70-85% 1RM |
| 力量建设期 | 3-4次/周 | 4-6组 | 80-90% 1RM |
| 力量峰值期 | 4-6次/周 | 1-3组 | 90-100% 1RM |
| 减量期 | 1-2次/周 | 2-3组 | 60-75% 1RM |

> **来源**: Tsatsakopoulos et al. (2023)[^8]

### 力量-频率关系研究

**R有两项经典研究[^9][^10]**:

1. **低频 vs 高频力量训练**:
   - 低频（1次/周）和高频（3次/周）在12周后1RM无显著差异
   - 高频组在力量耐力测试中表现更优

2. **每周2次 vs 3次训练**:
   - 12周后腿部力量增长相似
   - 每周3次组有边际优势

## 剂量-反应关系

### 训练量的量化框架

**一次训练典型容量**:
```
容量 = 组数 × 次数 × 负荷（重量）
```

**每周总容量计算**:
```
周训练量 = Σ(组数 × 次数 × 负荷) for 所有训练
```

### 非线性关系特征

剂量-反应曲线特征：

1. **阈值效应**: 需要最小刺激量才能产生适应
2. **线性阶段**: 在一定范围内增加量产生线性增益
3. **平台期**: 继续增加量但增益减小
4. **负效应区**: 超过恢复能力反而退步

### 基于证据的最佳实践范围

| 指标 | 最小有效剂量 | 最优剂量 | 最大安全剂量 |
|-----|------------|---------|-------------|
| 每周组数/肌群 | 6-8组 | 10-20组 | 25-30组 |
| 每周训练次数 | 1次 | 2-4次 | 6次以上 |
| 每次训练组数 | 3-4组 | 8-12组 | 15-20组以上 |
| 组间休息（ hypertrophy） | 60秒 | 90-120秒 | - |

> **来源**: Ralston et al. (2018)[^11]

## 实践建议

### 基于训练经验的频率建议

#### 初学者（< 1年系统训练）

**推荐**: 全身训练 2-3次/周

**原理**:
- 神经适应为主，高频率有助于技术巩固
- 总训练量积累比频率更重要
- 恢复能力强，可较频繁刺激

**示例课表**:
```
周一: 全身训练
周三: 全身训练
周五: 全身训练（可选）
```

#### 中级训练者（1-3年系统训练）

**推荐**: 分化训练 3-4次/周

**原理**:
- 肌肉肥大成为主要目标
- 需要更高训练量
- 每肌群每周10-15组

**示例课表**:
```
周一: 推类（上半身）
周二: 拉类（上半身）
周四: 腿+核心
周五: 全身（轻量）
```

#### 高级训练者（> 3年系统训练）

**推荐**: 高频分化 4-6次/周

**原理**:
- 局部刺激需要更频繁
- 训练量阈值更高
- 需要精细的周期化

**示例课表**:
```
周一: 胸+三头
周二: 背+二头
周三: 腿+肩
周四: 休息/主动恢复
周五: 胸+三头
周六: 背+二头
周日: 腿+核心
```

### 基于目标的频率调整

| 目标 | 推荐频率 | 关键变量 |
|-----|---------|---------|
| 肌肉肥大 | 2-4次/周 | 总训练量、蛋白质摄入 |
| 最大力量 | 2-4次/周 | 强度、神经适应 |
| 力量耐力 | 2-3次/周 | 次数、组间休息 |
| 体脂减少 | 3-5次/周 | 能量消耗、训练密度 |
| 运动表现 | 2-4次/周 | 专项性、周期化 |

### 频率周期化策略

**线性递增**:
```
第1-4周: 2次/周
第5-8周: 3次/周
第9-12周: 4次/周
第13-16周: 减量/恢复
```

**波动模型**:
```
第1周: 4次
第2周: 3次
第3周: 4次
第4周: 2次（减量周）
```

### 特殊情况处理

#### 恢复能力评估

监测恢复的实用指标：
- 晨脉（醒来静息心率）
- 睡眠质量与时长
- 肌肉酸痛程度（DOMS评分）
- 训练主观用力感（RPE）
- 情绪与认知状态

#### 训练频率不足的信号

- 训练后恢复异常缓慢
- 持续疲劳 > 48小时
- 力量/体能停滞
- 关节疼痛
- 睡眠障碍

#### 训练频率过高的信号

- 持续性疲劳
- 表现下降
- 频繁受伤
- 免疫力下降
- 心理倦怠

### 个体化考量因素

| 因素 | 倾向低频 | 倾向高频 |
|-----|---------|---------|
| 年龄 | > 40岁 | < 30岁 |
| 恢复能力 | 较差 | 良好 |
| 生活压力 | 高 | 低 |
| 睡眠 | < 7小时 | > 8小时 |
| 营养 | 一般 | 优化 |
| 训练经验 | 不足 | 丰富 |
| 训练强度 | 极高（> 90% 1RM） | 中等（60-80% 1RM） |
