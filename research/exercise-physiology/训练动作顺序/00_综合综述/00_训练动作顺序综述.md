# 训练动作顺序综述 (Exercise Order in Resistance Training)

**研究日期**: 2026-05-04
**研究类型**: 综合文献综述

---

## 摘要

训练动作顺序（Exercise Order）是抗阻训练处方中的关键变量之一，影响急性训练反应及慢性适应效果。本综述综合分析动作顺序对肌肉力量、肥大、疲劳及运动表现的影响机制，并基于现有证据提供实践建议。

## 一、动作顺序原理

### 1.1 定义与重要性

训练动作顺序指在一次训练课中安排不同训练动作的先后次序。Simão等（2012）在《Sports Medicine》发表的综述指出，动作顺序是影响抗阻训练急性反应和慢性适应的关键变量之一。

([Simão et al., 2012](https://link.springer.com/article/10.2165/11597240-000000000-00000))

### 1.2 动作分类

**按参与肌群分类：**
- **复合动作（Compound Exercises）**: 涉及多个关节和多肌群参与
  - 例如：深蹲、硬拉、卧推、划船
- **孤立动作（Isolation Exercises）**: 单一关节和单一肌群参与
  - 例如：二头肌弯举、腿弯举、侧平举

**按肌群大小分类：**
- **大肌群动作**: 胸、背、腿
- **小肌群动作**: 手臂、肩、小腿

**按强度分类：**
- 高强度动作（1-6RM）
- 中等强度动作（8-12RM）
- 低强度动作（15+RM）

## 二、复合动作优先原则

### 2.1 生理学依据

复合动作优先的理论基础在于：

1. **神经动员效率**: 复合动作需要较高的神经协调，在训练早期疲劳程度较低时能够动员更多运动单位
2. **能量系统参与**: 复合动作通常需要磷酸原系统和糖酵解系统共同参与，在训练早期能量储备充足
3. **激素反应**: 大重量复合动作可引起更大的急性激素反应（如睾酮、生长激素）

([Kraemer et al., 1991](https://pubmed.ncbi.nlm.nih.gov/2007511/))

### 2.2 研究证据

**急性反应研究：**
- Cadore等（2013）研究发现，在疲劳状态良好的情况下进行复合动作，能够完成更多的总训练量
- ([Cadore et al., 2013](https://pubmed.ncbi.nlm.nih.gov/23124222/))

**⚠️ TODO 2024-2026验证需求：**
- 需要更多随机对照试验验证复合优先对肌肉肥大的长期影响
- 现有研究多为急性反应研究，缺乏足够的慢性适应证据

### 2.3 实践建议

**推荐顺序（适用于大多数训练目标）：**

```
1. 复合大重量动作（深蹲、硬拉、卧推）
2. 其他复合动作（引体向上、划船）
3. 单关节/孤立动作（二头肌弯举、三头肌下压）
```

## 三、疲劳因素分析

### 3.1 疲劳对动作表现的影响

**外周疲劳机制：**
- 代谢产物积累（乳酸、H⁺）
- ATP耗竭
- 肌纤维兴奋-收缩耦合受损

**中枢疲劳机制：**
- 运动皮层兴奋性下降
- 运动单位放电频率降低

([Gandevia, 2001](https://pubmed.ncbi.nlm.nih.gov/11548001/))

### 3.2 疲劳与管理原则

**关键研究发现：**

1. **训练量一致时**：先完成复合动作再完成孤立动作，总训练量相似
2. **运动表现影响**：动作顺序显著影响每组重复次数和总完成量
3. **技术退化风险**：疲劳状态下进行复杂动作增加受伤风险

([Simoes et al., 2012](https://link.springer.com/article/10.2165/11597240-000000000-00000))

### 3.3 疲劳监测在动作顺序中的应用

**主观监测工具：**
- RPE（自觉用力程度）量表
- DOMS（延迟性肌肉酸痛）评估
- 睡眠质量和恢复问卷

**客观监测工具：**
- 心率变异性（HRV）
- 垂直跳跃测试
- 肌肉力量测试

## 四、动作顺序策略

### 4.1 ACSM推荐原则

美国运动医学会（ACSM）进展模型建议：

> "建议力量训练计划按以下顺序安排动作以优化运动强度保持：大肌群动作先于小肌群动作；多关节动作先于单关节动作；高强度动作先于低强度动作。"

([ACSM Progression Models, 2009](https://pubmed.ncbi.nlm.nih.gov/19204579/))

### 4.2 推-拉-腿原则（Push-Pull-Legs）

一种流行的动作顺序安排方法：

| 顺序 | 动作类型 | 示例 |
|------|----------|------|
| 1 | 推动作（Push） | 卧推、肩推 |
| 2 | 拉动作（Pull） | 引体向上、划船 |
| 3 | 腿动作（Legs） | 深蹲、腿举 |

**优点：**
- 平衡对抗肌群的发展
- 允许充分恢复时间
- 动作模式自然流畅

### 4.3 上下身分离原则（Upper-Lower Split）

| 日期 | 训练内容 |
|------|----------|
| 第1天 | 上身推类（胸、肩、三头） |
| 第2天 | 下身（腿、臀） |
| 第3天 | 上身拉类（背、二头） |
| 第4天 | 全身/休息 |

### 4.4 拮抗剂成对原则（Antagonist Pairing）

将对抗肌群连续进行训练：

```
例子1：胸背配对
- 卧推 3x8-12
- 划船 3x8-12

例子2：二头三头配对
- 二头弯举 3x10-12
- 三头下压 3x10-12
```

## 五、特殊人群与目标的动作顺序

### 5.1 肌肉肥大（Hypertrophy）目标

**推荐策略：**
1. 优先复合动作以最大化机械张力
2. 孤立动作在复合动作后进行，针对性刺激
3. 高训练量是关键，疲劳管理尤为重要

**⚠️ TODO 2024-2026验证需求：**
- 缺乏高质量RCT比较不同动作顺序对肌肉肥大的长期效果

### 5.2 最大力量（Strength）目标

**推荐策略：**
1. 训练开始即进行最大力量动作
2. 1-5RM的高强度复合动作优先
3. 避免疲劳累积影响技术执行

### 5.3 爆发力（Power）目标

**推荐策略：**
1. 神经激活优先，在最大清醒状态下训练
2. 复合爆发力动作（抓举、借力推）
3. 放在训练课最前面

### 5.4 体能/减脂目标

**推荐策略：**
1. 代谢压力优先
2. 较大训练量
3. 动作顺序可更灵活

## 七、常见误区

### 7.1 动作顺序常见错误

| 误区 | 正确做法 |
|------|----------|
| 训练开始就做孤立动作 | 先复合后孤立 |
| 小肌群先于大肌群 | 大肌群优先 |
| 低强度动作排在前面 | 高强度优先 |
| 每次训练随机安排顺序 | 系统化安排 |

### 7.2 ⚠️ TODO 2024-2026研究澄清需求

- 现有研究多为急性实验室研究，向实际训练的转化需要更多验证
- 不同训练水平个体的最优动作顺序可能不同

## 八、结论与建议总结

### 8.1 核心建议

1. **复合动作优先**: 在训练早期（疲劳程度最低时）进行多关节复合动作
2. **大肌群优先**: 大肌群动作先于小肌群动作
3. **高强度优先**: 高强度动作先于低强度动作
4. **技术优先**: 复杂/技术要求高的动作安排在训练早期
5. **个体化调整**: 根据训练目标和个体差异灵活调整

### 8.2 注意事项

- 动作顺序的影响可能因训练目标和个体差异而异
- 疲劳管理是动作顺序安排的核心考量
- 周期性调整动作顺序可能有助于长期适应
