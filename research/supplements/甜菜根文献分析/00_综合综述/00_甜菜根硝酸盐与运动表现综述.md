## 作用机制（NO3- → NO2- → NO通路）

### 经典通路（口-肠-唾液循环）

```
膳食硝酸盐 (NO3-)
       ↓ 吸收（胃肠道）
血液硝酸盐
       ↓ 浓缩（唾液腺主动摄取）
唾液硝酸盐
       ↓ 口腔细菌还原 ⚠️关键步骤
唾液亚硝酸盐 (NO2-)
       ↓ 吞咽吸收
血液亚硝酸盐
       ↓ 酸性/缺氧组织还原
一氧化氮 (NO)
       ↓
血管舒张、血流↑、线粒体效率↑
```

### 关键要点

1. **口腔细菌（Oral Microbiome）的核心作用**
   - 硝酸盐还原菌（如*Veillonella*、*Actinomyces*、*Streptococcus*）将NO3-还原为NO2-
   - 2024年研究证实：补充硝酸盐富集饮食可改变口腔脱氮细菌群落
   - **漱口（mouth rinse）会破坏这一过程，消除效益** ([Springer, 2024](https://link.springer.com/article/10.1007/s11756-024-01681-x))

2. **NO的生理作用**
   - 血管舒张 → 改善肌肉血流灌注
   - 降低血管阻力 → 降低心肌耗氧
   - 改善线粒体呼吸链效率 → 减少运动中VO2
   - 参与肌纤维钙处理 → 影响收缩效率

3. **缺氧/酸性环境优先效应**
   - NO2- → NO的还原在缺氧组织中更活跃
   - 运动时肌肉正是低氧环境，故NO产生增加
   - 这解释了为何硝酸盐对次极量运动效益更明显

## 剂量-反应关系

### 有效剂量范围

| 剂量（NO3-） | 对应BRJ量 | 效应 |
|--------------|----------|------|
| 0.1 mmol/kg | ~35 ml | 可能有效 |
| **0.3-0.5 mmol/kg** | **~70-140 ml** | **最佳效益** |
| >0.5 mmol/kg | >140 ml | 效益平台期/不增加 |

### 典型方案

- **急性补充**: 70-140 ml浓缩甜菜根汁（含300-500 mg NO3-），运动前2-3小时
- **慢性补充**: 2-7天，每天70-140 ml
- **注意**: 效益在单次大剂量后即可出现，慢性补充可能进一步增强

## 运动表现效益

### 1. 最大摄氧量（VO2max）

- **效益幅度**: 1-5%（研究间异质性较大）
- **机制**: 降低动静脉氧差，改善线粒体P/O比
- **适用性**: 主要见于亚极量长时间运动
- ⚠️ **需验证**: 2024-2026年高质量RCT证据

### 2. 时间trial表现

- **效益幅度**: 比赛时间缩短1-2.5%
- **典型研究**: 4-16.1 km计时赛，效益约1-2%
- **机制**: 允许更高功率输出，延迟疲劳
- ([Frontiers in Physiology, 2017](https://www.frontiersin.org/articles/10.3389/fphys.2017.00401/pdf?isPublishedV2=False))

### 3. 亚极量运动效率

- **效益**: VO2降低3-5%（固定功率下）
- **实际意义**: 节省能量，或同等能耗完成更多工作
- **最强证据区**: 60-75% VO2max强度

### 4. 高强度间歇运动（HIIT）

- **效果不一**: 部分研究显示效益，部分无显著改善
- **可能原因**: 峰值功率输出受益较小（NO通路非主要限制因素）
- ⚠️ **需验证**: 2024-2026年系统评价

### 5. 短跑/无氧功率

- **效果有限或无**: 研究结果不一致
- **原因**: 无氧供能主导时NO通路作用较小

## 最佳摄入时机

### 急性补充方案

| 时机 | 效应 |
|------|------|
| 运动前**2-3小时** | **最佳**（NO2-血浓度达峰） |
| 运动前30-60分钟 | 可能有效（峰浓度提前） |
| 运动前>3小时 | 效益下降 |
| 运动即刻 | 无急性效益 |

### 典型推荐

> **70-140 ml甜菜根汁，运动前2-3小时摄入**

### 慢性补充

- 2-7天连续补充可能进一步增强效益
- 但单次急性补充已可产生显著效果

## 漱口效应与口腔细菌

### 核心发现

**口腔硝酸盐还原细菌是甜菜根效益的关键媒介**

- 抗菌漱口水（含氯己定等）完全消除甜菜根汁的效益
- 机制：杀灭将NO3-还原为NO2-的口腔细菌
- 这意味着"NO3- → NO2- → NO"通路高度依赖口腔微生物组

### 实际意义

| 情况 | 影响 |
|------|------|
| 抗菌漱口水使用者 | 可能无法获得效益 |
| 广谱抗生素使用者 | 可能影响转化效率 |
| 口腔卫生良好者 | 效益可能更稳定 |

### ⚠️ TODO：待验证

- [ ] 益生菌（含硝酸盐还原菌）是否可以恢复效益
- [ ] 个体口腔微生物组成差异对效果的影响程度

## 与L-瓜氨酸对比（不同NO通路）

### 通路差异

| 特征 | 膳食硝酸盐（甜菜根） | L-瓜氨酸 |
|------|---------------------|----------|
| **NO前体** | NO3- → NO2- → NO | L-Cit → L-Arg → NO |
| **转化位置** | 口腔+血液+组织 | 肾脏+血管内皮 |
| **转化效率** | 较快（2-3小时达峰） | 较慢（需酶促反应） |
| **口腔细菌需求** | 必须（关键） | 不需要 |
| **NO直接来源** | NO2- | L-Arg（需NOS） |

### 效益比较

| 指标 | 硝酸盐 | L-瓜氨酸 |
|------|--------|----------|
| 血管功能 | 明确 | 明确 |
| 运动表现 | 中等 | ⚠️ 证据有限 |
| 肠道血流 | 不明确 | 研究支持 |
| 热应激环境 | 有限 | 2024年有研究 |
| 血压 | 降低 | 降低 |

### 关键研究

- **Meta分析**（[J Int Soc Sports Nutr, 2021](https://link.springer.com/article/10.1186/s12970-021-00472-y)）: 56项硝酸盐研究证实效益；L-瓜氨酸仅3项研究，无显著效益
- **2024年L-瓜氨酸研究**（[European J Appl Physiol, 2024](https://link.springer.com/10.1007/s00421-024-05671-4)）: 7天6g/d Citrulline在热环境下跑步表现无改善

### 实际建议

- **优先选择硝酸盐**: 证据更充分，效益更明确
- **L-瓜氨酸适用场景**:
  - 肠道血流改善需求（运动相关胃肠问题）
  - 热应激环境
  - 对硝酸盐不耐受者
- **可考虑联合**: 理论上不同NO通路可能有协同，但证据有限

## 安全性

### 总体评价

- **一般认为是安全的**（GRAS认证）
- 天然食物来源（甜菜根）
- 数十年的研究历史

### 常见不良反应

| 反应 | 发生率 | 处理 |
|------|--------|------|
| 尿液/粪便粉红（甜菜尿） | 常见 | 无害，继续使用 |
| 胃肠道不适 | 少见-中等 | 减少剂量 |
| 腹泻 | 少见 | 减少剂量 |

### 特殊人群注意

- **肾功能障碍**: 谨慎（硝酸盐代谢负担）
- **低血压倾向**: 注意血压叠加效应
- **怀孕/哺乳**: 数据有限，避免或遵医嘱

## 摘要

膳食硝酸盐（主要来源为甜菜根汁）通过**NO3- → NO2- → NO**通路增强运动表现。该通路依赖**口腔硝酸盐还原细菌**，漱口或抗生素使用可消除其效益。有效剂量为**0.3-0.5 mmol/kg NO3-**（约70-140 ml甜菜根汁），运动前**2-3小时**摄入。效益主要见于：
- VO2max适度提升（1-5%）
- 时间trial表现改善（1-2.5%）
- 亚极量运动效率提高

与L-瓜氨酸相比，硝酸盐有更充分的运动表现证据，而L-瓜氨酸可能在肠道血流和热应激环境有独特价值。总体安全性良好，主要注意胃肠道耐受性和尿液变色（无害）。
