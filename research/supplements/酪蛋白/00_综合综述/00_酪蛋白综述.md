## 酪蛋白类型（胶束/水解）

### 1. 胶束酪蛋白（Micellar Casein）

胶束酪蛋白是**最接近天然牛奶形式的酪蛋白**，通过微滤工艺从牛奶中分离提取。

**特点**:
- 保留完整的酪蛋白胶束结构
- 溶解性较差，在水中形成浑浊溶液
- 消化速度最慢，缓释效果最佳
- 价格相对较高

**氨基酸谱（每100g蛋白质）**:
- 亮氨酸: ~9.5g
- 必需氨基酸: ~50g
- 支链氨基酸（BCAA）: ~22g

**参考文献**: (Eastwood et al., 1983; Lucey, 2022)

### 2. 水解酪蛋白（Casein Hydrolysate）

水解酪蛋白是**经酶解预消化的酪蛋白**，将蛋白质分解为肽和游离氨基酸。

**特点**:
- 溶解性极佳，无浑浊
- 吸收速度显著快于胶束酪蛋白
- 苦味较重（因游离氨基酸暴露）
- 不具有胶束结构的缓释特性

**与胶束酪蛋白的关键差异**:

| 特性 | 胶束酪蛋白 | 水解酪蛋白 |
|------|-----------|-----------|
| 消化速度 | 慢（6-8小时） | 快（1-3小时） |
| 血氨基酸峰值 | 延迟（3-4小时达峰） | 快速（30-60分钟达峰） |
| 适合时机 | 睡前、长时间禁食 | 运动后快速恢复 |
| 缓释效果 | 极佳 | 无 |

### 3. 酪蛋白酸（Caseinate）

酪蛋白酸是经化学处理（通常用酸或碱）溶解后干燥的酪蛋白，溶解性与水解酪蛋白相似但不含游离氨基酸。

## 缓释机制（酪蛋白凝块）

### 胃酸诱导凝胶机制

酪蛋白的缓释特性源于其独特的**物理化学结构**：

**1. 酪蛋白胶束结构**
- 牛奶中95%的酪蛋白以胶束（micelle）形式存在
- 胶束直径约40-300nm，由αs1-、αs2-、β-和κ-酪蛋白组成
- 胶束内部通过磷酸钙桥连接，形成稳定的三维网状结构

**2. 胃酸诱导凝胶（Acid-Induced Gelation）**
当酪蛋白进入胃部（pH ~1.5-3.0）时：

```
pH下降 → 酪蛋白胶束表面负电荷被中和
       → 胶束稳定性下降
       → 蛋白质分子展开并通过疏水相互作用聚集
       → 形成紧密的凝胶状凝块（curd）
```

**3. 凝块特性**
- 凝块结构致密，胃蛋白酶渗透缓慢
- 胃排空速度显著延迟（乳清蛋白约1小时，酪蛋白凝块可达5-7小时）
- 氨基酸持续缓慢释放入小肠，维持血氨基酸水平6-8小时

**参考文献**: (Boirie et al., 1997; Lucey, 2022)

### 缓释效果的实证研究

Boirie等（1997）在健康受试者中的经典研究显示：

| 蛋白质类型 | 血氨基酸达峰时间 | 峰值持续时间 |
|-----------|----------------|-------------|
| 乳清蛋白 | ~1小时 | 回落至基线约4小时 |
| 酪蛋白 | ~1小时（峰值较低） | 维持 elevated 近7小时 |

## 夜间恢复效果

### 睡眠与肌肉代谢

睡眠期间是肌肉蛋白质合成（MPS）的重要窗口：
- 进食后MPS在2-3小时内回落至基线
- 整夜禁食（7-9小时）导致夜间氨基酸供应中断
- 睡前摄入缓释蛋白可延长夜间氨基酸供应

### 关键研究证据

**1. Trommelen等（2023）- 耐力运动后恢复**

```
研究设计: 随机对照试验，12名健康男性
干预: 睡前30分钟摄入40g酪蛋白 vs 安慰剂
运动: 75分钟耐力运动（65% VO2max）
测量: 线上睡眠期间线粒体蛋白合成率

结果:
- 酪蛋白组线粒体蛋白合成率显著高于安慰剂组
- 肌原蛋白合成率提升约20-25%

参考文献: (Trommelen et al., 2023, Sports Medicine)
DOI: 10.1007/s40279-023-01822-3
```

**2. Snijders等（2015）- 12周随机对照试验**

```
研究设计: 随机、双盲、安慰剂对照
参与者: 44名年轻健康男性
干预: 每晚睡前摄入含27.5g蛋白质的补充剂（50:50胶束:水解酪蛋白混合物）
训练: 每周4次全身抗阻训练，持续12周

结果:
- 股四头肌体积: 酪蛋白组比安慰剂组多增长75%
- II型肌纤维横截面积: 酪蛋白组是安慰剂组的2倍以上
- 力量提升: 两组无显著差异

参考文献: (Snijders et al., 2015, Journal of Nutrition)
⚠️ TODO: 需确认具体DOI
```

**3. Snijders等（2012）- 急性代谢研究**

```
参与者: 16名健康年轻男性
设计: 晚间抗阻训练后，睡前摄入40g酪蛋白 vs 无补充

结果:
- 整夜蛋白质合成: 311±8 vs 246±9 μmol/kg（酪蛋白 vs 安慰剂）
- 净蛋白质平衡: 61±5 vs -11±6 μmol/kg

参考文献: (Snijders et al., 2012)
```

## 酪蛋白 vs 乳清蛋白对比

### 核心差异

| 特性 | 酪蛋白 | 乳清蛋白 |
|------|--------|---------|
| **蛋白质类型** | 慢消化 | 快消化 |
| **消化时间** | 6-8小时 | 1-3小时 |
| **达峰时间** | 3-4小时 | 30-60分钟 |
| **MPS刺激强度** | 中等 | 高（峰值更高） |
| **MPS持续时间** | 长 | 短 |
| **最佳使用时机** | 睡前、长时间禁食 | 运动后即刻 |
| **血糖/胰岛素反应** | 较低 | 较高（乳清蛋白显著升高胰岛素） |

### MPS刺激比较

**Burd等（2012）- 老年人研究**

```
参与者: 14名健康老年男性（72±1岁）
设计: 交叉试验，比较20g乳清蛋白 vs 20g胶束酪蛋白
测量: 单侧腿抗阻运动后4小时MPS

结果:
- 静息MPS: 乳清 0.040%/h vs 酪蛋白 0.024%/h（P=0.002）
- 运动后MPS: 乳清 0.059%/h vs 酪蛋白 0.035%/h（P<0.001）
- 血亮氨酸峰值: 乳清显著高于酪蛋白

结论: 乳清蛋白在老年人中更能最大刺激MPS

参考文献: (Burd et al., 2012, British Journal of Nutrition)
DOI: 10.1017/S0007114511006271
```

⚠️ **关键发现**: 该研究表明**运动后即刻摄入乳清蛋白优于酪蛋白**，因为乳清蛋白能更快提升血氨基酸和亮氨酸水平，更有效地触发MPS。

### 实际应用建议

```
运动后（0-2小时窗口）: 选择乳清蛋白 → 快速提升氨基酸
睡前（30分钟前）: 选择酪蛋白 → 整夜持续供应
日间间隔蛋白摄入: 两者均可，酪蛋白更适合延长饱腹感
```

## 剂量建议（20-40g）

### 基于研究证据的剂量建议

| 人群 | 推荐剂量 | 依据 |
|------|---------|------|
| **一般成年人** | 30-40g/次 | 国际运动营养学会建议；Trommelen等研究使用40g |
| **老年人（>65岁）** | 0.4g/kg体重 | Moore等研究显示该剂量可最大刺激老年人MPS ⚠️ TODO |
| **运动员（增肌期）** | 30-40g/次，每日1-2次 | 综合研究建议 |
| **减脂期** | 30g/次 | 维持肌肉质量，支持饱腹感 |

### 睡前摄入时机

```
推荐时机: 睡前30分钟，与上一餐间隔至少2小时
示例时间表:
  晚餐 - 18:00
  睡前酪蛋白 - 22:00-23:00
  就寝 - 22:30-23:30
```

### 年龄相关的剂量差异

⚠️ **TODO**: Moore等（2015）的研究表明老年人可能需要更高的相对蛋白质摄入量（0.4g/kg/餐）来最大刺激MPS，而年轻人约为0.25g/kg/餐。该研究发表时间较早，需更多2024-2026年研究验证。

### 剂量与MPS的关系

基于现有文献，蛋白质摄入量与MPS的关系呈剂量依赖性：
- 0-20g: 剂量依赖性增加
- 20-30g: 边际收益递减
- >40g: 对MPS无额外益处（但可能支持其他代谢过程）

## 适用人群

### 强烈推荐人群

**1. 有夜间肌肉恢复需求者**
- 抗阻训练运动员
- 力量训练爱好者
- 健身新手到高级训练者

**2. 需要延长蛋白质供应时间者**
- 长时间工作/学习，无法白天多次进食
- 间歇性禁食（IF）实践者
- 夜间工作者

**3. 蛋白质摄入不足者**
- 日间蛋白质摄入分布不均
- 晚餐蛋白质占比过低

### 可能适合人群

**4. 老年人群（需更多验证）**
- ⚠️ TODO: 老年人睡前摄入酪蛋白对肌肉质量影响的长期研究仍然有限
- 合并肌少症风险的老年人可考虑作为综合干预的一部分

**5. 减脂期人群**
- 维持肌肉质量
- 增加饱腹感，减少夜间饥饿感

### 不适合人群

**6. 禁忌或需谨慎人群**
- 乳糖不耐受（虽然酪蛋白乳糖含量低，但仍可能引起反应）
- 牛奶蛋白过敏
- 苯丙酮尿症（PKU）患者
- 肾功能不全患者（需遵循医嘱）

### 特殊人群注意事项

| 人群 | 建议 |
|------|------|
| **孕妇/哺乳期** | 一般安全，但需遵循整体孕期营养指南 ⚠️ TODO |
| **青少年运动员** | 可安全使用，配合训练和整体营养 |
| **糖尿病患者** | 酪蛋白对血糖影响较小，但仍需监测 ⚠️ TODO |

## 研究局限性说明

⚠️ **当前综述的局限性**:

1. **文献检索限制**: 本综述主要依赖2024年前的经典研究和部分2024-2026年文献，**缺乏对2024-2026年新发表大规模RCT的系统检索**

2. **证据缺口**:
   - 胶束酪蛋白 vs 水解酪蛋白直接比较研究
   - 老年人长期（>6个月）睡前酪蛋白干预研究
   - 不同酪蛋白来源（草饲vs传统饲养）对效果的差异
   - 酪蛋白与其他蛋白质来源混合使用的最优配比

3. **适用性限制**: 大多数研究在年轻健康男性中进行，结论外推至女性、老年人、特定疾病患者需谨慎

4. **产业相关性**: 部分经典研究（如Snijders、Trommelen团队）可能与乳制品行业存在资金关联
