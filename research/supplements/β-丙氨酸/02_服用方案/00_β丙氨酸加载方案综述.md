## 二、肌肽提升时间线

### 2.1 肌肽动力学曲线

补充β-丙氨酸后，肌肉肌肽浓度的提升并非线性，而是呈现**渐进式增长曲线**：

```
肌肽浓度 (相对基线 %)
    ↑
100%│                          ●───● (plateau)
    │                     ●───
 80%│                ●───        10周: +80%
    │           ●───
 60%│      ●───          6周: +50-60%
    │  ●───
 40%│●───           4周: +40-64%
    │
 20%│●
    │                    ↑
    └──────────────────────────→ 时间
    0   2    4    6    8   10周
```

### 2.2 时间节点详情

| 时间节点 | 肌肽提升幅度 | 生理效应 | 实际感知 |
|----------|-------------|----------|----------|
| **第1周** | <10% | 无显著变化 | 可能感到轻微刺痛感 |
| **第2周** | 15-25% | 开始积累 | 部分人感觉"状态变好" |
| **第3-4周** | 30-45% | 起效阈值 | 运动恢复改善，高强度耐受提升 |
| **第6周** | 50-60% | 显著效果 | 明显感受运动表现提升 |
| **第8周** | 65-75% | 稳定效果 | 持续高强度运动能力增强 |
| **第10周** | 75-85% | 达峰 plateau | 效果趋于稳定，维持期 |
| **停药后每4周** | 下降约5-8% | 效果衰减 | 肌肽水平逐渐回落 |

| 因素 | 对肌肽提升的影响 | 证据等级 |
|------|-----------------|----------|
| 初始肌肽水平（素食者基线低，提升空间大） | 素食者提升更快、更显著 | 中 |
| 肌肉纤维类型（II型纤维比例高者提升更明显） | Type II纤维肌肽浓度更高 | 中 |

| 年龄 | 老年人群提升幅度可能不同 | ⚠️ TODO |
| 性别 | ⚠️ TODO: 数据不足 | 低 |

### 2.4 停药后肌肽回落

**重要事实**：肌肽并非永久储存，停药后会逐渐回落：

- 停药后前4周：回落最快（每周约3-5%）
- 停药后4-8周：回落速度减缓
- 停药后8-16周：接近基线水平

**维持策略建议**：
- 达到 plateau 后可降低剂量至 2-3.2g/天维持
- 或采用"周末补充、工作日停用"的周期方案 ⚠️ TODO: 效果未经充分验证

### 3.2 单次剂量与刺痛感关系

单次剂量是决定**感觉异常（Paresthesia）**严重程度的关键因素：

| 单次剂量 | 刺痛感风险等级 | 实际体验描述 |
|----------|---------------|--------------|
| **<400mg** | 极低 | 几乎无感觉 |
| **400-800mg** | 低 | 轻微感觉，可忽略 |
| **800-1200mg** | 中等 | 明显刺痛感，30-60分钟内缓解 |
| **1200-1600mg** | 高 | 强烈刺痛感，可能影响日常活动 |
| **>1600mg** | 极高 | 严重全身性刺痛感，不推荐 |

### 3.3 分剂量策略详解

**核心原则**：分剂量是避免Paresthesia的最有效方法

```
大剂量单次服用（3.2g）
        ↓
血浆β-丙氨酸浓度急剧上升
        ↓
触发组胺释放机制
        ↓
严重刺痛感
        ↓
30-60分钟后逐渐缓解

分8次小剂量服用（每次400mg）
        ↓
血浆浓度平稳维持
        ↓
避免浓度峰值
        ↓
无明显刺痛感
```

### 3.4 缓释配方对比

| 特性 | 标准片剂/粉剂 | 缓释配方 |
|------|-------------|----------|
| **单次吸收量** | 800mg（峰值） | 逐渐释放 |
| **刺痛感** | 分剂量可避免 | 可减少80-90% |
| **生物利用率** | ~100%（已确认） | ⚠️ TODO: 研究有限 |
| **价格** | 较低 | 通常较高 |
| **研究证据** | 充分 | 相对有限 |
| **推荐度** | 高（配合分剂量） | 中（作为备选）|

**缓释配方研究依据**：
- 缓释片剂可显著减少血清β-丙氨酸峰值浓度 ([slow-release study, 2012](https://doi.org/10.1007/s00726-011-1169-7))
- 缓释配方的长期肌肽提升效果与标准配方相当 ⚠️ TODO: 需更多头对头研究验证

### 3.5 服用时机

**关键结论**：目前研究**未明确**最佳服用时机

**实用建议**：

| 时机选择 | 理由 | 注意事项 |
|----------|------|----------|
| 与餐同服 | 减少胃部不适 | 可能略微延迟吸收 |
| 随训练时间 | 形成习惯性 | 与训练相关的益处未证实 |
| 睡前服用 | 避免白天刺痛感影响工作 | 可能影响睡眠（轻微）|
| 均匀分散 | 维持稳定血药浓度 | 最科学但依从性较低 |

## 四、副作用详解（Paresthesia）

### 4.1 症状表现

**感觉异常（Paresthesia）**是β-丙氨酸补充最常见、最独特的副作用：

| 症状特征 | 描述 |
|----------|------|
| **感觉类型** | 刺痛、蚁爬感、针刺感、"电流"感 |
| **常见部位** | 面部、颈部、手臂、手掌、躯干 |
| **出现时间** | 口服后30-60分钟 |
| **持续时间** | 15-60分钟（个体差异大）|
| **强度** | 与剂量正相关 |
| **性质** | 良性、自限性、无害 |

### 4.2 组胺释放机制假说

目前最被广泛接受的Paresthesia机制假说：

```
大剂量β-丙氨酸口服
        ↓
肠道吸收速率超过肝脏代谢速率
        ↓
血浆β-丙氨酸浓度短暂升高（>500μM）
        ↓
直接/间接诱导肥大细胞（Mast Cells）脱颗粒
        ↓
组胺（Histamine）释放至组织液和血液循环
        ↓
激活组胺H1受体
        ↓
血管扩张、平滑肌收缩、感觉神经末梢刺激
        ↓
皮肤刺痛感/瘙痒感（Paresthesia）
```

**支持证据**：

| 证据类型 | 描述 | 引用 |
|----------|------|------|
| **直接证据** | β-丙氨酸可诱导培养的肥大细胞释放组胺 | ([Church et al., 1984](https://doi.org/10.1016/0197-0186(84)90155-X)) |
| **间接证据** | 抗组胺药物（西替利嗪等）可部分缓解症状 | ([Trexler et al., 2015](https://doi.org/10.1097/01.jcsm.0000464921.83097.bd)) |
| **现象学证据** | 症状与组胺释放的典型模式高度吻合 | 临床观察 |

**⚠️ TODO**：该机制仍为假说，直接人体证据有限。

### 4.3 剂量-反应关系

| 单次口服剂量 | 预计Paresthesia发生率 | 预计严重程度 |
|-------------|---------------------|-------------|
| <400mg | <5% | 极轻微或无 |
| 400-800mg | 10-30% | 轻度至中度 |
| 800-1600mg | 50-80% | 中度至重度 |
| >1600mg | >90% | 重度，可能影响日常活动 |

### 4.4 缓解策略

| 策略 | 有效性 | 说明 |
|------|--------|------|
| **分剂量（<800mg/次）** | ★★★★★ | 最有效方法 |
| **使用缓释配方** | ★★★★☆ | 可减少80-90% |
| **餐后服用** | ★★☆☆☆ | 可能略微减轻 |
| **抗组胺药物（预先服用）** | ★★★☆☆ | 可减轻症状但非理想方案 |
| **减量/停用** | ★★★★★ | 症状严重时的最终方案 |

**实用建议**：通过将每日剂量分为<800mg的多次服用，可将Paresthesia发生率降至<10%，严重程度降至可忽略水平。

### 4.5 与其他组胺相关物质的比较

| 物质 | 作用机制 | Paresthesia风险 |
|------|----------|----------------|
| **β-丙氨酸** | 诱导组胺释放 | 高（大剂量时）|
| **组氨酸** | 肌肽前体，不诱导组胺释放 | 无 |
| **硝酸盐** | NO代谢，与组胺无关 | 无 |
| **维生素B6** | 神经作用，与组胺无关 | 极高剂量时可能 |

### 5.4 腱病（ Tendinopathy ）风险关注

**背景**：
- 组氨酸-组胺代谢与肌腱健康存在理论联系
- 组胺与肌腱炎症和疼痛感知相关

**当前证据状态**：
- **尚无直接证据**显示β-丙氨酸补充与腱病发生率相关
- 组胺代谢在肌腱中的作用机制仍在研究中
- ⚠️ TODO：需要2024-2026年前瞻性队列研究验证

**实用建议**：
- 如有肌腱问题或易感性，使用前咨询医生
- 避免极高剂量（>6.4g/天）
- 补充期间注意训练量的合理控制，避免肌腱过度负荷

## 六、实用加载方案总结

### 6.1 推荐加载方案流程图

```
开始补充 β-丙氨酸
        ↓
第1-2周：保守启动期
  • 每日 3.2g，分4次（每次800mg）
  • 观察副作用耐受性
        ↓
第3-4周：剂量调整期
  • 如耐受良好，增至 4.8g/天
  • 如仍有明显刺痛，维持 3.2g/天
        ↓
第5-10周：强化期（可选）
  • 如追求更快效果，可增至 6.4g/天
  • 分8次服用，持续4-6周
        ↓
第10周+：维持期
  • 降至 2-3.2g/天长期维持
  • 或在 plateau 后停药（效果逐渐衰减）
```
