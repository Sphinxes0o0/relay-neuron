## 一、作用机制

### 1.1 细胞外缓冲系统

碳酸氢钠 (NaHCO₃) 口服后在胃肠道解离为钠离子 (Na⁺) 和碳酸氢根离子 (HCO₃⁻)：

```
NaHCO₃ → Na⁺ + HCO₃⁻
```

血液中碳酸氢盐浓度升高，增强了细胞外液对氢离子 (H⁺) 的缓冲能力。剧烈运动时，肌糖原无氧酵解产生大量乳酸和 H⁺：

```
乳酸 → 乳酸根 (La⁻) + H⁺
```

增加的细胞外 HCO₃⁻ 可接受 H⁺，形成碳酸并最终以 CO₂ 形式呼出：

```
HCO₃⁻ + H⁺ → H₂CO₃ → CO₂ + H₂O
```

### 1.2 肌肉与血液碳酸氢盐交换

研究表明，补充碳酸氢钠后：
- 肌肉与血浆间乳酸浓度梯度增大
- 促进肌乳酸向血液转运
- 降低肌纤维内酸度
- 延缓肌力下降 ([Zinner et al., 2011](https://link.springer.com/article/10.1007/s00421-010-1800-4))

间歇运动（如 4×30s Wingate 测试）效果较好：
- 促进运动后乳酸清除
- 改善第二次、第三次冲刺表现 ([Zinner et al., 2011](https://link.springer.com/article/10.1007/s00421-010-1800-4))

## 三、最佳补充方案

### 3.1 单剂量方案（推荐）

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **剂量** | 0.3 g/kg 体重 | 最小有效剂量 |
| **摄入时机** | 运动前 60-180 分钟 | 达峰时间约 90 分钟 |
| **服用方式** | 溶于 5 mL/kg 水 | 分次服用减少 GI 刺激 |
| **载体** | 冷水或室温水 | 避免热水分解 |

**示例**（70 kg 运动员）：
- 剂量：21 g 碳酸氢钠
- 溶解于：350 mL 水中
- 时机：运动前 90 分钟

### 3.2 渐进式加载方案（减少 GI 副作用）

| 天数 | 剂量 | 说明 |
|------|------|------|
| 第 1-2 天 | 0.05 g/kg | 适应期 |
| 第 3-4 天 | 0.1 g/kg | 过渡期 |
| 第 5-6 天 | 0.2 g/kg | 累积期 |
| 第 7 天 | 0.3 g/kg | 比赛日剂量 |

文献来源：([Grgic et al., 2018](https://doi.org/10.3390%2Fnu10060697)) - 10 天渐进方案无 GI 副作用报告

### 3.3 多日累积方案

- 0.3 g/kg，连续 3-5 天
- 提升细胞外碳酸氢盐储备稳定性 ⚠️ TODO [需更多 2024-2026 证据]

### 3.4 胶囊 vs 溶液

| 形式 | 优点 | 缺点 |
|------|------|------|
| 溶液 | 吸收更快 | GI 刺激更大 |
| 胶囊 | GI 耐受性更好 | 吸收略慢 |

## 四、胃肠道副作用及缓解策略

### 4.1 常见副作用

| 症状 | 发生率（0.3 g/kg） | 发生率（0.4-0.5 g/kg） |
|------|-------------------|------------------------|
| 腹泻 | 30-50% | 60-80% |
| 腹部痉挛 | 20-40% | 50-70% |
| 恶心 | 15-30% | 40-60% |
| 胃部灼热 | 10-25% | 30-50% |
| 胃气胀 | 20-35% | 45-65% |

### 4.2 缓解策略

1. **分次服用**：将单次剂量分为 2-3 次，间隔 15-20 分钟
2. **随餐服用**：⚠️ TODO [可能影响吸收速率，需验证]
3. **使用胶囊**：延缓释放，减少直接胃刺激
4. **渐进加载**：让胃肠道逐步适应

## 五、与 β-丙氨酸比较

| 维度 | 碳酸氢钠 (NaHCO₃) | β-丙氨酸 |
|------|-------------------|----------|
| **缓冲类型** | 细胞外缓冲 | 细胞内缓冲 |
| **作用部位** | 血浆、组织间液 | 肌纤维内部 |
| **靶分子** | 碳酸氢盐 (HCO₃⁻) | 肌肽 (Carnosine) |
| **提升方式** | 口服直接补充 HCO₃⁻ | 长期补充 β-丙氨酸 (4-8 周) |
| **见效时间** | 1-2 小时 | 4-8 周 |
| **持续时间** | 单次使用 | 长期停用后数周下降 |
| **副作用** | GI 不适 | 皮肤刺痛 (Paresthesia) |
| **最佳运动时长** | 1-7 分钟 | 1-10 分钟（尤其间歇） |
| **协同效应** | 可与 β-丙氨酸联用，有加成效果 |

### 5.1 作用机制差异图示

```
细胞外（血浆）                    肌纤维内部
┌─────────────────┐              ┌─────────────────┐
│   NaHCO₃        │              │   β-丙氨酸      │
│      ↓          │              │      ↓          │
│   HCO₃⁻        │              │   肌肽合成      │
│      ↓          │              │      ↓          │
│ 缓冲 H⁺        │              │  缓冲 H⁺        │
│ (直接快速)      │              │  (缓慢长效)      │
└─────────────────┘              └─────────────────┘
```

### 5.2 联合使用证据

研究表明，同时补充 β-丙氨酸（4 周 6.4 g/天）和碳酸氢钠（0.3 g/kg 比赛前）可产生**加成效应**：

- 柔道/柔术运动员上肢间歇运动表现改善显著优于单独使用 ([Trexler et al., 2015](https://link.springer.com/article/10.1007/s00726-013-1495-z))
- 自行车运动能量系统贡献分析显示协同效应 ([Derave et al., 2018](https://link.springer.com/article/10.1007/s00726-018-2643-2))

## 六、禁忌证与注意事项

### 6.1 绝对禁忌证

| 疾病/状况 | 原因 | 说明 |
|-----------|------|------|
| **严重高血压** | 钠负荷增加血容量 | 可能加剧血压升高 |
| **慢性肾脏病 (CKD)** | 肾脏排泄碳酸氢盐能力受损 | 可能导致代谢性碱中毒 |
| **严重心功能不全** | 钠负荷可能加重水钠潴留 | ⚠️ TODO [需具体阈值] |
| **胃溃疡活动期** | 碱性物质刺激胃黏膜 | 可能诱发出血 |
| **代谢性碱中毒** | 补充会加剧碱中毒 | 需纠正后评估 |

### 6.2 相对禁忌证/需谨慎

| 疾病/状况 | 建议 |
|-----------|------|
| **轻中度高血压** | 监测血压，限制钠摄入总量 |
| **肾功能轻度受损** | 咨询医生，使用最低有效剂量 |
| **胃食管反流** | 随餐服用，避免空腹 |
| **电解质紊乱（低钾、低钙）** | 纠正后使用 |

## 八、实践建议总结

### 8.1 适用人群

**推荐使用**：
- 200m-400m 游泳运动员
- 划船运动员（2000m）
- 自行车计时赛选手（4-20 分钟）
- 需要重复高强度冲刺的运动员
- 力量训练（12-15 RM 多组训练）

**效果不确定**：
- 团体运动运动员（需求多样）
- 超过 10 分钟的耐力运动
- 纯无氧爆发力运动（< 1 分钟）

**不推荐**：
- 高血压患者
- 肾脏疾病患者
- GI 敏感个体（无合适方案时）
