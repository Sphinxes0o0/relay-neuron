# N-乙酰半胱氨酸(NAC)抗氧化与肝脏保护综合综述

## 概述

N-乙酰半胱氨酸（N-Acetyl Cysteine，简称NAC）是半胱氨酸的乙酰化衍生物，是一种具有广泛生物活性的氨基酸类化合物。1956年首次作为祛痰药物开发，1960年代发现其对乙酰氨基酚（扑热息痛）中毒具有解毒作用。

NAC的核心生物活性包括：
- **谷胱甘肽前体**：NAC是细胞内主要抗氧化剂谷胱甘肽（GSH）的前体物质
- **直接抗氧化**：可直接清除活性氧（ROS）
- **黏液溶解作用**：可断裂二硫键，降低黏液黏稠度
- **抗炎作用**：抑制NF-κB等炎症通路

| 属性 | 描述 |
|------|------|
| 化学式 | C₅H₉NO₃S |
| 分子量 | 163.19 Da |
| CAS号 | 616-91-1 |
| 主要用途 | 祛痰、肝脏保护、抗氧化 |

本综述基于2024-2026年最新文献，系统总结NAC在抗氧化、肝脏保护、呼吸系统及运动恢复领域的机制与证据。

## 一、核心作用机制

### 1. 谷胱甘肽前体与抗氧化防御

```
NAC在细胞内的核心作用

细胞外
    ↓ (NAC经脱乙酰化转为半胱氨酸)
半胱氨酸
    ↓
细胞内谷胱甘肽(GSH)合成
    ↓
GSH水平↑ → 抗氧化能力↑
    ↓
活性氧(ROS)清除↑
    ↓
氧化应激↓ → 细胞保护

此外：
NAC → 直接清除ROS
    ↓
补充GSH消耗
```

| 机制 | 作用 |
|------|------|
| GSH前体 | 提供半胱氨酸，维持细胞GSH水平 |
| ROS清除 | 直接与ROS反应，中和自由基 |
| 线粒体保护 | 维持线粒体膜电位，减少线粒体损伤 |
| 蛋白质保护 | 防止巯基蛋白氧化 |

### 2. 黏液溶解作用

```
NAC的黏液溶解机制

NAC分子中的巯基(-SH)
    ↓
断裂黏蛋白中的二硫键(-S-S-)
    ↓
黏蛋白结构崩解
    ↓
黏液黏稠度↓ → 易于咳出

适应症：
- 慢性支气管炎
- 囊性纤维化
- 支气管扩张
- COPD
```

### 3. 抗炎作用

| 靶点 | NAC作用 |
|------|--------|
| NF-κB | 抑制活化，减少促炎因子 |
| MAPK | 调节MAPK信号通路 |
| NLRP3 | 抑制NLRP3炎症小体 |
| TNF-α | 降低TNF-α表达 |

## 二、肝脏保护与解毒作用

### 核心靶点：乙酰氨基酚（APAP）中毒

```
乙酰氨基酚肝毒性机制

APAP过量
    ↓
肝脏代谢超负荷
    ↓
NAPQI（毒性代谢物）↑↑
    ↓
GSH耗竭
    ↓
线粒体蛋白加成物形成
    ↓
线粒体功能障碍
    ↓
ROS爆发 → 细胞死亡
    ↓
肝细胞坏死

NAC解毒机制：
    ↓
补充GSH（直接结合NAPQI）
    ↓
提供替代性巯基供体
    ↓
维持线粒体功能
    ↓
防止肝细胞坏死
```

### 临床证据

| 证据等级 | 应用 | 说明 |
|---------|------|------|
| **强** | APAP中毒一线治疗 | FDA批准，早期使用（8-10小时内）效果最佳 |
| 中等 | 非APAP急性肝衰竭 | 研究中，显示潜在益处 |
| 初步 | 酒精性肝病 | 抗氧化途径可能有益 |
| 初步 | 病毒性肝炎辅助 | 与抗病毒治疗联用 |

### 乙酰氨基酚中毒治疗方案

| 参数 | 方案 |
|------|------|
| 给药途径 | 静脉注射（首选）或口服 |

## 七、安全性与副作用

### 常见副作用

| 副作用 | 发生率 | 处理 |
|--------|--------|------|
| 腥臭味 | 常见 | 随餐服用或选择胶囊 |

| 恶心/呕吐 | 少见 | 减量或停药 |
| 头痛 | 少见 | 一般轻微 |

### 药物相互作用

| 药物 | 相互作用 |
|------|---------|
| 活性炭 | 可能吸附NAC，降低吸收 |

### 禁忌症

- 对NAC或乙酰半胱氨酸过敏者
- 活动性消化道溃疡（口服）

## 九、关键结论

1. **肝脏保护证据最强**：NAC作为乙酰氨基酚中毒的一线解毒药物，证据充分，疗效明确

2. **呼吸系统有价值**：黏液溶解作用明确用于支气管炎、COPD等疾病管理

3. **抗氧化作用理论清晰**：作为谷胱甘肽前体，NAC可提升细胞抗氧化能力，但运动恢复领域证据仍不足

4. **安全性良好**：600-1800mg/日剂量范围耐受性良好，主要副作用为腥臭味和胃肠道不适

5. **与其他抗氧化剂协同**：与维生素C、E等联用有理论依据，但需更多研究验证
