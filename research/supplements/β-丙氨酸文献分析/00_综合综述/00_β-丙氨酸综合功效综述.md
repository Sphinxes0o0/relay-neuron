## 一、作用机制

### 1.1 肌肽合成与缓冲作用

β-丙氨酸是一种**非必需氨基酸**,在肝脏中内源性合成。其核心作用机制如下:

```
β-丙氨酸 + L-组氨酸 → 肌肽 (Carnosine)
         ↑
    肌肽合成酶 (Carnosine Synthase)
```

**关键事实**:
- β-丙氨酸是肌肽合成的**限速前体** ([Sigma-Aldrich Tech Doc](https://www.sigmaaldrich.cn/CN/zh/tech-docs/paper/448642))
- 肌肉中组氨酸含量相对充足,肌肽合成主要受β-丙氨酸 availability 限制
- 肌肽在快速收缩骨骼肌中浓度最高

### 1.2 pH缓冲机制

高强度运动时,氢离子(H⁺)大量堆积导致肌肉pH下降,造成代谢性酸中毒,是疲劳的重要因素:

- 肌肽可缓冲约**40%的**肌肉细胞内氢离子 ([知乎综述](https://zhuanlan.zhihu.com/p/538935755))
- 肌肽的pKa值为**6.88**,使其在骨骼肌pH范围(6.5-7.0)内具有最佳缓冲能力
- 通过提升肌肽水平,可延缓细胞内酸化,延长高强度运动能力

### 1.3 与其他缓冲系统的区别

| 缓冲系统 | 位置 | 作用特点 |
|----------|------|----------|
| 肌肽/氨生素 | 细胞内 | 主要缓冲细胞内H⁺ |
| 碳酸氢钠 | 细胞外 | 主要缓冲细胞外H⁺ |
| 磷酸盐 | 细胞内/外 | 双重作用 |

## 二、运动表现效果

### 2.1 循证效果总结

| 运动类型 | 效果 | 证据强度 |
|----------|------|----------|
| **1-4分钟高强度运动** | 显著改善 | 高 (多项RCT) |
| **重复冲刺能力** | 改善 | 高 |
| **无氧阈值** | 提高 | 中 |
| **最大力量 (1RM)** | ⚠️ TODO: 无明确提升证据 | - |
| **有氧代谢能力** | ⚠️ TODO: 无明确提升证据 | - |
| **HIIT表现** | 改善(理论支持) | ⚠️ TODO: 需更多研究 |
| **VO2max** | ⚠️ TODO: 无直接提升证据 | - |

**最佳效果区间**: 运动时间1-4分钟的开放式终点任务/时间试验 ([ISSN](https://zhuanlan.zhihu.com/p/534911488))

### 2.2 与肌酸的作用机制对比

| 特性 | β-丙氨酸 | 肌酸 |
|------|----------|------|
| **主要机制** | 细胞内pH缓冲 | ATP再生(PCr系统) |
| **提升效果** | 延缓代谢性疲劳 | 提升爆发力和力量 |
| **最佳运动时长** | 1-10分钟高强度 | <30秒全力输出 |
| **肌肉效应** | 提升缓冲能力 | 增加肌肉量和力量 |
| **叠加效果** | 可与肌酸协同 | 可与β-丙氨酸协同 |
| **对最大肌力** | ⚠️ TODO: 无明确提升 | 有提升 |

**重要区分**: β-丙氨酸**不能**提高最大肌力,也不能提高有氧代谢能力,但能够改善无氧阈值 ([知乎](https://zhuanlan.zhihu.com/p/538935755))

### 2.3 运动类型特异性

**理论支持有效的运动**:
- 短距离游泳/跑步 (50-400m)
- 自行车冲刺 (30s-4min)
- 举重/力量举 (1-3分钟全力)
- 团体运动(篮球、足球等)的反复冲刺
- 间歇性高强度训练(HIIT)

- 持续超过25分钟的运动 ([ISSN立场](https://zhuanlan.zhihu.com/p/534911488))
- 纯力量举(1RM测试)

## 三、推荐剂量方案

### 3.1 标准剂量

| 参数 | 推荐值 |
|------|--------|
| **每日剂量** | 4-6克 |
| **最少起效时间** | 2-4周连续补充 |
| **达峰时间** | 10周(肌肽增加~80%) |
| **安全性研究上限** | 24周(3.2g/天) |

### 3.2 分剂量策略(避免副作用)

**问题**: 单次剂量>800mg易引发**感觉异常(Paresthesia)** - 皮肤刺痛/麻木感

**解决方案**:
- 每次剂量控制在**1.6g以下**
- 每日分3-4次服用
- 或使用**缓释配方(SR)**

| 配方类型 | 优点 | 缺点 |
|----------|------|------|
| 快速释放(RR) | 吸收快 | 易引起刺痛 |
| 缓释(SR) | 避免刺痛,持续供给 | 生物利用率研究有限 |

## 四、安全性与副作用

### 4.1 安全性结论

根据现有文献:

> "β-丙氨酸目前被认为是安全的" ([Safety review, 2023](https://link.springer.com/10.1007/s11332-023-01052-0))

**支持证据**:
- 健康人群在推荐剂量(4-6g/天)下安全 ([ISSN](https://zhuanlan.zhihu.com/p/534911488))
- 最长安全研究:24周(3.2g/天) ([Safety review](https://link.springer.com/10.1007/s11332-023-01052-0))
- ⚠️ TODO: 24周以上长期安全性尚无数据

### 4.2 唯一明确副作用: 感觉异常(Paresthesia)

| 症状 | 描述 |
|------|------|
| 表现 | 皮肤刺痛、麻木、麻胀感 |
| 原因 | β-丙氨酸激活MrgD受体 |
| 剂量阈值 | 单次>800mg |
| 持续时间 | 通常短暂(数分钟内) |
| 缓解方法 | 分剂量(<1.6g/次)或缓释配方 |

**注意**: 这种感觉异常**无害**,但可能令人不适

### 4.3 特殊人群

| 人群 | 注意事项 |
|------|----------|
| 素食者/vegan | 可能获益更大(基线肌肽水平较低) |
| 老年人 | 可能有额外神经肌肉保护效果 |
| 运动员 | 可与肌酸等其他补剂协同使用 |

⚠️ TODO: 孕妇、哺乳期、儿童安全性数据不足

## 七、实用建议总结

### 适用于

- 参与1-10分钟高强度运动的运动员
- 需要改善重复冲刺能力的运动员
- 希望延缓神经肌肉疲劳的训练者
- 素食者(肌肽基线水平较低,提升空间大)

### 不适用于 ⚠️ TODO

- 纯力量举运动员(1RM表现)
- 有氧耐力运动员(>25分钟)
- 对麻刺感极度敏感人群

### 服用方案

```
每日剂量: 4-6g
分服次数: 3-4次
单次上限: <1.6g
起效时间: 2-4周
最佳效果: 10周+
```
