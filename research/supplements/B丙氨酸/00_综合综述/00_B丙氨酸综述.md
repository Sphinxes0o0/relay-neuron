## 作用机制（肌肽缓冲系统）

### 1. 肌肽（Carnosine）合成路径

β-丙氨酸是合成肌肽的限速前体：

```
β-丙氨酸 + L-组氨酸 → 肌肽 (Carnosine)
         ↑
    肌肽合成酶 (Carnosine Synthase)
```

**关键事实**:
- β-丙氨酸在肌肉中与L-组氨酸结合生成肌肽
- 肌肽合成酶活性决定肌肽生成速率
- β-丙氨酸是肌肽合成的**限速底物**

### 2. 肌肽的生理功能

| 功能 | 机制 | 运动相关性 |
|------|------|------------|
| **pH缓冲** | 接受H⁺离子→延缓肌肉酸化 | 直接关联运动表现 |
| **抗氧化** | 清除自由基 | 运动后恢复 |
| **金属离子螯合** | 结合Cu²⁺、Zn²⁺ | 代谢调节 |
| **膜稳定** | 保护肌纤维膜 | 损伤防护 |

### 3. 运动中的酸中毒问题

高强度运动时代谢产酸机制：

```
葡萄糖 → 乳酸 + H⁺ (糖酵解)
ATP水解 → H⁺ + Pi (磷酸原系统)
```

## 剂量与时机

### 1. 推荐剂量方案

#### 标准补充方案

| 阶段 | 剂量 | 持续时间 | 目的 |
|------|------|----------|------|
| **加载期** | 3.2–6.4 g/天 | 2-4周 | 快速提升肌肽水平 |
| **维持期** | 0.8–1.6 g/天 | 持续 | 保持肌肽饱和 |

#### 分次服用原则

⚠️ TODO **2024-2026年更新**: 是否存在更优分次方案

```
原因: 单次大剂量 → 瞬时高浓度 → Tingling/刺痛
```

**分次方案示例**（以4.0g/天为例）:

## 实践建议总结

### 适用人群

| 推荐程度 | 人群 | 说明 |
|----------|------|------|
| **强推荐** | 高强度运动1-4分钟者（如400-800m跑） | 证据最强 |
| **中推荐** | HIIT运动员、功能性训练者 | 理论获益 |
| **弱推荐** | 长时间耐力运动员（>10分钟） | 证据有限 |
| **不推荐** | 单次最大力量表现为主者 | 证据不支持 |

### 使用建议

1. **剂量**: 3.2-6.4 g/天，分2-4次服用
2. **疗程**: 至少4周评估效果
3. **副反应管理**: 分次服用可减少tingling
4. **监测**: 记录运动表现变化
