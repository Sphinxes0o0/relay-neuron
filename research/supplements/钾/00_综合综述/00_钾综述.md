## 2. 钾的生理功能

### 2.1 体内分布与稳态

钾（K⁺）是体内含量最丰富的细胞内阳离子。正常成年人总体钾含量约为 50-55 mmol/kg 瘦体重，其中约 **98%** 分布于细胞内，细胞外钾浓度仅 3.5-5.0 mmol/L。

**关键数值**:
- 细胞内钾浓度: 350-400 mmol/L
- 细胞外钾浓度: 3.5-5.0 mmol/L
- 血清钾浓度参考范围: 3.5-5.0 mmol/L
- 正常24小时尿钾排泄: 25-125 mmol/天

### 2.2 静息膜电位与动作电位

钾离子是维持神经和肌肉细胞静息膜电位的核心离子（约 -90 mV）。根据 **Nernst 方程**：

> E_K = (RT/zF) × ln([K⁺]_out/[K⁺]_in)

动作电位期间，电压门控钠通道开放导致去极化，随后钾通道开放（延迟整流钾通道 I_Kr 和 I_Ks）引起 **钾外流**，导致膜复极化。这一过程对以下功能至关重要：

- 骨骼肌收缩启动
- 心肌动作电位平台期后的复极化
- 神经冲动传导

### 2.3 肌肉收缩与舒张机制

1. **神经-肌肉接头去极化** → 肌膜动作电位
2. **横管系统（T-tubule）传播** → 肌质网（SR）钙离子释放
3. **Ca²⁺-肌钙蛋白结合** → 肌丝滑行（收缩）
4. **SERCA泵主动回收 Ca²⁺** → 肌质网 → 舒张
5. **钾外流复极化** → 膜电位恢复，准备下一次收缩

### 2.4 蛋白质合成与糖代谢

钾在细胞内参与多个关键代谢过程：

- **蛋白质合成**: 氨基酸转运和核糖体功能需要钾依赖性过程
- **糖原合成**: 1 g 糖原储存伴随约 0.45 mmol 钾进入细胞（胰岛素效应）
- **糖酵解**: 多种激酶活性受细胞内钾浓度影响

### 2.5 血管舒张与血压调节

钾离子通过以下机制调节血管张力：

- 激活 Na⁺/K⁺-ATPase，增加平滑肌细胞超极化
- 促进一氧化氮（NO）释放
- 抑制肾素-血管紧张素-醛固酮系统（RAAS）

## 3. 运动员钾流失

### 3.1 出汗途径

汗液是运动员钾丢失的最主要途径。汗钾浓度个体差异显著，受以下因素影响：

| 影响因素 | 汗钾浓度变化趋势 |
|----------|------------------|
| 训练强度 | 强度增加 → 汗钾浓度升高 |
| 热适应 | 热适应 → 汗钾浓度降低（肾脏保钾效率提高） |
| 个体差异 | 范围: 2.5-9.5 mmol/L |
| 饮食钾摄入 | 低钾饮食 → 汗钾浓度降低（代偿性） |
| 汗液收集部位 | 前臂 vs 躯干有显著差异 |

### 3.2 典型丢失量估算

| 运动类型 | 持续时间 | 估计汗液丢失 | 估计汗钾丢失 |
|----------|----------|-------------|-------------|
| 耐力运动（马拉松） | 3-4 小时 | 3-8 L | 300-1,200 mg |
| 足球比赛 | 90 分钟 | 1.5-3 L | 150-450 mg |
| 间歇性高强度训练 | 1 小时 | 0.5-1.5 L | 75-225 mg |
| 热环境下训练 | 2 小时 | 2-4 L | 200-600 mg |

⚠️ TODO — 需2024-2026年实测数据验证以上估算值，建议参考最近代谢平衡研究。

**文献依据**:
- ([Shirreffs et al., 2005](https://pubmed.ncbi.nlm.nih.gov/16286466/)) 报道足球运动员90分钟比赛平均汗钾丢失为 280 mg
- ([Luo et al., 2022](https://pubmed.ncbi.nlm.nih.gov/35861347/)) 报告热适应后汗钾浓度从 6.2 mmol/L 降至 4.8 mmol/L

### 3.3 运动期间的钾释放

运动过程中，活跃骨骼肌纤维会向细胞外释放钾，这一现象在以下过程中尤为重要：

- **肌纤维去极化**: 动作电位传播过程中钾外流
- **代谢性钾释放**: 运动诱导肌糖原分解伴随钾离子释放
- **高强度运动效应**: 重复性动作电位导致细胞外钾积累（[Juel et al., 2000](https://pubmed.ncbi.nlm.nih.gov/10956337/)）

⚠️ TODO — 需2024-2026年研究验证细胞外钾积累与运动性疲劳的因果关系。

### 3.4 低钾血症风险

运动员低钾血症（血清 K⁺ < 3.5 mmol/L）的常见原因：

1. **大量出汗**（最常见）
2. **腹泻或呕吐**
3. **使用利尿剂**
4. **过度补水**（稀释性低钾血症）
5. **低碳水化合物/高钠饮食**
6. **长时间低强度运动**（如长距离耐力项目伴随过度出汗）

## 4. 剂量建议

### 4.1 一般人群 vs 运动员

| 人群 | 推荐的膳食钾摄入量 |
|------|-------------------|
| 一般成年人（美国） | 2,600-3,400 mg/天 |
| 一般成年人（中国） | 2,000 mg/天 |
| 运动员 | 4,000-5,500 mg/天 |
| 高温/潮湿环境训练 | 5,500-6,000 mg/天 |

### 4.2 钾的食物来源

|------|-------------|------|

| 菠菜 | 558 | 草酸影响吸收 |

| 椰子水 | 475 | 电解质天然来源 |
| 牛油果 | 485 | 健康脂肪来源 |
| 三文鱼 | 420 | 蛋白质+钾双重来源 |
| 酸奶 | 255 | 乳制品来源 |

### 4.3 补钾制剂

| 制剂类型 | 含钾量 | 吸收率 | 注意事项 |
|----------|--------|--------|----------|
| 氯化钾片 | 8-10 mmol/片 | ~90% | 空腹吸收更佳，胃肠道刺激 |
| 柠檬酸钾 | 8.5 mmol/片 | ~90% | 胃肠道耐受性较好 |
| 磷酸钾 | 10.8 mmol/片 | ~85% | 需注意磷负荷 |
| 复合电解质粉 | 100-200 mg/包 | ~85% | 运动中补充首选 |

⚠️ TODO — 需2024-2026年研究比较不同钾制剂在运动前、中、后的吸收效率和胃肠耐受性。

### 4.4 补充时机建议

**运动前（1-2小时）**:
- 200-400 mg 钾（通过食物或低剂量补剂）
- 配合 300-400 mL 水

**运动中（>60分钟）**:
- 每小时补充 200-300 mg 钾
- 通过运动饮料（复合电解质）补充

**运动后（2小时内）**:
- 优先通过食物补充（促进糖原再合成和蛋白质合成）
- 目标: 300-400 mg 钾 + 1.0-1.2 g碳水化合物/kg体重

## 5. 安全性

### 5.1 高钾血症

**诊断标准**:
- 血清钾 > 5.5 mmol/L: 轻度
- 血清钾 > 6.0 mmol/L: 中度
- 血清钾 > 6.5 mmol/L: 重度（心律失常风险显著增加）

**常见原因**:
- 肾功能不全（排泄障碍）
- 钾制剂过量补充
- 血管紧张素转换酶抑制剂（ACE-I）或 ARB 类药物
- 保钾利尿剂联用
- 组织损伤（横纹肌溶解）
- 代谢性酸中毒

### 5.3 药物相互作用

| 药物类别 | 相互作用机制 |
|----------|-------------|
| ACE抑制剂（依那普利等） | 减少醛固酮 → 降低钾排泄 |
| ARB类（氯沙坦等） | 同上 |
| 保钾利尿剂（螺内酯） | 直接拮抗醛固酮 |
| NSAIDs（布洛芬等） | 抑制前列腺素 → 减少肾钾排泄 |
| 肝素 | 抑制肾上腺醛固酮合成 |
| 环孢素 | 减少肾钾排泄 |

### 5.4 肾功能评估

**重要提示**: 在建议运动员补钾前，应常规评估肾功能：

- **血肌酐**: 正常范围 0.7-1.3 mg/dL
- **估算肾小球滤过率（eGFR）**: > 90 mL/min/1.73m² 为正常
- **尿蛋白/肌酐比**: 排除隐匿性肾损伤

### 5.5 安全补充原则

1. **优先食物来源**: 每日摄入钾的 80% 以上应来自天然食物
2. **避免高浓度单次补充**: 单次补钾不超过 400 mg（以减少胃肠刺激和高钾血症风险）
3. **分次补充**: 将补钾均匀分配至各餐
4. **监测基础指标**: 定期检测血清钾和肾功能
5. **个体化调整**: 根据训练环境、出汗量、饮食结构个体化制定方案
6. **特殊人群谨慎**: 有高血压、肾脏疾病、心脏疾病家族史者补钾前应咨询医生
