## 2. 化学结构差异

### 2.1 L-瓜氨酸（L-Citrulline）

```
NH2-C(=O)-NH-CH(CH2-CH2-NH-C(=O)-OH)-COOH
```

- **分子式**：C₆H₁₃N₃O₃
- **分子量**：175.2 g/mol
- **化学分类**：非蛋白质氨基酸（条件必需氨基酸）
- **化学特性**：
  - 白色结晶粉末
  - 溶于水，不溶于有机溶剂
  - pI（等电点）~5.9

### 2.2 瓜氨酸苹果酸（Citrulline Malate, CM）

- **组成**：L-瓜氨酸 + 苹果酸（Malic acid）以盐形式结合
- **常见比例**：**2:1**（2分子瓜氨酸 : 1分子苹果酸）

```
L-Citrulline + Malic Acid → Citrulline Malate
```

- **分子量**：约256.3 g/mol（按2:1比例计算）
  - L-瓜氨酸：175.2 g/mol × 2 = 350.4
  - 苹果酸：134.1 g/mol
  - 总计：~484.5 → 实际盐形式分子量取决于结合方式

### 2.3 苹果酸（Malic Acid）的作用

苹果酸是 Krebs 循环（柠檬酸循环）的中间产物：

```
苹果酸 → 草酰乙酸 → 柠檬酸 → α-酮戊二酸 → ...
```

**潜在效益**（⚠️ TODO: 大部分尚需人体研究验证）：
- 可能支持细胞能量产生
- 可能减少运动中乳酸积累
- 可能改善运动后肌肉恢复

## 4. 吸收率与生物利用度

### 4.1 吸收机制

**L-瓜氨酸吸收**：
- 主要通过小肠上皮细胞的**中性氨基酸转运蛋白（LAT1, LAT2）**吸收
- 吸收率较高，单次摄入后血浆浓度峰值出现在 **30-60分钟**
- 口服生物利用度：⚠️ TODO: 不同研究报道差异较大（30-90%），需标准化测定方法

**CM吸收**：
- 理论上苹果酸可能促进某些氨基酸转运蛋白的活性
- 但尚无直接比较研究证明CM吸收优于纯L-瓜氨酸
- ⚠️ TODO: 需要独立药代动力学研究比较两种形式的吸收曲线

### 4.2 代谢路径

```
口服 L-瓜氨酸/CM
       ↓
    肠道吸收
       ↓
   门静脉循环
       ↓
   肝脏代谢
       ↓
  转化为 L-精氨酸
       ↓
  精氨酸 → NO合成酶 → 一氧化氮（NO）
       ↓
  精氨酸 → 精氨酸酶 → 鸟氨酸 → 脯氨酸/多胺
```

## 7. 推荐选择

### 7.1 决策框架

```
目的明确性
    │
    ├── 提高运动表现（力量/耐力）
    │       → 推荐：L-瓜氨酸 6-8g/天
    │
    ├── 追求最大"泵感"
    │       → 推荐：L-瓜氨酸 8-10g/天（运动前）
    │
    ├── 预算有限但需要性价比
    │       → 推荐：L-瓜氨酸（纯品）
    │
    ├── 对纯瓜氨酸口感/胃耐受性差
    │       → ⚠️ TODO: 可尝试CM，但需调整剂量
    │
    └── 预配方产品用户（已有CM在内）
            → ⚠️ 注意总瓜氨酸摄入量，避免过量
```

### 7.2 最终推荐

| 用户类型 | 推荐选择 | 理由 |
|---------|---------|-----|
| **大多数健身爱好者** | **L-瓜氨酸（纯品）** | 更高瓜氨酸含量、更好性价比、更明确证据基础 |
| **专业运动员** | **L-瓜氨酸（纯品）** 配合第三方检测 | 剂量精确、避免杂质、符合反兴奋剂要求 |
| **特定医疗目的** | ⚠️ **咨询医生后决定** | 需要个体化评估 |

### 7.3 使用建议

1. **起始剂量**：从 3-4 g/天开始，观察胃肠道耐受性
2. **目标剂量**：6-8 g/天（运动前30-60分钟服用）
3. **监测指标**：
   - 主观泵感变化
   - 运动表现（力量/耐力）
   - 恢复速度
4. **产品选择**：
   - 选择提供**第三方检测认证**的品牌
   - 优先选择**纯L-瓜氨酸**产品
   - ⚠️ 如选择CM，确认实际比例并相应调整剂量
