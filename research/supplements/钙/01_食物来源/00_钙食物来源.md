# 钙的食物来源与生物利用度

> **文件信息**
> - **创建日期**: 2026-05-05
> - **研究领域**: 营养学 / 矿物质代谢
> - **关键词**: calcium, food sources, bioavailability, dairy, plant calcium, vitamin D

---

## 摘要

钙是人体含量最多的矿物质，约99%储存于骨骼和牙齿中。钙的生物利用度因食物基质、伴随营养素及抗营养因子而差异显著。本文件系统综述钙的食物来源、生物利用度影响因素及运动员特殊需求，为运动营养实践提供参考依据。

## 2. 乳制品 vs 植物钙源

### 2.1 乳制品钙源特点

**优势**:
- 天然钙含量高且分布均匀
- 乳糖促进小肠钙吸收（乳糖酶活性正常者）
- 蛋白质基质有助于钙保持可溶性状态
- 含有促进骨健康的乳清蛋白和酪蛋白肽段
- 维生素D强化奶（美国市场普遍）进一步提升利用率

**生物利用度数据**:
- 牛奶钙吸收率: 约30-40% ([Avioli et al., 1975](https://doi.org/10.1001/archinte.1975.00330010069008))
- 奶酪钙吸收率: 约30-40% (略低于牛奶，因加工过程)
- 酸奶钙吸收率: 约30-40% (发酵可能轻微改善)

⚠️ TODO: 需2024-2026年乳制品钙利用度系统综述验证

### 2.2 植物钙源特点与挑战

**主要挑战**:

1. **草酸盐(Oxalate)抑制**
   - 草酸与钙形成不溶性草酸钙复合物
   - 菠菜、甜菜叶、瑞士甜菜草酸含量高→钙几乎不可利用
   - 羽衣甘蓝、白菜草酸含量低→钙吸收率相对较高

2. **植酸盐(Phytate)抑制**
   - 存在于全谷物、豆类、坚果种子中
   - 螯合钙及其他二价阳离子(铁、锌)
   - 传统发酵/浸泡方法可降低植酸含量

3. **纤维干扰**
   - 不可溶性纤维可物理结合钙
   - 高纤维饮食可能降低钙吸收

**植物钙源改善策略**:
| 方法 | 适用食物 | 效果 |
|------|---------|------|
| 浸泡 | 豆类、坚果 | 降低植酸20-50% |
| 发芽 | 豆类、谷物 | 植酸酶激活，降解植酸 |
| 发酵 | 豆类、谷物 | 显著降低植酸 |
| 钙盐凝固 | 豆腐 | 提高钙含量3-5倍 |
| 强化 | 植物奶、果汁 | 直接提高可利用钙量 |

⚠️ TODO: 2024-2026年植物基钙补充剂生物利用度比较研究

### 2.3 乳制品vs植物钙源综合比较

| 指标 | 乳制品 | 植物钙源 |
|------|--------|---------|
| 钙含量稳定性 | 高 | 变异大 |
| 生物利用度 | 30-40% | 5-40% (取决于种类) |
| 蛋白质贡献 | 高质量完整蛋白 | 部分完整蛋白 |
| 维生素D协同 | 常有强化 | 需额外补充 |
| 饱和脂肪 | 中等 (奶酪) | 无 |
| 乳糖不耐受适用性 | 低 | 高 |
| 素食/纯素适用 | 否 | 是 |
| 成本 | 中等 | 中等-高 |

## 3. 吸收影响因素

### 3.1 促进钙吸收的因素

#### 3.1.1 维生素D

**作用机制**:
- 维生素D₃(胆钙化醇) → 肝脏25-羟化 → 肾脏1α-羟化 → 活性形式1,25-(OH)₂D₃
- 1,25-(OH)₂D₃诱导小肠上皮细胞合成钙结合蛋白(CaBP-D9k)
- 促进TRPV6钙通道表达
- 增强肠上皮细胞对钙的跨细胞主动转运

**推荐摄入量** (Institute of Medicine 2011):
- 19-70岁成人: 600 IU/日 (15 μg)
- 71岁以上: 800 IU/日 (20 μg)
- 运动员建议: ⚠️ TODO: 需2024-2026年研究验证是否需要更高剂量

**阳光照射与维生素D合成**:
- 紫外线B (UVB, 290-315 nm) 照射皮肤7-脱氢胆固醇
- 影响因素: 纬度、季节、时间、皮肤色素、衣物覆盖、防晒霜
- 高纬度地区(>37°)冬季可能无法有效合成

#### 3.1.2 乳糖

- 乳糖酶分解乳糖产生葡萄糖和半乳糖
- 酸性环境有助于钙溶解和吸收
- 乳糖不耐受者钙吸收率可能下降10-15%

#### 3.1.3 氨基酸

- 赖氨酸和精氨酸促进钙吸收
- 赖氨酸→与钙形成可溶性氨基酸-钙复合物
- 高蛋白饮食(适量)可能正向影响钙吸收

#### 3.1.4 一些有机酸

- 柠檬酸、苹果酸: 与钙形成可溶性复合物
- 水果中的有机酸可能有轻微协同作用

### 3.2 抑制钙吸收的因素

#### 3.2.1 草酸盐(Oxalate)

**化学性质**:
- 草酸(H₂C₂O₄)与钙形成CaC₂O₄·H₂O (草酸钙)
- 溶解度极低，在生理pH下几乎不溶解

**高草酸食物** (钙生物利用度<10%):
- 菠菜: 草酸含量 600-1000 mg/100g
- 甜菜叶: 草酸含量 300-500 mg/100g
- 瑞士甜菜: 草酸含量 300-600 mg/100g
- 瑞士甜菜叶: 草酸含量 300-900 mg/100g
- 可可/巧克力: 草酸含量 200-500 mg/100g

⚠️ TODO: 2024-2026年草酸钙生物利用度精确测定研究

#### 3.2.2 植酸盐(Phytate)

**化学性质**:
- 肌醇六磷酸酯 (myo-inositol-1,2,3,4,5,6-hexakisphosphate)
- 强螯合剂，与Ca²⁺、Fe²⁺、Zn²⁺等离子形成不溶性复合物

**高植酸食物**:
- 全麦面粉: 植酸含量 300-800 mg/100g
- 糙米: 植酸含量 200-500 mg/100g
- 豆类(黄豆、扁豆): 植酸含量 500-1500 mg/100g
- 坚果(杏仁、核桃): 植酸含量 200-500 mg/100g

**降低植酸的方法**:
1. 浸泡(12-24小时) → 去除20-30%植酸
2. 发芽(2-5天) → 植酸酶降解植酸，可降低50-80%
3. 发酵(24-72小时) → 微生物植酸酶作用，降低可达90%
4. 减压蒸汽处理 → 破坏植酸，保留营养素

⚠️ TODO: 2024-2026年发酵降低植酸的系统评价

#### 3.2.3 膳食纤维

- 不溶性纤维(纤维素、木质素)物理结合钙
- 可溶性纤维(果胶、树胶)也可能结合钙
- 高纤维摄入(>30g/日)可能降低钙吸收效率10-15%

#### 3.2.4 高磷摄入

- 磷酸盐与钙竞争吸收
- 可乐等磷酸饮料大量饮用可能影响钙平衡
- 加工食品中的磷酸盐添加剂值得关注

#### 3.2.5 咖啡因

- 咖啡因(约100mg/杯)轻微增加尿钙排出
- 适量咖啡(≤3杯/日)对钙平衡影响有限
- 大量饮用(>5杯/日)可能需额外关注钙摄入

#### 3.2.6 钠

- 高钠摄入增加尿钙排出
- 每增加1000mg钠摄入，尿钙增加约20-40mg
- 高盐饮食人群需注意钙摄入

### 3.3 吸收影响因素汇总表

| 因素 | 方向 | 影响程度 | 备注 |
|------|------|---------|------|
| 维生素D | 促进 | 强 | 最主要调节因子 |
| 乳糖 | 促进 | 中 | 仅适用于乳糖耐受者 |
| 蛋白质(适量) | 促进 | 中 | 氨基酸协同 |
| 草酸盐 | 抑制 | 强 | 形成不溶性复合物 |
| 植酸盐 | 抑制 | 强 | 螯合作用 |
| 膳食纤维(高) | 抑制 | 中 | 物理结合 |
| 高磷(加工食品) | 抑制 | 中 | 竞争吸收 |
| 咖啡因 | 抑制 | 轻-中 | 适量影响小 |
| 高钠 | 抑制 | 中 | 增加钙排出 |

### 4.2 运动员钙需求

#### 影响因素

1. **训练强度与频率**
   - 高强度训练→应力性骨折风险↑
   - 耐力运动→骨质流失风险↑

2. **能量平衡状态**
   - 热量限制/减重期: 钙利用效率可能下降
   - 女性运动员三联征(低能量+月经紊乱+骨质疏松)风险

3. **出汗率**
   - 出汗含钙约20-60 mg/L
   - 高温环境训练运动员钙流失增加

4. **骨应激损伤史**
   - 应力性骨折史: 需更高钙摄入(1200-1500 mg/日)

#### 运动员推荐摄入量

| 运动员类型 | 推荐钙摄入量 | 备注 |
|-----------|-------------|------|
| 一般运动人群(中等强度) | 1000 mg/日 | 同一般成人 |
| 高强度训练/竞技运动员 | 1200-1500 mg/日 | ⚠️ TODO: 2024-2026年研究验证 |

| 绝经后女性运动员 | 1200 mg/日 | 骨质流失加速期 |
| 素食/纯素运动员 | 1200-1500 mg/日 | 需注意植物钙生物利用度 |

## 5. 运动员骨骼健康需求

### 5.1 运动对骨骼的双向效应

**机械负荷的正向作用**:
- 骨骼是活组织，对机械负荷具有适应性
- 负重运动(跑步、跳跃、抗阻训练)刺激成骨细胞活性
- 高冲击运动(如体操、篮球、排球)促进骨密度增加
- 研究显示: 竞技网球运动员击球侧肱骨 BMD 高 15-20% ([Ducher et al., 2009](https://doi.org/10.1359/jbmr.081265))

**过度训练的负向作用**:
- 能量不足+训练负荷过大→骨吸收>骨形成
- 频率、强度、持续时间需与营养摄入匹配

### 5.2 运动员骨健康风险人群

| 风险人群 | 风险因素 | 建议 |
|---------|---------|------|
| 耐力运动员(跑步、自行车) | 低体脂、能量限制 | 监测骨密度 |
| 女性耐力运动员 | 月经紊乱、低雌激素 | 三联征筛查 |
| 男性低体重运动员 | 热量限制 | 评估能量状态 |
| 负重运动不足的运动员 | 训练单一化 | 增加交叉训练 |
| 青少年运动员 | 骨骼矿化关键期 | 确保充足钙和维生素D |
| 素食/纯素运动员 | 植物钙生物利用度低 | 个体化营养方案 |

### 5.3 骨骼健康综合策略

**营养要素**:

1. **钙** (核心元素)
   - 推荐: 1200-1500 mg/日(高风险运动员)
   - 分布: 每日3-4次，每次≤500mg(提高吸收效率)
   - 富含钙+维生素D食物优先

2. **维生素D** (关键协同)
   - 推荐: 1500-2000 IU/日 (部分运动员可能需要更高)
   - 监测: 25(OH)D血清浓度
   - 目标: >30-40 ng/mL (75-100 nmol/L)
   - ⚠️ TODO: 2024-2026年运动员维生素D需求系统评价

3. **维生素K2** (骨钙素羧化)
   - 推荐: 90-120 μg/日
   - 来源: 纳豆、奶酪、蛋黄
   - 作用: 促进骨钙素活化，钙沉积于骨骼
   - ⚠️ TODO: 2024-2026年维生素K2与运动表现研究

4. **蛋白质** (骨基质)
   - 推荐: 1.2-1.6 g/kg体重/日
   - 分布: 均匀分配至各餐
   - 亮氨酸特别重要

5. **镁** (骨骼矿物化)
   - 推荐: 400-420 mg/日(男), 310-320 mg/日(女)
   - 来源: 坚果、全谷物、绿叶蔬菜

**运动要素**:

1. **负重运动** (每周3-5次)
   - 每次30-60分钟
   - 多方向运动(前后、左右、旋转)

2. **抗阻训练** (每周2-3次)
   - 大负荷多关节动作(深蹲、硬拉、挺举)
   - 渐超负荷原则

3. **避免过度训练**
   - 训练量/强度渐增<5-10%/周
   - 充足恢复(48-72小时)

### 5.4 运动员钙补充剂使用建议

**何时考虑补充**:

- 饮食钙摄入不足(<800mg/日)
- 乳糖不耐受或素食/纯素饮食
- 高温环境大量出汗
- 应力性骨折史
- 骨密度低下(T-score < -1.0)
- 青少年生长发育期

**补充剂选择**:

| 类型 | 吸收率 | 优点 | 缺点 |
|------|-------|------|------|
| 碳酸钙 | 30-40% (随餐) | 含钙量高(40%)、便宜 | 需胃酸活化，可能便秘 |
| 柠檬酸钙 | 30-40% (空腹可用) | 胃酸非必需、易吸收 | 含钙量低(21%)、价格较高 |
| 磷酸钙 | ~30% | 接近骨骼矿物成分 | 吸收一般 |
| 乳酸钙 | ~30% | 温和 | 含钙量低(13%) |

**使用注意事项**:
- 单次不超过500-600mg元素钙
- 碳酸钙随餐服用效果更佳
- 柠檬酸钙可空腹服用
- 与铁、锌补充剂分开服用(至少间隔2小时)
- 避免与高草酸/高植酸食物同时摄入

---

## 6. 结论与建议

### 6.1 关键结论

1. **钙的食物来源差异显著**: 乳制品提供最高生物利用度的钙，但植物基钙源在正确选择和加工后可达到相近效果。

2. **生物利用度是关键**: 仅关注"钙含量"不够，含钙量高的食物(如菠菜)因草酸含量高，实际可利用钙可能很低。

3. **维生素D是核心调节因子**: 无论钙摄入量多高，维生素D不足将严重影响钙吸收效果。

4. **运动员需求更高**: 高强度训练者、青少年运动员、素食运动员等特殊人群需个体化钙营养方案。

### 6.2 实践建议

**一般人群**:
- 优先选择低草酸蔬菜(羽衣甘蓝、白菜、西兰花)
- 乳制品为最佳钙源(如能耐受)
- 注意维生素D状态，必要时补充
- 分散钙摄入(每日3次)提高吸收效率

**运动员**:
- 评估个人钙摄入是否达标
- 关注骨健康标志物(骨密度、月经功能)
- 营养与训练负荷匹配，避免能量不足
- 必要时在运动营养师指导下使用补充剂
