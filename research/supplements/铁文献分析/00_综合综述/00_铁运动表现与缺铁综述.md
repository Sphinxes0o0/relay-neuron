## 铁的生理功能

### 1. 氧气运输

#### 1.1 血红蛋白 (Hemoglobin)

- 每个血红蛋白分子含4个亚铁血红素基团
- 每个亚铁血红素含1个Fe²⁺原子，可结合1个O₂分子
- 健康成年女性血红蛋白浓度: 120-160 g/L
- 运动员安静时总血红蛋白量约800g

#### 1.2 肌红蛋白 (Myoglobin)

- 位于骨骼肌细胞内
- 每个肌红蛋白含1个亚铁血红素
- 功能: 储存和释放氧气给线粒体
- 峰值氧输送能力依赖肌红蛋白浓度

### 2. 氧化酶功能

铁是多种氧化酶的关键成分:

| 酶 | 功能 | 缺铁影响 |
|----|------|----------|
| 细胞色素C (Cytochrome C) | 电子传递链 | ATP生成↓ |
| 琥珀酸脱氢酶 | 柠檬酸循环 | 能量代谢↓ |
| NADH脱氢酶 | 电子传递链 | 运动能力↓ |

### 3. 铁稳态调节

#### 铁调素 (Hepcidin)

- 肝脏分泌的肽类激素
- 功能: 负向调节肠道铁吸收和巨噬细胞铁释放
- 调控机制:
  - 铁储备增加 → 铁调素↑ → 铁吸收↓
  - 炎症/感染 → 铁调素↑ → 铁吸收↓
  - 运动后铁调素可升高20-60%，持续6-24小时

## 铁缺乏连续体

铁缺乏是一个渐进过程，分为三个阶段:

### 阶段1: 铁储备耗竭 (Depleted Iron Stores)

```
指标变化:
├── 血清铁蛋白 (Ferritin): ↓ (<30 μg/L)
├── 骨髓可染铁: 减少/消失
├── 血红蛋白: 正常 (≥120 g/L)
└── 临床表现: 无明显症状
```

### 阶段2: 非贫血性缺铁 (Non-Anemic Iron Deficiency)

```
指标变化:
├── 血清铁蛋白: ↓ (<20 μg/L)
├── 血清铁: ↓
├── 转铁蛋白饱和度: ↓ (<20%)
├── 游离原卟啉: ↑
├── 血红蛋白: 正常 (≥120 g/L)
└── 临床表现: 疲劳、注意力下降、运动恢复减慢
```

### 阶段3: 缺铁性贫血 (Iron Deficiency Anemia)

```
指标变化:
├── 血红蛋白: ↓ (<120 g/L，女性运动员<115 g/L)
├── 血细胞比容: ↓
├── MCV: ↓ (<80 fL，小细胞性)
├── MCH: ↓ (<27 pg，红细胞低色素性)
├── 血清铁蛋白: ↓ (<15 μg/L)
└── 临床表现: 严重疲劳、苍白、VO2max显著下降
```

## 女性运动员的特殊风险

### 1. 月经失血

- 每次月经失铁量: 约0.5-1.0 mg/天
- 月经量多者可达2.0 mg/天
- 约30-50%女性运动员存在月经相关铁丢失增加

### 2. 运动诱导的铁丢失

| 机制 | 说明 |
|------|------|
| 机械性溶血 (足击溶血) | 长跑运动员足部反复撞击导致红细胞破坏 |
| 胃肠失血 | 剧烈运动减少内脏血流，致肠道铁丢失 |
| 出汗失铁 | 汗液含铁约0.1-0.2 mg/L |
| 尿失血 | 运动性血尿，红细胞破坏 |
| 炎症反应 | 运动诱导IL-6升高 → 铁调素↑ |

### 3. 能量限制与饮食因素

- 女性运动员更常见低碳水/低热量饮食
- 植物性饮食（非血红素铁吸收率低: 2-20%）
- 茶、咖啡中的鞣酸抑制非血红素铁吸收

### 4. 铁调素敏感性

- 耐力运动后铁调素升高更显著
- 马拉松运动员运动后铁调素可升高100%以上
- 高强度训练期间铁调素持续偏高

## 对耐力表现的影响

### 1. 氧气运输能力下降

```
缺铁 → 血红蛋白↓ → 动脉血氧含量↓
     → VO2max下降 (每下降1g/dL Hb，VO2max约下降1-2%)
```

### 2. 骨骼肌氧化能力受损

- 肌红蛋白减少 → 线粒体氧供应↓
- 细胞色素氧化酶活性↓ → 电子传递链效率↓

### 3. 研究证据

| 研究类型 | 发现 | 结论 |
|----------|------|------|
| 横断面研究 | IDA运动员VO2max比铁充足运动员低8-15% | ⚠️ TODO |
| 补铁干预研究 | 缺铁运动员补铁4周后VO2max提高5-10% | ⚠️ TODO |
| Meta分析 | 补铁对铁缺乏运动员耐力表现有显著改善作用 | ⚠️ TODO |

### 4. 阈值效应

- Hb从140→130 g/L: VO2max下降约3-5%
- Hb从130→120 g/L: VO2max下降约5-8%
- Hb<115 g/L: 显著影响有氧能力

## 检测与诊断

### 1. 关键生物标志物

| 指标 | 正常范围 (女性运动员) | 缺铁阈值 | 临床意义 |
|------|----------------------|----------|----------|
| **血清铁蛋白** | 30-200 μg/L | <30 μg/L | 反映铁储备最敏感指标 |
| **转铁蛋白饱和度** | 20-50% | <20% | 反映循环铁可用性 |
| **血清铁** | 9-27 μmol/L | <9 μmol/L | 瞬时铁状态 |
| **血红蛋白** | 120-160 g/L | <120 g/L | 贫血诊断 |
| **MCV** | 80-100 fL | <80 fL | 小细胞性贫血 |
| **MCH** | 27-33 pg | <27 pg | 低色素性贫血 |
| **可溶性转铁蛋白受体 (sTfR)** | 2.9-8.5 mg/L | >8.5 mg/L | 组织缺铁指标 |

### 2. 运动员铁状态评估建议

```
筛查频率:
├── 常规: 每3-6个月
├── 高风险运动员 (耐力项目、月经量多): 每月
└── 症状出现时: 立即检测

评估流程:
1. 血清铁蛋白 (Ferritin)
   ├── ≥30 μg/L: 铁储备充足
   ├── 15-29 μg/L: 轻度缺铁
   ├── <15 μg/L: 显著缺铁
   │
2. 结合血红蛋白 (Hb)
   ├── Hb正常 + Ferritin低: 非贫血性缺铁
   └── Hb低 + Ferritin低: 缺铁性贫血
```

## 铁剂形式比较

### 1. 硫酸亚铁 (Ferrous Sulfate)

| 特性 | 评价 |
|------|------|
| 生物利用率 | 参考标准，吸收率约10-15% |
| 元素铁含量 | 20% |
| 胃肠道刺激 | 较高 (20-30%出现恶心、便秘) |
| 成本 | 低 |
| 备注 | 最常用，但耐受性较差 |

### 2. 甘氨酸亚铁 (Ferrous Bisglycinate)

| 特性 | 评价 |
|------|------|
| 生物利用率 | 较高，比硫酸亚铁高2-3倍 |
| 元素铁含量 | 20% |
| 胃肠道刺激 | 较低 |
| 成本 | 较高 |
| 备注 | 氨基酸螯合形式，对胃肠道更温和 |

### 3. 葡萄糖酸亚铁 (Ferrous Gluconate)

| 特性 | 评价 |
|------|------|
| 生物利用率 | 中等，约10% |
| 元素铁含量 | 12% |
| 胃肠道刺激 | 中等 |
| 成本 | 中等 |
| 备注 | 常用于儿童液体制剂 |

## 副作用与吸收增强

### 1. 常见副作用

| 副作用 | 发生率 | 缓解方法 |
|--------|--------|----------|
| 胃部不适/恶心 | 15-30% | 随餐服用、改用缓释制剂 |
| 便秘 | 10-25% | 增加膳食纤维、饮水、使用甘氨酸亚铁 |
| 腹泻/稀便 | 5-15% | 随餐服用、减量 |
| 黑色粪便 | 100% | 正常现象，无需处理 |
| 牙齿变色 | 少见 | 使用吸管服用液体铁剂 |

### 2. 吸收增强策略

#### 维生素C (抗坏血酸)

- 机制: 将Fe³⁺还原为可吸收的Fe²⁺
- 推荐: 每30mg元素铁配100-200mg维生素C
- 效果: 吸收率提高2-4倍
- 来源: 柑橘类水果、草莓、 peppers
- 服用时间: 与铁剂同服

#### 其他增强因素

| 物质 | 机制 | 效果 |
|------|------|------|
| 肉因子 (Meat Factor) | 促进非血红素铁吸收 | +50-100% |
| 乳酸 | 降低肠道pH | +30-50% |
| 植乳杆菌Lp299v | 代谢产物促进铁吸收 | ⚠️ TODO - 需要更多研究 |

### 3. 吸收抑制剂

| 物质 | 机制 | 建议 |
|------|------|------|
| 鞣酸 (茶、咖啡) | 形成不溶性铁复合物 | 铁剂前后1-2小时避免 |
| 钙 | 竞争性吸收 | 铁钙分开服用 |
| 植酸 (全谷物、豆类) | 螯合铁 | 发酵处理、补充维生素C |
| 草酸 (菠菜、大黄) | 形成不溶性草酸铁 | 避免大量食用 |
