## 摘要

维生素K2（甲萘醌，Menaquinone）是一类具有多不饱和侧链的脂溶性维生素，主要通过激活维生素K依赖性蛋白（VKDPs）发挥生物学功能。本文件系统综述2024-2026年间关于维生素K2在骨健康维护与心血管保护方面的研究进展，重点分析MK-4、MK-7、MK-9等不同形式的生物学特性差异，探讨骨钙素羧化机制、心血管保护证据、维生素D协同作用、最佳剂量及膳食来源等关键问题。

## 1. K2形式比较：MK-4 vs MK-7 vs MK-9

### 1.1 化学结构特征

维生素K2是一系列甲萘醌化合物的总称，其侧链长度（异戊二烯单元数目）决定具体形式：

| 形式 | 侧链碳原子数 | 主要存在形式 | 半衰期 |
|------|-------------|-------------|--------|
| MK-4 | 4个异戊二烯单元 | 组织储存、转化生成 | 约1-2小时 |
| MK-7 | 7个异戊二烯单元 | 发酵食品、肠道菌群 | 约72小时 |
| MK-9 | 9个异戊二烯单元 | 肠道菌群产物 | 较长 |

**来源**: ([Halder et al., 2019](https://pubmed.ncbi.nlm.nih.gov/31329205/)) ⚠️ TODO: 确认2024-2026年最新结构研究

### 1.2 MK-4特性分析

**MK-4（甲萘醌-4）** 是维生素K2的短链形式：

- **生物利用度**: 快速吸收但代谢清除较快
- **组织分布**: 主要分布于肝脏、肾脏、骨骼等组织
- **转化来源**: 可由肠道微生物将K1转化为MK-4
- **主要功能**: 激活骨钙素（osteocalcin）和基质Gla蛋白（MGP）
- **剂量要求**: 由于半衰期短，需要较高剂量（毫克级）才能维持有效血药浓度

### 1.3 MK-7特性分析

**MK-7（甲萘醌-7）** 是目前研究最广泛的长链形式：

- **生物利用度**: 长半衰期（约72小时）提供更稳定的血药浓度
- **组织分布**: 优先蓄积于血管和骨骼组织
- **主要来源**:
  - 纳豆（日本传统发酵大豆食品）
  - 某些奶酪（如荷兰高达奶酪）
  - 肠道细菌（特别是产正义球菌属）
- **有效性**: 在较低剂量（微克级）即可有效激活VKDPs

**临床证据**: Atiei等（2025）在Long COVID随机对照试验中使用240 µg/天纯MK-7进行干预，显示积极的免疫调节效应 ([Atiei et al., 2025](https://pubmed.ncbi.nlm.nih.gov/")) ⚠️ TODO: 补充完整PMID

### 1.4 MK-9特性分析

- **来源**: 主要由肠道厌氧菌产生
- **特性**: 侧链最长，脂溶性最高
- **研究深度**: 相对较少，多为观察性研究
- **潜在优势**: 可能在心血管组织中有更长停留时间 ⚠️ TODO: 验证此假设

### 1.5 形式比较总结

```
MK-4: 短半衰期 + 需要高剂量 + 广泛组织分布
MK-7: 长半衰期 + 低剂量有效 + 血管/骨骼优先蓄积 ← 当前研究热点
MK-9: 长半衰期 + 脂溶性高 + 研究相对有限
```

**结论**: MK-7因其在低剂量下的高效性和稳定的长半衰期，目前被认为是膳食补充最具优势的形式 ([Knapen et al., 2013](https://pubmed.ncbi.nlm.nih.gov/23934675/))。

## 2. 骨钙素羧化机制

### 2.1 维生素K依赖性蛋白（VKDPs）概述

维生素K2的核心生物学功能是作为γ-谷氨酰羧化酶的必需辅因子，催化特定蛋白质分子中谷氨酸（Glu）残基向γ-羧基谷氨酸（Gla）残基的转化。

**主要骨相关VKDPs**:

1. **骨钙素（Osteocalcin, OCN）**: 骨骼中含量最丰富的非胶原蛋白质
2. **基质Gla蛋白（Matrix Gla Protein, MGP）**: 抑制血管和软组织钙化
3. **骨保护素（Osteoprotegerin）**: 调节破骨细胞分化 ⚠️ TODO: 确认MGP在骨代谢中的直接作用

### 2.2 骨钙素羧化机制详解

```
步骤1: 前骨钙素合成
   骨母细胞合成前骨钙素（前OCN），包含3个Glu残基

步骤2: 维生素K依赖性羧化
   γ-谷氨酰羧化酶使用维生素K2作为辅因子
   将Glu残基转化为Gla残基
   反应需要: O₂、CO₂、还原型维生素K和K2

步骤3: 羧化骨钙素形成
   完全羧化的骨钙素（cOCN）具有高钙离子亲和力
   cOCN能够:
   - 与羟基磷灰石结合，促进骨矿化
   - 激活胰岛素敏感性（内分泌功能）
   - 调节能量代谢

步骤4: 未羧化骨钙素（ucOCN）
   维生素K不足时，产生低活性或无活性的ucOCN
   ucOCN无法有效结合钙质，导致骨矿化效率下降
```

### 2.3 骨钙素的双重功能

| 功能类型 | 羧化状态 | 生物学作用 |
|---------|---------|-----------|
| **骨矿化** | 完全羧化（cOCN） | 促进羟基磷灰石沉积，增强骨强度 |
| **内分泌** | 低羧化（部分活性） | 调节胰岛素敏感性、葡萄糖代谢、肌肉功能 |

### 2.5 钙化抑制机制

**基质Gla蛋白（MGP）** 是血管钙化的关键抑制剂：

```
MGP抑制钙化机制:
1. 直接结合羟基磷灰石晶体，阻止其继续增长
2. 结合骨形态发生蛋白（BMP），抑制其促钙化活性
3. 维持血管平滑肌细胞的成骨表型转化平衡

MGP需要维生素K2依赖性羧化才能发挥活性:
- 未羧化MGP（ucMGP）无抗钙化功能
- 循环中ucMGP水平可作为维生素K状态的生物标志物
```

## 4. 维生素D协同作用

### 4.1 协同机制概述

维生素K2与维生素D在骨代谢和钙稳态中具有多层次协同作用：

```
维生素D → 促进肠道钙吸收 → 血钙升高
    ↓
维生素K2 → 激活骨钙素 → 促进钙沉积于骨骼
    ↓
协同效应: 维持血钙平衡，优化骨矿化
```

### 4.2 分子机制

**骨钙素基因表达调控**:
- 维生素D response element (VDRE) 存在于骨钙素基因启动子区域
- 维生素D3可上调骨钙素基因转录
- 维生素K2作为辅因子确保新合成骨钙素的羧化活化

**基质蛋白协同**:
- MGP同样受维生素D调控
- 维生素D增强MGP表达，维生素K2确保其羧化激活

**已知研究**:
- 联合补充K2和维生素D3可能比单独补充更能改善骨代谢标志物
- 在骨质疏松管理中，K2+D3联合方案日益受到关注

### 4.4 钙剂补充情境

现代营养学建议在钙剂补充时同步考虑维生素D和K2：

| 镁 | 参与维生素K依赖性羧化酶活性调节 |

## 7. 研究结论与未来方向

### 7.1 当前证据总结

| 领域 | 证据等级 | 结论 |
|-----|---------|-----|
| K2形式比较 | 中等 | MK-7因长半衰期和低剂量高效性成为首选补充形式 |
| 骨钙素羧化 | 强 | 机制明确，K2对骨矿化至关重要 |
| 骨密度改善 | 中等-有限 | 绝经后女性有一定益处，其他人群证据有限 |
| 心血管保护 | 中等（观察性） | 膳食K2与心血管事件负相关 |
| 血管钙化抑制 | 中等 | RCT支持AVC进展减缓 |
| K2+D3协同 | 有限 | 机制合理，临床证据待加强 |

### 7.2 知识缺口（⚠️ TODO清单）

1. **大型RCT缺口**: 尚无以心血管事件（死亡/心梗/卒中）为一级终点的大型K2干预RCT
2. **MK-9研究不足**: 长链MK-9的独特优势未被充分阐明
3. **精准剂量**: 不同人群（年龄、性别、疾病状态）的最佳K2剂量未确定
4. **联合干预**: K2+D3+钙+镁的最佳配比缺乏头对头研究
5. **种族差异**: 亚洲人群对K2补充的反应是否与西方人群不同
6. **长期安全性**: 5年以上持续补充的安全性数据缺乏

### 7.3 未来研究方向

1. 开展K2心血管结局大型多中心RCT
2. 深入研究MK-9的独特生物学功能
3. 开发基于ucMGP/ucOCN的个体化营养方案
4. 探索K2与其他营养素的协同最佳配比
5. 研究肠道菌群对K2内源性合成的影响
