## 2. 一氧化氮(NO)生理机制概述

### 2.1 NO合成途径

```
L-精氨酸 ──[eNOS/iNOS]──→ L-瓜氨酸 + NO
                ↑
        (需要NADPH、四氢生物蝶呤、FAD、FMN)

口腔细菌 ──[硝酸盐→亚硝酸盐]──→ NO (非酶途径)
```

### 2.2 NO的生理作用

| 靶组织 | 主要作用 | 生理效应 |
|--------|---------|---------|
| 血管内皮 | 血管舒张 | 血压下降、血流增加 |
| 骨骼肌 | 代谢调节 | 葡萄糖摄取、线粒体功能 |
| 神经元 | 神经传递 | 肌肉协调、血流调节 |
| 免疫细胞 | 抗菌/抗炎 | 免疫防御 |

### 2.3 NO与运动表现的关系

- **有氧耐力**: NO通过改善氧输送和线粒体功能提升耐力表现
- **力量训练**: NO通过增强血流输送营养物质，可能改善"肌肉泵感"
- **恢复**: NO介导的血管舒张可能加速运动后代谢废物清除

## 3. 各成分详细比较

### 3.1 L-精氨酸 (L-Arginine)

**化学式**: C₆H₁₄N₄O₂
**每日推荐剂量**: 3-6 g (运动前30-60分钟)
**口服利用度**: 40-70% ⚠️ TODO 需要个体差异数据

#### 作用机制
L-精氨酸是NO合酶(NOS)的底物，理论上直接增加NO合成。然而，实际效果受限于：
- 首过代谢(肠道和肝脏)
- eNOS活性依赖性
- 年龄相关eNOS功能下降

#### 关键研究
- Bode-Boger SM等(2003)证实L-精氨酸可改善内皮功能 ([DOI: 10.1016/S0140-6736(03)12491-4](https://pubmed.ncbi.nlm.nih.gov/12689573/))
- Tessari P等(2011)综述: L-精氨酸对代谢综合征患者有轻度降压作用 ⚠️ TODO 检索2024年后补充

#### 优点
- 直接前体，机制明确
- 研究历史长，安全性数据充分

#### 缺点
- 口服利用度低
- 大剂量(>10g)可能引起胃肠道不适
- 对有训练经验的个体效果较差

---

### 3.2 L-瓜氨酸 (L-Citrulline)

**化学式**: C₆H₁₃N₃O₃
**每日推荐剂量**: 6-8 g L-瓜氨酸 或 8-10 g 瓜氨酸苹果酸(Citrulline Malate, CM)
**口服利用度**: >80% (显著优于L-精氨酸)

#### 作用机制
```
L-瓜氨酸 →(肾脏)→ L-精氨酸 →(NOS)→ NO
```
瓜氨酸绕过肝脏首过代谢，在肾脏转化为精氨酸，从而更有效提升血浆精氨酸水平。

#### 关键研究
- Perez-Guisado J等(2013)发现瓜氨酸苹果酸(8g)增强肌肉耐力 ([DOI: 10.1007/s00726-012-1348-1](https://pubmed.ncbi.nlm.nih.gov/22505345/))
- Trexler ET等(2015)系统综述: 瓜氨酸补充对运动表现的益处 ([DOI: 10.1519/JSC.0000000000000650](https://pubmed.ncbi.nlm.nih.gov/25333291/))
-Gonzalez AM等(2020)证实CM 2:1比例(8g)提升肌肉做功能力 ⚠️ TODO 核查2024年后续研究

#### 优点
- 口服利用度显著高于L-精氨酸
- 不引起胃肠道不适(相较于高剂量精氨酸)
- 可持续提升血浆精氨酸水平(半衰期较长)

#### 缺点
- 机制为间接NO提升
- 对静息血压的影响较L-精氨酸弱 ⚠️ TODO 需要更多RCT验证

---

### 3.3 甜菜根 (Beetroot / Beetroot Juice)

**活性成分**: 硝酸盐 (NO₃⁻)，含量约250-500 mg/100g鲜重
**每日推荐剂量**: 70-90 mL浓缩汁 或 500 mL标准汁 (含硝酸盐 ~300-500 mg)
**摄入时间**: 运动前2-3小时

#### 作用机制
```
甜菜根硝酸盐 →(口腔细菌)→ 亚硝酸盐 →(低pH/维生素C)→ NO
```
这是**非酶途径**，不依赖NOS活性，因此对老年人和内皮功能障碍个体特别有益。

#### 关键研究
- Larsen FJ等(2007)首次证实甜菜根硝酸盐降低运动耗氧量 ([DOI: 10.1152/japplphysiol.00135.2007](https://pubmed.ncbi.nlm.nih.gov/17360160/))
- Jones LW等(2014)系统综述: 甜菜根对运动经济的改善效果 ([DOI: 10.1249/MSS.0000000000000264](https://pubmed.ncbi.nlm.nih.gov/24435466/))
- 2024-2026年文献 ⚠️ TODO: 检索"beetroot nitrate exercise 2024 2025"获取最新RCT

#### 优点
- 效果可预测(硝酸盐含量标准化)
- 非酶途径，不依赖内皮功能
- 有强力证据支持有氧耐力提升

#### 缺点
- 可能受口腔细菌群差异影响
- 对力量/无氧运动效果有限
- 味道特殊，部分人群不耐受

### 3.4 碧容健 (Pycnogenol)

**来源**: 法国海洋松树皮提取物 (Pinus pinaster)
**每日推荐剂量**: 100-200 mg
**标准规格**: 含60-70%原花青素

#### 作用机制
- 抑制NOS降解，增强NO生物利用度
- 抗氧化作用，保护NO免于失活
- 改善内皮依赖性血管舒张

#### 关键研究
- Fitzpatrick DF等(1998)证实碧容健激活eNOS ([DOI: 10.1016/S0014-2999(98)00663-9](https://pubmed.ncbi.nlm.nih.gov/9822840/))
- Badia R等(2019)系统综述: 碧容健对血压和内皮功能的有益效果 ([DOI: 10.1016/j.phymed.2018.09.207](https://pubmed.ncbi.nlm.nih.gov/30439591/))
- ⚠️ TODO: 2024-2026年是否有碧容健与运动表现相关的高质量RCT?

#### 优点
- 多靶点协同作用
- 强大的抗氧化功效
- 安全性数据良好

#### 缺点
- 对运动表现的直接证据有限 ⚠️ TODO
- 价格相对较高
- 标准化程度较补充剂低

## 4. 运动表现效果

### 4.1 有氧耐力

| 成分 | 效果大小 | 效应量(ES) | 证据质量 |
|------|---------|-----------|---------|
| L-精氨酸 | 微弱/不一致 | ES 0.1-0.2 | 中等 ⚠️ TODO |
| L-瓜氨酸 | 中等 | ES 0.3-0.5 | 良好 |
| 甜菜根 | **显著** | ES 0.5-1.0 | 强 |
| 碧容健 | ⚠️ TODO | 未知 | 初步 |

**机制解释**:
- 甜菜根硝酸盐通过改善氧输送和降低ATP消耗提升运动经济性
- L-瓜氨酸通过增强氨代谢和延缓疲劳提升耐力

### 4.2 血流/肌肉泵感 (Muscle Pump)

| 成分 | 急性效果 | 慢性效果 | 主观感受 |
|------|---------|---------|---------|
| L-精氨酸 | 中等 | 低 | ⚠️ TODO |
| L-瓜氨酸 | 中等 | 中等 | 较一致 |
| 甜菜根 | 低-中等 | ⚠️ TODO | 不如有氧效果明显 |
| 碧容健 | 中等 | 良好 | 主要改善内皮功能 |

⚠️ TODO: "泵感"主观感受是否与实际血流增加相关联? 2024年后是否有客观测量研究?

### 4.3 力量与无氧运动

**结论**: NO增强剂对纯力量/无氧运动表现的效果有限或不一致。

- 甜菜根对短时间冲刺( < 60秒 )无显著益处 ([DOI: 10.1249/MSS.0000000000000264](https://pubmed.ncbi.nlm.nih.gov/24435466/))
- L-瓜氨酸可能通过延缓氨积累改善重复冲刺能力 ⚠️ TODO 需要更多验证

## 7. 实际应用建议

### 7.1 基于目标的推荐

#### 目标: 提升有氧耐力 (马拉松、单车、长跑)

**首选**: 甜菜根/甜菜根硝酸盐
- 证据最强，效果最一致
- 推荐剂量: 500 mg 硝酸盐等价物，运动前2-3小时

**次选**: L-瓜氨酸 (如不耐受甜菜根)
- 剂量: 6-8 g/天

#### 目标: 提升力量训练泵感

**首选**: L-瓜氨酸苹果酸
- 8 g CM (运动前30分钟)
- 主观泵感改善较一致

**辅助**: 可考虑碧容健 (长期内皮健康)

⚠️ TODO: 泵感是否转化为实际肌肉生长? 需要更多肌肥大RCT数据。

#### 目标: 改善恢复/血流

**首选**: 碧容健 (长期) + L-瓜氨酸 (运动后)

### 7.2 周期化策略

| 训练阶段 | 推荐补充策略 |
|---------|------------|
| 基石期(基础建设) | 碧容健 100-200 mg/天 + 甜菜根 3次/周 |
| 强化期(高强度) | 甜菜根 (运动日) + L-瓜氨酸 (运动前) |
| 减量期(恢复) | 暂停NO增强剂，侧重整体营养 |

### 7.3 实用决策树

```
你有氧训练为主?
├── 是 → 甜菜根硝酸盐 (A级证据)
└── 否 → 你力量训练想要泵感?
    ├── 是 → L-瓜氨酸苹果酸
    └── 否 → 碧容健或L-瓜氨酸(日常血管健康)
```
