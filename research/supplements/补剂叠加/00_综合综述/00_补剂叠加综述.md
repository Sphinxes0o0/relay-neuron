## 补剂叠加原理

### 1. 药代动力学协同 (Pharmacokinetic Synergy)

补剂通过不同代谢途径发挥作用，叠加使用可产生1+1>2的效果：

```
吸收阶段 ──→ 分布阶段 ──→ 代谢阶段 ──→ 排泄阶段
    ↑            ↑            ↑            ↑
  多种补剂    靶向组织    不同通路    减少毒性
  同时摄入    协同作用    互补促进
```

**核心原理**:
- **吸收协同**: 某些补剂可增强肠道吸收率（如维生素C促进铁吸收）
- **靶向协同**: 不同补剂作用于同一组织的不同细胞机制
- **代谢通路互补**: 避免竞争同一代谢酶，扩大效应范围

### 2. 生理机制协同 (Physiological Synergy)

| 生理过程 | 相关补剂 | 协同机制 |
|---------|---------|---------|
| 肌肉蛋白质合成 (MPS) | 亮氨酸 + 肌酸 + HMB | 同时激活mTOR通路 |
| ATP再合成 | 肌酸 + 碳水化合物 | 加速磷酸肌酸恢复 |
| 抗氧化防御 | 维生素C + 维生素E + 辅酶Q10 | 协同清除自由基 |
| 抗炎反应 | Omega-3 + 姜黄素 + HMB | 多靶点炎症调节 |

### 3. 时间窗口叠加 (Temporal Stacking)

根据不同时相选择补剂组合：

- **训练前 (30-60分钟)**: 咖啡因 + 硝酸盐 + β-丙氨酸 → 提升运动表现
- **训练中**: 支链氨基酸 + 电解质 → 减少肌肉分解
- **训练后 (黄金窗口)**: 乳清蛋白 + 肌酸 + 亮氨酸 → 最大化合成代谢
- **睡前**: 酪蛋白 + HMB → 持续的夜间肌肉修复

## 常见组合与证据强度

### 组合一：增肌三角 (Muscle Building Triangle)

**组合**: 肌酸 + 乳清蛋白 + HMB

**推荐剂量**:
- 肌酸: 5g/天 (一水肌酸)
- 乳清蛋白: 25-40g/次，训练后
- HMB: 3g/天

**作用机制**:

1. **肌酸** → 增加骨骼肌磷酸肌酸储备 → 提升ATP再合成速率
2. **乳清蛋白** → 提供必需氨基酸 → 激活mTOR通路
3. **HMB** → 抑制泛素-蛋白酶体途径 → 减少肌肉蛋白分解

**科学依据**:

在一项针对抵抗训练男性的双盲RCT中，单独使用肌酸或HMB时瘦体重分别增加1.9kg和1.3kg，而两者联用时增加3.4kg，显示出明显的协同效应 ([J Appl Physiol, 2003](https://doi.org/10.1152/japplphysiol.01005.2002)) ⚠️ TODO: 需2024-2026重复验证

---

### 组合二：力量表现堆叠 (Strength Performance Stack)

**组合**: 肌酸 + β-丙氨酸 + 咖啡因

**推荐剂量**:
- 肌酸: 5g/天
- β-丙氨酸: 3-6g/天 (分次服用)
- 咖啡因: 3-6mg/kg 体重，训练前30分钟

**作用机制**:

```
肌酸 ──→ 磷酸肌酸系统 ──→ 短时高强度能量
   │
   └──→ 细胞水合作用 ──→ 蛋白质合成环境

β-丙氨酸 ──→ 肌肽合成 ──→ 缓冲H+ ──→ 延缓疲劳
   │
   └──→ 抗氧化特性

咖啡因 ──→ 中枢神经刺激 ──→ 运动单位募集
   │
   └──→ 磷酸二酯酶抑制 ──→ cAMP积累
```

**科学依据**:

系统综述显示，肌酸+β-丙氨酸联合使用相比单独使用，对1RM深蹲成绩的提升幅度增加约15-20% ([Sports Med, 2016](https://doi.org/10.1007/s40279-016-0487-z)) ⚠️ TODO: 需2024-2026大型RCT验证

---

### 组合三：减脂保存堆叠 (Fat Loss Preservation Stack)

**组合**: 蛋白质 + 咖啡因 + HMB + 绿茶提取物

**推荐剂量**:
- 蛋白质: 2-2.5g/kg体重/天
- 咖啡因: 200-400mg/天
- HMB: 3g/天
- 绿茶提取物 (EGCG): 500-600mg/天

**作用机制**:

| 补剂 | 减脂机制 | 证据 |
|-----|---------|-----|
| 蛋白质 | 食物热效应高(20-30%)，提升饱腹感 | 强 |
| 咖啡因 | 提升基础代谢3-11%，增强脂肪氧化 | 强 |
| HMB | 抑制肌肉分解，保存瘦体重 | 中等 |
| 绿茶EGCG | 抑制COMT，增强去甲肾上腺素作用 | 中等 |

**科学依据**:

Tinsley等(2017)的Meta分析显示，咖啡因+绿茶提取物组合可使体脂率降低额外0.5-1.2% ([Obesity Reviews](https://doi.org/10.1111/obr.12509)) ⚠️ TODO: 需2024-2026长期研究验证

---

### 组合四：耐力表现堆叠 (Endurance Stack)

**组合**: 硝酸盐 + β-丙氨酸 + 咖啡因 + 铁

**推荐剂量**:
- 甜菜根汁 (硝酸盐): 500ml (≈300mg硝酸盐) 或 6-8mmol/L
- β-丙氨酸: 3-6g/天
- 咖啡因: 3-6mg/kg
- 铁: 8-18mg/天 (视铁蛋白水平)

**作用机制**:

1. **硝酸盐 → NO → 血管舒张 → 肌肉血流↑ → 氧输送↑**
2. **β-丙氨酸 → 肌肽 → 细胞pH缓冲 → 延缓疲劳**
3. **咖啡因 → CNS刺激 → 感知疲劳降低**
4. **铁 → 血红蛋白/肌红蛋白合成 → 氧携带能力↑**

**科学依据**:

多项RCT证实，甜菜根汁+β-丙氨酸组合可延长力竭时间约15-25% ([J Int Soc Sports Nutr, 2019](https://doi.org/10.1186/s12970-019-0284-5)) ⚠️ TODO: 需2024-2026验证最优配比

---

### 组合五：恢复加速堆叠 (Recovery Stack)

**组合**: 乳清蛋白 + 谷氨酰胺 + 支链氨基酸 + Omega-3

**推荐剂量**:
- 乳清蛋白: 20-25g，训练后
- 谷氨酰胺: 5-10g/天
- 支链氨基酸(BCAA): 5-10g训练中/后
- Omega-3: 2-3g EPA+DHA/天

**作用机制**:

```
训练后炎症反应
       ↓
┌──────┼──────┐
↓      ↓      ↓
Omega-3  谷氨酰胺  蛋白质
(抗炎)  (肠屏障)  (MPS)
       ↓
   恢复加速
```

**科学依据**:

研究发现，训练后30分钟内摄入蛋白质+BCAA相比单独摄入蛋白质，肌肉蛋白质合成率提升约15% ([Am J Clin Nutr, 2017](https://doi.org/10.3945/ajcn.116.143289)) ⚠️ TODO: 需2024-2026验证长期效果

## 协同效应机制

### 1. 分子信号通路的交叉对话

**mTOR通路协同**:

```
亮氨酸 ──→ mTORC1 ──→ 蛋白质翻译起始
    +                    +
肌酸 ──→ AMPK ──→       |
    ↓                    ↓
    └──────→ 蛋白质合成增强
```

**关键发现**:
- 肌酸通过提升细胞渗透压为蛋白质合成创造有利环境
- 亮氨酸直接激活mTORC1
- 两者联合使用对mTOR通路的激活程度大于单独使用

### 2. 代谢底物循环互补

```
葡萄糖代谢 ←────────────→ 脂肪酸代谢
    ↑                        ↑
  咖啡因                    咖啡因
    ↑                        ↑
  肌酸 ──→ ATP储备 ──→ 能量供应
```

### 3. 神经递质系统协同

```
咖啡因 ──→ 腺苷受体拮抗 ──→ 觉醒提升
    +
L-酪氨酸 ──→ 多巴胺/去甲肾上腺素合成 ──→ 注意力↑
    +
Omega-3 ──→ 神经膜流动性 ──→ 神经传递效率↑
```

- 蛋白质: 1.6-2g/kg

## 安全性注意事项

### 1. 已知相互作用

| 补剂组合 | 潜在问题 | 处理方式 |
|---------|---------|---------|
| 高剂量咖啡因 + 肌酸 | 可能增加脱水风险 | 确保充足水分摄入 |
| 铁 + 钙 | 吸收竞争 | 分开服用 (间隔2h+) |
| 银杏 + 抗凝药 | 出血风险增加 | 避免或咨询医生 |
| HMB + 他汀类药物 | 横纹肌溶解风险 | 监测肌酸激酶 ⚠️ TODO |

### 2. 禁忌人群

- **肌酸**: 肾功能异常者慎用
- **β-丙氨酸**: 皮肤瘙痒为正常反应，非过敏
- **咖啡因**: 心脏病患者、高血压患者慎用
- **HMB**: 孕妇/哺乳期安全性未确定

### 3. 最大安全剂量参考

| 补剂 | 最大安全剂量 | 备注 |
|-----|-------------|-----|
| 肌酸 | 20g/天 (负荷期) / 5g/天 (维持) | 长期使用安全性良好 |
| 咖啡因 | 400mg/天 (健康成人) | 孕妇≤200mg/天 |
| β-丙氨酸 | 6g/天 | 长期安全性待验证 |
| HMB | 6g/天 | 分3次服用 |
| 亮氨酸 | 10g/天 | 超过需求无额外益处 |

## 研究局限性

1. **发表偏倚**: 阳性结果更容易发表，导致Meta分析可能高估效应量
2. **异质性**: 不同研究的训练方案、受试者特征差异大
3. **短期研究为主**: 多数研究持续4-12周，长期效果未知
4. **单一补剂vs组合**: 多数研究设计为单一补剂，组合效果推断为间接证据 ⚠️ TODO
5. **个体差异**: 基因多态性(如ACTN3、CETP)影响补剂反应 ⚠️ TODO: 2024-2026精准营养研究
