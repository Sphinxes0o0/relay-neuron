## 植物学与活性成分

### 基本信息

| 项目 | 详情 |
|------|------|
| **学名** | Rhodiola rosea L. |
| **科属** | 景天科 (Crassulaceae) |
| **别名** | 金黄红景天 (Golden Root)、北极根 (Arctic Root)、Roseroot |
| **产地** | 北欧、俄罗斯西伯利亚、加拿大北极地区、中国西藏 |
| **使用历史** | 传统北欧/苏联医学用于抗疲劳、提神 |

### 关键活性化合物

#### 苯丙类化合物 (Phenylpropanoids)

| 成分 | 浓度范围 | 生理作用 | 重要性 |
|------|----------|----------|--------|
| **Rosavin** (酪醇) | 0.5-3.0% | 抗应激、神经保护 | 红景天特征性成分 |
| **Salidroside** (红景天苷) | 0.5-2.0% | 抗氧化、线粒体保护 | 关键活性代谢物 |
| **Tyrosol** (酪醇) | 0.1-1.0% | 抗炎、抗疲劳 | 协同作用 |

#### 其他活性成分

| 成分 | 浓度 | 生理作用 |
|------|------|----------|
| 熊果苷 (Arbutin) | 微量 | 抗氧化 |
| 百脉根苷 (Lotusine) | 微量 | 神经调节 |
| 挥发油 | 微量 | 芳香作用 |

### 标准含量

> **重要**: 优质红景天提取物应标准化至 **3% rosavins + 1% salidroside**
>
> 这是临床研究中最常用的SHR-5提取物规格

⚠️ **注意**: 部分产品仅标准化salidroside，rosavin含量可能不足，需查看COA检测报告

## 适应原机制

### 1. 酪氨酸羟化酶 (Tyrosine Hydroxylase) 激活

```
压力状态 → CNS疲劳 → 酪氨酸羟化酶活性 ↓ → 多巴胺/去甲肾上腺素 ↓

红景天作用 → 激活酪氨酸羟化酶 → 多巴胺/去甲肾上腺素合成 ↑
```

**核心机制**:
- 直接激活合成去甲肾上腺素(NE)和多巴胺(DA)的限速酶
- **仅在应激/疲劳时增强**，不影响基础神经递质水平
- 效果与哌醋甲酯(Methylphenidate)相当，但无成瘾性

### 2. HPA轴 (下丘脑-垂体-肾上腺轴) 调节

```
应激 → CRH ↑ → ACTH ↑ → 皮质醇 ↑

红景天 → 调节HPA轴 → 降低应激诱导的皮质醇峰值
```

**作用特点**:
- 减少应激时的皮质醇过度分泌
- 加速皮质醇恢复到基线水平
- **不影响基础皮质醇水平**（安全性关键）

### 3. cAMP-磷酸二酯酶 (cAMP-PDE) 调节 ⚠️ TODO

```
红景天 → 抑制磷酸二酯酶 → cAMP降解减少 → 细胞内cAMP ↑

cAMP升高 → 激活PKA → 下游效应:
  - 糖原分解增强
  - 脂肪分解增强
  - 心肌收缩力增强
```

⚠️ **TODO**: 2024-2026年研究需进一步阐明cAMP-PDE抑制的具体分子靶点和效力

### 4. 适应原 (Adaptogen) 定义一致性

红景天符合Brekhman-Dyman适应原三原则:

| 原则 | 红景天符合性 | 证据 |
|------|-------------|------|
| **安全性** | 符合 | 对正常生理功能无显著影响 |
| **非特异性** | 符合 | 增加机体对多种 stressors (物理/化学/生物)的抵抗力 |
| **双向调节** | 符合 | 根据状态平衡神经系统（既提神又镇静） |

### 5. 抗氧化与线粒体机制

| 机制 | 效应 |
|------|------|
| Nrf2-ARE通路激活 | 上调SOD、GPx抗氧化酶 |
| ROS清除 | 降低MDA、ROS水平 |
| 线粒体保护 | 保护电子传递链Complex I/III |
| 抗凋亡 | 抑制caspase-3激活 |

### 6. 神经保护机制

```
酪氨酸羟化酶激活 → NE/DA合成 ↑ → 警觉性 ↑, 疲劳感 ↓
```

- 增强中枢神经系统兴奋性
- 改善注意力和精神运动表现
- 抗凋亡作用（通过BDNF上调）

### 与其他提神物质的对比

| 物质 | 机制 | 提神效果 | 副作用 | 成瘾性 |

### SHR-5标准化提取物规格

| 参数 | 值 |
|------|-----|
| **推荐剂量** | 200-400mg SHR-5 |
| **Rosavin含量** | 3% (约6mg/200mg提取物) |
| **Salidroside含量** | 1% (约2mg/200mg提取物) |
| **服用方式** | 空腹或饭后30分钟 |
| **起效时间** | 单次200mg约1小时后减少疲劳感知 |

### 服用时机

| 时机 | 效果 | 推荐度 | 注意事项 |
|------|------|--------|----------|
| **晨间 (6-8点)** | 最佳提神效果 | ★★★★★ | 推荐首选用 |
| **午前 (10-11点)** | 减少工作疲劳 | ★★★★☆ | 有效 |
| **午后 (14-15点)** | 避免午后低谷 | ★★★☆☆ | 低剂量 |
| **傍晚 (18点后)** | **不推荐** | ★☆☆☆☆ | 可能影响睡眠 |

⚠️ **重要**: 避免晚间服用 - 红景天可能影响睡眠，尤其是高剂量

### 周期方案

| 方案 | 持续时间 | 效果 | 备注 |
|------|----------|------|------|
| 短期使用 | 4-12周 | 可见效果 | 多数研究使用 |

## 安全性

### 总体安全性评价

> **总结**: 红景天在推荐剂量(200-600mg/天标准化提取物)下总体安全，副作用发生率低(2-5%)

### 常见副作用

| 副作用 | 发生率 | 严重程度 | 处理方法 |
|--------|--------|----------|----------|
| 轻度刺激 | 2-5% | 轻微 | 降低剂量 |
| 胃部不适 | 2-5% | 轻微 | 与食物同服 |
| 头痛 | 1-3% | 轻微 | 减少剂量或停用 |
| 失眠 | <2% | 轻微 | 避免晚间服用 |
| 焦虑/烦躁 | <2% | 轻微 | 降低剂量 |
| 心悸 | <1% | 中度 | 高剂量时出现，避免与咖啡因合用 |

### 罕见副作用

- 血压波动
- 头晕
- 口干

### 禁忌人群

| 人群 | 原因 | 证据 |
|------|------|------|
| **双相情感障碍** | 可能诱发躁狂 | 临床报告 |
| 妊娠/哺乳期 | 安全性数据不足 | 建议避免 |
| 儿童 | 安全性数据不足 | 建议避免 |
| **MAOIs使用者** | 高血压危象风险 | ⚠️ 绝对禁用 |

⚠️ **重要警告**: 红景天不适合双相情感障碍患者 - 可能触发躁狂发作

⚠️ **MAOIs绝对禁忌**: 红景天与单胺氧化酶抑制剂(如苯乙肼、反苯环丙胺)禁止合用

### 药物相互作用

| 药物类型 | 相互作用 | 风险等级 | 建议 |
|----------|----------|----------|------|
| **MAOIs** | 高血压危象 | 极高 | **绝对禁用** |
| SSRIs | 5-HT综合征风险 | 中等 | 谨慎使用，监测 |
| SNRIs | 5-HT综合征风险 | 中等 | 谨慎使用，监测 |
| 甲状腺药物 | 可能增强 | 低-中等 | 监测甲状腺功能 |
| 抗凝药物 | 理论增强 | 低 | 谨慎使用 |
| 糖尿病药物 | 可能影响血糖 | 低 | 监测血糖 |
