## 2. EPA vs DHA 结构差异

### 2.1 化学结构对比

```
EPA (C20:5 n-3)
────────────────
CH₃-CH₂-(CH₂-CH=CH)₅-COOH
     ↑
  20碳骨架，5个双键（n-3位置）
  分子式: C₂₀H₃₀O₂
  分子量: 302.4 g/mol

DHA (C22:6 n-3)
────────────────
CH₃-CH₂-(CH₂-CH=CH)₆-COOH
     ↑
  22碳骨架，6个双键（n-3位置）
  分子式: C₂₂H₃₂O₂
  分子量: 328.5 g/mol
```

### 2.2 关键结构差异

| 特征 | EPA | DHA |
|------|-----|-----|
| 碳原子数 | 20 | 22 |
| 双键数量 | 5 | 6 |
| n-3脂肪酸链长 | 较短 | 较长 |
| 膜流动性影响 | 中等 | 高度增加膜流动性 |
| 脑组织穿透性 | 较低 | 较高 |
| β-氧化效率 | 较高 (链较短) | 较低 (链较长) |

### 2.3 代谢路径差异

EPA 和 DHA 在体内的代谢路径存在显著差异：

1. **去饱和酶作用**: Δ6-去饱和酶对18:3 n-3的效率较低，限制了长链PUFA合成
2. **链延长**: EPA可链延长为DHA，但转化率较低（<5%）
3. **β-氧化**: EPA比DHA更容易发生β-氧化，导致其生物利用度略低

## 3. EPA：抗炎与心血管健康

### 3.1 抗炎机制

EPA通过以下机制发挥强效抗炎作用：

#### 3.1.1 消退素 (Resolvins)
- **RvE1, RvE2, RvE3**: 主要由EPA衍生
- 作用: 促进炎症消退、减少中性粒细胞浸润
- 效应: 拮抗白三烯B4 (LTB4) 促炎信号 ⚠️ TODO DOI待补充

#### 3.1.2 保护素 (Protectins)
- **PD1 (Protectin D1)**: EPA和DHA共同前体
- 神经保护作用: 减少缺血性神经元损伤
- ⚠️ TODO 需要2024-2026神经保护专项研究

#### 3.1.3 前列腺素与白三烯调节
- EPA竞争性抑制花生四烯酸(AA)衍生的促炎前列腺素(PGE2)和白三烯(LTB4)
- EPA衍生的TXA3血栓素具有弱促血小板聚集作用（vs. TXA2）

### 3.2 心血管临床证据

#### 3.2.1 REDUCE-IT 试验 (里程碑研究)

| 项目 | 数据 |
|------|------|
| 研究设计 | 随机、双盲、安慰剂对照 |
| 样本量 | 8,179例 (EPA n=4,089; 安慰剂 n=4,090) |
| 人群 | 已接受他汀治疗的高甘油三酯血症患者 (TG 135-499 mg/dL) |
| 干预 | Icosapent ethyl (高纯度EPA) 4g/日 |
| 主要终点 | 心血管死亡+心肌梗死+卒中+冠脉血运重建 |
| 结果 | **25%相对风险降低** (HR 0.75; 95% CI 0.68-0.83) |
| 次要终点 | 心血管死亡减少20% |
| 发表 | Bhatt DL et al., *NEJM* 2019 |
| DOI | https://doi.org/10.1056/NEJMoa1812792 |

## 4. DHA：脑部、视网膜与神经系统

### 4.1 脑部富集与功能

#### 4.1.1 含量分布

| 组织 | DHA占比 (占总脂肪酸) | 备注 |
|------|---------------------|------|
| 脑灰质 | 8-10% | 主要富集于突触膜 |
| 脑白质 | 2-4% | 髓鞘主要成分 |
| 视网膜 | 50-60% | 光感受器盘膜主要磷脂 |
| 心脏 | 2-4% | 次要富集 |
| 肌肉 | <1% | 含量较低 |

#### 4.1.2 神经保护机制

1. **膜流动性**: DHA增加神经细胞膜流动性，促进神经信号传导
2. **突触可塑性**: 促进BDNF (脑源性神经营养因子) 表达 ⚠️ TODO DOI待补充
3. **神经发生**: 支持海马神经元新生
4. **抗凋亡**: 抑制caspase-3激活

## 5. EPA:DHA 比例建议

### 5.1 现有研究证据

#### 5.1.1 3:2比例的历史背景

传统鱼油补充剂多采用 **3:2 EPA:DHA** 比例，理由：
- 深海鱼类中自然比例约为3:2
- EPA+DHA协同提供基础抗炎和神经保护

## 6. 剂量建议与安全性

### 6.1 剂量建议汇总

| 人群/目标 | EPA+DHA总量 | EPA:DHA比例 | 备注 |
|-----------|-------------|-------------|------|
| 一般健康维护 | 1-2 g/日 | 3:2 或 1:1 | 膳食补充 |
| 心血管二级预防 | 2-4 g/日 | EPA主导 (≥2:1) | REDUCE-IT用4g纯EPA |
| 高甘油三酯血症 | 3-4 g/日 | EPA或3:2 | 需医疗监督 |
| 认知功能维护 | 1-2 g/日 | DHA主导 (1:2) | ⚠️ 证据有限 |
| 孕期/哺乳期 | 200-300 mg DHA+EPA | DHA优先 | ⚠️ DHA 200mg最低 |
| 儿童 (4-12岁) | ⚠️ TODO | ⚠️ 待定 | 需儿科建议 |

### 6.2 高剂量使用注意事项

⚠️ TODO 补充2024-2026年最新安全性数据

#### 6.2.1 出血风险
- EPA可抑制血小板聚集
- 高剂量 (≥4g/日) 可能增加出血时间
- 与抗凝药 (华法林、阿司匹林) 合用需监测 ⚠️ TODO DOI待补充

#### 6.2.2 LDL-C升高
- DHA可能导致LDL-C轻度升高 (5-10%)
- 机制: 肝LDL受体下调
- 通常为一过性，但高心血管风险患者需监测 ⚠️ TODO DOI待补充

#### 6.2.3 胃肠道反应
- 高剂量可能出现腹泻、消化不良
- 分次服用或随餐服用可减轻

### 6.3 鱼油产品质量

| 参数 | 推荐标准 | 说明 |
|------|----------|------|
| 纯度 | ≥80% EPA+DHA | 避免氧化产物 |
| EPA+DHA总量 | 每粒≥500mg | 确保有效剂量 |
| 氧化指标 | POV<5 meq/kg, MDA<3 μg/g | 避免氧化酸败 |
| 重金属 | 符合USP/EFSA标准 | 铅/汞/砷检测 |
| 认证 | 第三方检测 (IFOS, NSF) | 质量保证 |
