## 概述

维生素D是一种脂溶性维生素，对钙磷代谢、骨骼健康、免疫功能和肌肉功能至关重要。人体获取维生素D的主要途径包括：

- **外源性来源**：食物和膳食补充剂
- **内源性来源**：皮肤经紫外线B（UVB，290-315nm）照射后自行合成

维生素D在体内首先在肝脏转化为25-羟基维生素D [25(OH)D]，随后在肾脏转化为活性形式1,25-二羟基维生素D [1,25(OH)₂D]。25(OH)D是评估维生素D状态的公认生物标志物。

### 食物来源分析

#### 动物来源

**高含量来源（>500 IU/100g）**:
- 深海多脂鱼类（剑鱼、野生三文鱼）是维生素D最丰富的天然食物来源
- 鱼肝油浓度最高，但因维生素A含量也高，不建议作为常规膳食来源

**中等含量来源（100-500 IU/100g）**:
- 罐装鱼类（金枪鱼、沙丁鱼）维生素D含量较低但食用频率高
- 鸡蛋蛋黄中维生素D含量较高（全蛋因蛋黄比例问题平均值较低）

#### 植物来源

- **紫外线照射蘑菇**：紫外线照射可显著提高蘑菇的维生素D₂含量，是素食者的重要来源
- **强化食品**：牛奶、谷物、豆奶等强化食品是美国等国家人群维生素D的主要膳食来源

### 烹饪与加工影响

- 维生素D对热稳定，烹饪损失较小
- 油炸、烘烤等加工方式对维生素D影响有限

## 日光照射合成机制

### 皮肤合成途径

```
7-脱氢胆固醇（皮肤真皮层）
    ↓ UVB照射（290-315nm）
前维生素D₃（pre-vitamin D₃）
    ↓ 体温自发异构化（24-48小时）
维生素D₃（胆钙化醇）
    ↓ 肝脏
25-羟基维生素D [25(OH)D]
    ↓ 肾脏
1,25-二羟基维生素D [1,25(OH)₂D]（活性形式）
```

### 影响因素

| 因素 | 对合成的影响 |
|------|------------|
| **UVB强度** | 正午、高海拔、低纬度地区合成增加；冬季高纬度地区显著降低 |
| **皮肤暴露面积** | 暴露面积越大，合成越多（手臂和腿部暴露约提供每日需求的20-30%） |
| **肤色** | 黑色素越多，UVB吸收越少，合成率越低 |
| **年龄** | 皮肤合成效率随年龄下降（70岁时约为20岁的25%） |
| **防晒霜** | SPF 30可减少约95%的维生素D合成 |
| **玻璃穿透** | UVB无法穿透普通玻璃（坐在窗边不会合成维生素D） |

### 日照需求估算

**近似估算**：
- 浅肤色人群：春季/夏季每周2-3次，每次10-15分钟（暴露手臂和腿部）
- 深肤色人群：可能需要3-5倍的日照时间

| 纬度 | 冬季合成能力 |

## 缺乏风险

### 高风险人群

| 风险因素 | 说明 |
|---------|------|
| **纬度较高地区居民** | 冬季UVB合成不足 |
| **深肤色人群** | 黑色素抑制UVB吸收 |
| **老年人** | 皮肤合成效率下降，肾脏转化能力降低 |
| **肥胖人群** | 维生素D在脂肪组织中隔离，生物利用度降低 |
| **宅居人群** | 日照时间不足 |
| **宗教/文化着装** | 皮肤暴露面积受限 |
| **吸收不良综合征** | 肠道疾病影响脂肪吸收 |
| **使用皮质类固醇** | 影响维生素D代谢 |

### 缺乏的健康后果

#### 骨骼系统

- **佝偻病**（儿童）：骨骼矿化障碍，导致骨骼软化和畸形
- **骨软化症**（成人）：骨骼矿化不足，导致骨痛和肌肉无力
- **骨质疏松**：骨密度降低，骨折风险增加

#### 肌肉系统

- II型肌肉纤维萎缩
- 肌肉力量下降
- 平衡能力受损，跌倒风险增加

#### 免疫系统

- 自身免疫疾病风险增加
- 呼吸道感染风险增加

### 缺乏的临床表现

| 症状 | 说明 |
|------|------|
| 疲劳和乏力 | 非特异性，但常见 |
| 骨痛 | 特别是下背部和腿部 |
| 肌肉无力/疼痛 | 近端肌肉为主 |
| 情绪变化 | 抑郁风险增加 |
| 伤口愈合延迟 | 免疫功能受损 |

### 维生素D与运动表现

#### 骨骼健康

- 高强度运动对骨骼负荷大，需要充足的维生素D维持骨密度
- 女运动员三联征（低能量availability、月经功能障碍、骨密度降低）与维生素D状态相关

#### 肌肉功能

- 维生素D受体（VDR）在骨骼肌细胞表达
- 25(OH)D水平与肌肉力量、爆发力相关

#### 免疫功能

- 剧烈运动后免疫抑制期延长风险
- 维生素D调节先天免疫和适应性免疫
- 上呼吸道感染（URTI）在运动员中发生率较高

### 运动员补充建议

⚠️ TODO - 补充2024-2026年运动员维生素D补充指南

#### 基础建议

1. **筛查**：赛季前和赛季中检测血清25(OH)D水平
2. **日照**：在安全前提下增加户外训练时间
3. **膳食**：优先选择富含维生素D的食物

#### 补充策略

| 血清25(OH)D水平 | 建议策略 |
|----------------|---------|
| < 20 ng/mL | 治疗剂量：每日5,000-10,000 IU，持续8-12周后复查 |
| 20-30 ng/mL | 维护剂量：每日1,000-2,000 IU |
| 30-60 ng/mL | 维持剂量：每日600-1,000 IU |
| > 60 ng/mL | 暂停补充，3个月后复查 |

⚠️ TODO - 以上为一般建议，运动员具体补充方案应咨询运动医学医生

#### 补充注意事项

- 维生素D为脂溶性，建议随含脂肪膳食服用
- 与钙剂联合补充可能有益于骨骼健康
- 定期监测血清钙和25(OH)D水平
- 避免过度补充（> 10,000 IU/日长期使用可能有风险）

### 不同运动项目的特殊考虑

| 项目类型 | 特殊考虑 |
|---------|---------|
| **耐力运动** | 长期待在室内，维生素D不足风险高 |
| **力量运动** | 骨骼负荷大，钙和维生素D需求可能更高 |
| **冬季运动** | 日照不足，尤其在高纬度地区 |
| **室内运动**（游泳、体操等） | 缺乏UVB暴露，风险增加 |
| **户外运动** | 相对风险较低，但需注意防晒与合成的平衡 |
