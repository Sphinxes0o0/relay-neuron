## 1. KSM-66 vs 普通南非醉茄提取物区别

### 1.1 核心差异概览

| 比较维度 | KSM-66 | 普通南非醉茄提取物 |
|---------|--------|-------------------|
| **Withanolide含量** | ≥5%（标准化） | 1.5–3%（不稳定） |
| **提取来源** | 仅根部 | 根部、叶部或混合 |
| **提取工艺** | 专利水醇提取 + 牛奶预处理 | 单一溶剂提取 |
| **临床研究数量** | 30+项RCT | 有限或无 |
| **认证标准** | GRAS, Organic, Non-GMO | 不一致 |
| **批次稳定性** | 高 | 中-低 |

### 1.2 KSM-66 独家特征

**牛奶预处理工艺**：
- 遵循阿育吠陀传统方法，使用牛奶进行预处理
- 据称可增强活性成分的生物利用度
- ⚠️ TODO: 需要更多现代研究验证此工艺对withanolide生物利用度的影响

**根部专一性**：
- 仅使用南非醉茄根部，避免叶片成分
- 根部富含withaferin A、withanone等活性成分
- 叶子主要含糖苷醉茄内酯（镇静作用更强但研究较少）

### 1.3 不同提取物的Withanolide谱比较

| 成分 | KSM-66（根部） | Sensoril（根+叶） | 普通提取物 |
|------|--------------|-------------------|-----------|
| Withaferin A | 天然比例 | 较低 | 变异大 |
| Withanone | 天然比例 | 较低 | 变异大 |
| 醉茄内酯糖苷 | 全谱保留 | 较高 | 变异大 |
| 总withanolide | ≥5% | ≥10% | 1.5–3% |

### 1.4 生物利用度差异

## 2. 皮质醇调节机制

### 2.1 HPA轴调节核心机制

南非醉茄通过调节**下丘脑-垂体-肾上腺轴（HPA axis）**降低皮质醇水平，这是其作为"适应原"的核心机制。

```
┌─────────────────────────────────────────────────────────────────┐
│                    HPA轴（丘脑-垂体-肾上腺轴）                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    下丘脑                    垂体前叶              肾上腺皮质           │
│  ┌─────────┐            ┌─────────┐          ┌─────────┐         │
│  │   CRH   │───────────→│   ACTH  │─────────→│ 皮质醇   │         │
│  │  释放激素 │            │  促肾上腺 │          │ 分泌    │         │
│  └────┬────┘            │  皮质激素 │          └────┬────┘         │
│       ↑                  └────┬────┘               │              │
│       │                       │                    │              │
│   Ashwagandha                 │                    │              │
│   ↓ CRH释放                  │                    ↓              │
│   间接调节                   │              ↓ 皮质醇分泌           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 分子作用机制

| 作用靶点 | 机制 | 效应 |
|---------|------|------|
| **CRH基因表达** | ↓ CRH mRNA转录 | 减少CRH合成 |
| **ACTH分泌** | ↓ ACTH刺激 | 减少肾上腺激活 |
| **糖皮质激素受体(GR)** | ↑ GR敏感性 | 增强皮质醇负反馈 |
| **11β-HSD1酶** | 抑制活性 | ↓局部皮质醇再生 |
| **HSP70蛋白** | 调节应激蛋白 | 增强细胞应激耐受 |

│  ┌───────────────┐               ┌───────────────┐           │

│  │ ↑ GABAA受体  │               │ ↓ 焦虑相关   │           │

│          │                               │                    │
│          ↓                               ↓                    │
│  ┌─────────────────────────────────────────────────────┐      │
│  │              皮质醇 ↓ （协同降低）                     │      │
│  └─────────────────────────────────────────────────────┘      │
│                         │                                      │

│              ┌─────────────────────┐                          │

## 3. 抗应激适应证据

### 3.1 适应原定义与南非醉茄分类

根据Edwards等人的适应原定义（2000年），南非醉茄被归类为**一级适应原**，具备以下特征：

| 适应原标准 | 南非醉茄 | KSM-66证据 |
|-----------|---------|-----------|
| 对CNS无镇静/兴奋作用 | ✓ | ★★★★☆ |
| 对HPA轴有调节作用 | ✓ | ★★★★★ |
| 对实验性应激有保护作用 | ✓ | ★★★★☆ |
| 长期使用安全性良好 | ✓ | ★★★★☆ |

| **老年人** | 从低剂量开始 | 一般耐受良好 |

### 5.6 药物相互作用

| 药物类别 | 相互作用 | 机制 | 建议 |
|---------|---------|------|-----|
| **CNS抑制剂**（苯二氮卓类、安眠药） | 增强镇静 | GABAergic叠加 | 减少剂量或避免合用 |
| **甲状腺激素** | 可能影响甲状腺功能 | 适应原作用 | 监测甲状腺参数 |
| **免疫抑制剂** | 可能产生交互 | 免疫调节作用 | 谨慎使用 |
| **降压药物** | 可能影响血压 | 适应原作用 | 监测血压 |
| **抗糖尿病药物** | 可能影响血糖 | 代谢调节 | 监测血糖 |
