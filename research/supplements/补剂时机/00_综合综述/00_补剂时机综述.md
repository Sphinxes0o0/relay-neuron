## 摘要

补剂摄入时机（Nutrient Timing）是运动营养学中的核心议题，涉及训练前后特定营养素的摄入窗口对运动表现、肌肉合成、恢复和适应的影响。本综述综合分析了训练前（Pre-workout）、训练后（Post-workout）及睡前（Pre-sleep）三个关键时间窗口的最佳补剂策略，涵盖蛋白质、碳水化合物、咖啡因、肌酸等主要补剂的时机优化证据。

## 二、补剂时机原理

### 2.1 训练前补剂原理

**代谢窗口理论**：训练前摄入补剂可在运动开始时维持体内营养素水平，支持以下生理过程：

```
训练前补剂作用机制
        │
        ├─→ 提升能量基质（ATP、PCr、糖原）
        │
        ├─→ 增强神经驱动（咖啡因、酪氨酸）
        │
        ├─→ 改善肌肉血流（硝酸盐）
        │
        └─→ 启动合成代谢环境（EAA、亮氨酸）
```

**关键机制**：
- **mTOR通路预激活**：训练前摄入氨基酸可" primed"合成代谢通路
- **激素环境优化**：训练前蛋白质摄入可增强胰岛素敏感性
- **神经兴奋性**：咖啡因降低感知用力程度(RPE)

### 2.2 训练后补剂原理

**肌肉蛋白合成(MPS)窗口**：训练后30分钟至2小时是MPS的黄金窗口：

```
训练后合成代谢时间线
        │
    训练结束
        │
        ├─0-30min: MPS开始激活（雷帕霉素敏感）
        │
        ├─0-2h: MPS峰值响应期 ← 关键窗口
        │
        ├─2-4h: MPS逐渐下降
        │
        └─24-48h: 完全恢复基线
```

**关键机制**：
- 胰岛素敏感性增强
- 肌肉血流量增加（促进营养素输送）
- 卫星细胞激活
- 糖原合成加速

### 2.3 睡前补剂原理

**夜间恢复理论**：睡眠期间生长激素分泌增加，是肌肉恢复的关键时段：

```
睡前补剂作用机制
        │
        ├─→ 提供缓释氨基酸（酪蛋白）
        │
        ├─→ 维持夜间血浆氨基酸水平
        │
        ├─→ 支持生长激素/IGF-1作用
        │
        └─→ 改善睡眠质量（镁、褪黑素）
```

## 三、训练前补剂 (Pre-Workout Supplementation)

### 3.1 咖啡因 (Caffeine)

**推荐剂量**: 3-6 mg/kg体重（运动前30-60分钟）

**作用机制**:
- 腺苷受体拮抗 → 中枢兴奋
- 促进脂肪动员（增加血浆FFA）
- 增强肌钙蛋白敏感性
- 降低感知用力程度(RPE) 5-10%

**科学证据**:
- **效果量**: 中-大（特别是耐力和力量输出）
- **持续时间**: 摄入后30分钟达峰，半衰期3-5小时
- **个体差异**: CYP1A2基因多态性影响代谢

**文献引用**: Goldfarb et al. (2024) 在 [J Int Soc Sports Nutr](https://pubmed.ncbi.nlm.nih.gov/) 研究表明训练前咖啡因显著改善阻力训练表现 ⚠️ TODO: 需要2024-2026年文献补充

### 3.2 肌酸 (Creatine)

**推荐剂量**: 3-5g（训练前或后均可）

**作用机制**:
- 增加肌肉磷酸肌酸(PCr)储备
- 加速ATP再生
- 支持快速恢复（重复爆发力）

**科学证据**:
- **冲击期**: 20-25g/天 × 5-7天（非必需）
- **维持期**: 3-5g/天（与时机无关）
- ⚠️ TODO: 训练前摄入的额外益处尚需2024-2026年验证

**文献引用**: Kreider et al. (2017) 立场声明 [J Int Soc Sports Nutr](https://pubmed.ncbi.nlm.nih.gov/29106874)

### 3.3 必需氨基酸(EAA)与亮氨酸

**推荐剂量**: EAA 10-15g 或 亮氨酸 2-3g

**作用机制**:
- 触发mTORC1通路
- 促进肌肉蛋白合成(MPS)
- 训练前预激活合成代谢

**科学证据**:
- 亮氨酸是主要合成代谢信号分子
- EAA联合训练比单独训练MPS增幅更大 ⚠️ TODO: 需要2024-2026年验证

### 3.4 硝酸盐/甜菜根 (Nitrate/Beetroot)

**推荐剂量**: 300-600mg NO₃⁻（训练前2-3小时）

**作用机制**:
- NO₃⁻ → NO → 血管舒张
- 改善肌肉血流和氧输送
- 降低运动中氧耗

**科学证据**:
- 对耐力表现效益中等
- 对高强度间歇训练效果更明显 ⚠️ TODO: 需要2024-2026年验证

### 3.5 酪氨酸 (Tyrosine)

**推荐剂量**: 500-2000mg（训练前30-60分钟）

**作用机制**:
- 多巴胺和去甲肾上腺素前体
- 改善急性应激下的认知表现
- 减少疲劳感

**科学证据**: ⚠️ TODO: 需要更多高质量研究

## 四、训练后补剂 (Post-Workout Supplementation)

### 4.1 蛋白质（乳清蛋白）

**推荐剂量**: 20-40g（训练后30分钟至2小时内）

**作用机制**:
```
乳清蛋白 → 快速消化吸收
        │
        ├─→ 支链氨基酸(BCAA)释放
        │
        ├─→ 亮氨酸激活mTORC1
        │
        └─→ 肌肉蛋白合成(MPS)峰值
```

**科学证据**:
- 高生物价值(BV): 104-110
- 快速氨基酸血症（30-90分钟达峰）
- 训练后MPS响应增强20-50%

**文献引用**: Morton et al. (2018) [J Physiol](https://pubmed.ncbi.nlm.nih.gov/) 证明训练后蛋白质剂量-响应关系 ⚠️ TODO: 需要2024-2026年验证

### 4.2 必需氨基酸(EAA)

**推荐剂量**: 10-15g（训练后）

**作用机制**:
- EAA刺激MPS效率高于单独BCAA
- 所有9种必需氨基酸缺一不可
- 协同支持肌肉蛋白质周转

### 4.3 碳水化合物

**推荐剂量**: 1.0-1.2 g/kg/h（训练后前2小时，分次摄入）

**作用机制**:
- 糖原合成底物
- 胰岛素分泌 → 合成代谢环境
- 促进蛋白质合成

**科学证据**: ⚠️ TODO: 需要2024-2026年文献补充

### 4.4 肌酸（训练后）

**推荐剂量**: 3-5g

**作用机制**:
- 训练后补充可能增强糖原再合成
- 支持恢复期蛋白质合成
- 与蛋白质摄入可能有协同效应 ⚠️ TODO: 需要更多研究

### 4.5 欧米伽-3 脂肪酸

**推荐剂量**: 2-3g EPA+DHA

**作用机制**:
- 抗炎作用
- 可能增强mTOR信号
- 支持肌肉代谢 ⚠️ TODO: 需要2024-2026年验证

## 五、睡前补剂 (Pre-Sleep Supplementation)

### 5.1 酪蛋白 (Casein)

**推荐剂量**: 30-40g（睡前30分钟）

**作用机制**:
```
酪蛋白 → 缓慢消化（5-7小时）
        │
        ├─→ 夜间持续氨基酸释放
        │
        ├─→ 维持过夜MPS
        │
        └─→ 防止睡眠期间肌肉分解
```

**科学证据**:
- 相比乳清蛋白，过夜MPS维持效果更优
- 老年人群效果更显著 ⚠️ TODO: 需要2024-2026年验证

**文献引用**: Res et al. (2012) [Am J Clin Nutr](https://pubmed.ncbi.nlm.nih.gov/) 证明睡前酪蛋白促进过夜肌肉合成

### 5.2 褪黑素 (Melatonin)

**推荐剂量**: 0.5-3mg（睡前30分钟）

**作用机制**:
- 调节昼夜节律
- 改善睡眠质量
- 可能间接支持生长激素分泌

**科学证据**: ⚠️ TODO: 需要更多运动相关研究

### 5.3 镁 (Magnesium)

**推荐剂量**: 200-400mg（睡前）

**作用机制**:
- 神经肌肉放松
- 改善睡眠质量
- 支持300+酶反应（包括ATP酶）

**科学证据**: 低镁状态可能损害运动表现和恢复 ⚠️ TODO: 需要2024-2026年验证

### 5.4 锌+镁 (ZMA)

**推荐剂量**: 锌30mg + 镁200-400mg（睡前）

**作用机制**:
- 协同改善睡眠
- 支持睾酮合成（间接）
- 神经保护

**科学证据**: ⚠️ TODO: 需要更多高质量研究

## 七、时机策略争议

### 7.1 "合成代谢窗口"是否关键？

**传统观点**: 训练后30-60分钟是"黄金窗口"，必须立即补充

**现代证据**:
- ⚠️ TODO: 2024-2026年研究显示"窗口"可能比想象中更宽
- 每日总蛋白质摄入量和质量更重要
- 训练后2小时内摄入即可
- 睡前蛋白质对过夜恢复至关重要

**结论**: 时机仍然重要，但每日总量和分布可能比单一时间点更关键

### 7.2 蛋白质多餐分布

**推荐策略**:
```
每日蛋白质分布（示例，100-120g/天）
        │
        ├─ 早餐: 25-30g
        │
        ├─ 训练前: 20-25g
        │
        ├─ 训练后: 25-30g
        │
        └─ 睡前: 25-30g
```

## 九、研究结论

### 9.1 强证据支持

1. **训练前咖啡因**可提升运动表现（效果量中-大）
2. **训练后蛋白质**触发MPS（20-40g乳清或EAA）
3. **睡前酪蛋白**维持过夜肌肉合成
4. **每日总蛋白质量**比时机更关键

### 9.2 中等证据支持

1. 训练前肌酸增强训练容量
2. 训练后碳水促进糖原恢复
3. 甜菜根/硝酸盐改善耐力表现
4. 镁改善睡眠质量
