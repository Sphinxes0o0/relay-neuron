# 周期化补剂策略：训练周期与补剂调整

**研究日期**: 2026-05-06
**研究轮次**: Round 34
**文献类型**: 综合研究综述
**关键词**: 周期化, 补剂策略, 增肌期, 减脂期, 恢复期, 比赛期, 训练周期

---

## 摘要

补剂策略应与训练周期相协调，而非固定不变。本综述提出周期化补剂框架——在增肌期、减脂期、比赛期和恢复期采用不同的核心补剂重点，实现补剂资源的最大化利用和训练适应的最优化支持。

---

## 核心结论表

| 训练周期 | 核心补剂重点 | 辅助补剂 | 减量/停用 |
|---------|-------------|---------|-----------|
| **增肌期** | 蛋白质/肌酸/亮氨酸 | ZMA/维生素D | 可选HMB |
| **减脂期** | 蛋白质/HMB/咖啡因 | 代谢增强剂 | 肌酸(减量) |
| **比赛期/测试期** | 咖啡因/甜菜根 | 碳水化合物 | 非必需补剂 |
| **恢复期/减量** | Omega-3/镁/褪黑素 | 益生菌 | 咖啡因(减量) |

## 1. 周期化原理

### 1.1 训练周期基础

```
宏观周期 (Mesocycle) ──→ 12-16周
    ↓
中周期 (Microcycle) ──→ 4-6周
    ↓
微观周期 (Training Session) ──→ 每日
```

| 周期类型 | 时长 | 补剂重点 |
|----------|------|----------|
| 增肌期 | 8-16周 | 合成代谢支持 |
| 减脂期 | 8-16周 | 肌肉保存+代谢 |
| 比赛期 | 1-4周 | 表现优化 |
| 恢复期 | 1-2周 | 再生支持 |

### 1.2 补剂周期化的理由

| 原因 | 解释 |
|------|------|
| **资源优化** | 不同阶段需求不同，集中资源于关键阶段 |
| **耐受性维持** | 避免长期使用导致敏感性下降 |
| **目标对齐** | 补剂支持训练目标而非泛泛使用 |
| **成本效益** | 避免在不必要时使用昂贵补剂 |

## 2. 各周期补剂策略

### 2.1 增肌期补剂策略

#### 核心目标
- 最大化肌肉蛋白质合成
- 支持训练恢复
- 维持合成代谢激素环境

#### 核心补剂

| 补剂 | 剂量 | 时机 | 证据等级 |
|------|------|------|----------|
| **蛋白质** | 2-2.5g/kg | 均匀分次 | A级 |
| **肌酸** | 5g/天 | 任意时间 | A级 |
| **亮氨酸** | 3-5g/次 | 蛋白质餐时 | B级 |

#### 可选补剂

| 补剂 | 理由 | 证据 |
|------|------|------|
| HMB | 减少肌肉分解 | 中等 |
| ZMA | 锌镁支持，睡眠质量 | 中等 |
| 维生素D | 肌肉功能 | 中等(缺乏者) |
| 复合益生菌 | 肠道健康 | 中等 |

#### 增肌期方案

```
基础方案:
- 蛋白质: 2-2.5g/kg/天
- 肌酸: 5g/天
- 维生素D: 2000 IU/天

可选强化:
- HMB: 3g/天(训练强度极高时)
- ZMA: 睡前(睡眠问题者)
- 益生菌: 10^10 CFU(肠道问题者)
```

### 2.2 减脂期补剂策略

#### 核心目标
- 保存肌肉组织
- 维持代谢率
- 控制食欲
- 支持脂肪动员

#### 核心补剂

| 补剂 | 剂量 | 时机 | 证据等级 |
|------|------|------|----------|
| **蛋白质** | 2.5-3g/kg | 均匀分次 | A级 |
| **HMB** | 3g/天 | 分次 | B级 |
| **咖啡因** | 3-5mg/kg | 训练前 | A级 |

#### 可选补剂

| 补剂 | 理由 | 证据 |
|------|------|------|
| 绿茶提取物 | 脂肪氧化↑ | 中等 |
| Omega-3 | 抗炎 | 中等 |
| 姜黄素 | 炎症管理 | 中等 |
| 低碳水时肌酸 | 可减量 | - |

#### 减脂期方案

```
核心方案:
- 蛋白质: 2.5-3g/kg/天
- HMB: 3g/天
- 咖啡因: 3-5mg/kg(训练前)

可选强化:
- 绿茶EGCG: 300-500mg/天
- Omega-3: 2-3g/天
- 复合益生菌: 维持肠道健康
```

#### 肌酸在减脂期的使用

| 观点 | 理由 |
|------|------|
| **继续使用** | 细胞水合维持，肌肉保存 |
| **减量使用** | 减脂期水分需求下降 |
| **暂停使用** | 节省资源，周期化耐受 |

### 2.3 比赛期/测试期补剂策略

#### 核心目标
- 最大化运动表现
- 优化比赛日状态
- 延迟疲劳

#### 核心补剂

| 补剂 | 剂量 | 时机 | 证据等级 |
|------|------|------|----------|
| **咖啡因** | 3-6mg/kg | 比赛前45-60min | A级 |
| **甜菜根/硝酸盐** | 300-500mg | 比赛前2-3h | A级 |
| **碳水化合物** | 30-90g/h | 赛中(耐力项目) | A级 |

#### 比赛周期补剂安排

| 时间点 | 补剂策略 |
|--------|----------|
| 赛前1周 | 维持基础补剂，测试比赛日补剂 |
| 赛前48h | 碳水负荷(如使用) |
| 赛前2-3h | 甜菜根(硝酸盐) |
| 赛前45min | 咖啡因 |
| 赛中 | 碳水+电解质(耐力项目) |
| 赛后 | 蛋白质+碳水恢复 |

### 2.4 恢复期/减量期补剂策略

#### 核心目标
- 支持组织修复
- 管理炎症
- 优化睡眠质量
- 为下一周期做准备

#### 核心补剂

| 补剂 | 剂量 | 时机 | 证据等级 |
|------|------|------|----------|
| **Omega-3** | 2-3g | 随餐 | B级 |
| **镁** | 200-400mg | 睡前 | B级 |
| **褪黑素** | 0.5-1mg | 睡前(如需要) | B级 |

#### 可选补剂

| 补剂 | 理由 | 证据 |
|------|------|------|
| 南非醉茄 | 皮质醇管理 | 中等 |
| 谷氨酰胺 | 肠道修复 | 中等 |
| 益生菌 | 肠道健康 | 中等 |
| 氨基葡萄糖 | 关节支持 | 中等 |

#### 恢复期方案

```
核心方案:
- Omega-3: 2-3g EPA+DHA/天
- 镁: 200-400mg/天(甘氨酸形式)
- 充足蛋白质: 1.6-2g/kg/天

可选:
- 南非醉茄: 300-600mg(如皮质醇高)
- 褪黑素: 0.5-1mg(如睡眠问题)
- 益生菌: 维持
```

## 3. 周期化实施

### 3.1 年度补剂周期示例

```
第1-16周: 增肌期
├── 基础: 蛋白质+肌酸+维生素D
├── 可选: HMB(前4周), ZMA
└── 评价: 每4周评估进展

第17-24周: 减脂期
├── 基础: 蛋白质+HMB+咖啡因
├── 可选: 绿茶, Omega-3
└── 评价: 每周监测体重/围度

第25-28周: 比赛/测试期
├── 基础: 咖啡因+甜菜根+碳水
├── 赛前测试补剂反应
└── 评价: 比赛表现

第29-30周: 恢复期
├── 基础: Omega-3+镁+蛋白质维持
├── 可选: 南非醉茄, 益生菌
└── 评价: 感觉/睡眠/活力

(循环往复)
```

### 3.2 咖啡因周期化

| 策略 | 方法 | 目的 |
|------|------|------|
| 减量期 | 2-4周低剂量或停用 | 恢复敏感性 |
| 比赛期 | 高剂量(3-6mg/kg) | 最大化表现 |
| 训练日 | 中等剂量(2-3mg/kg) | 维持耐受 |

### 3.3 肌酸周期化

| 策略 | 方法 | 目的 |
|------|------|------|
| 持续使用 | 5g/天维持 | 持续饱和 |
| 增肌期使用 | 增肌期全程 | 合成环境 |
| 减脂期可选 | 维持或减量 | 肌肉保存 |

## 4. 特殊情况处理

### 4.1 过度训练期

| 信号 | 补剂调整 |
|------|----------|
| 持续疲劳 | 增加恢复补剂，减少刺激剂 |
| 睡眠问题 | 添加镁/褪黑素/南非醉茄 |
| 免疫下降 | 增加维生素D/锌/益生菌 |
| 皮质醇↑ | 南非醉茄/磷脂酰丝氨酸 |

### 4.2 受伤/疾病期

| 情况 | 补剂策略 |
|------|----------|
| 炎症期 | Omega-3/姜黄素/蛋白 |
| 组织修复期 | 蛋白质/维生素C/锌 |
| 卧床/恢复 | 维持基础补剂 |
| 抗生素期间 | 益生菌(间隔2h) |

### 4.3 旅行/时差期

| 情况 | 补剂策略 |
|------|----------|
| 跨时区 | 褪黑素+镁 |
| 长途飞行 | 水分+电解质+走动 |
| 适应期 | 南非醉茄(应激) |

## 5. 补剂优先级排序

### 5.1 按证据等级排序

| 优先级 | 补剂 | 证据等级 | 适用周期 |
|--------|------|----------|----------|
| 1 | 蛋白质 | A级 | 全部 |
| 2 | 肌酸 | A级 | 增肌期/恢复期 |
| 3 | 咖啡因 | A级 | 比赛期/训练期 |
| 4 | 维生素D | B级 | 全部(缺乏者) |

| 6 | HMB | B级 | 减脂期 |
| 7 | 镁 | B级 | 恢复期/睡眠问题 |

## 6. 实践建议

### 6.1 补剂日志

建议记录:
- 使用的补剂和剂量
- 训练内容和表现
- 睡眠质量和恢复感
- 体重和围度变化
- 主观能量/情绪状态

### 6.2 评估周期

| 评估点 | 评估内容 |
|--------|----------|
| 每4周 | 体重/围度/力量 |
| 每8周 | 更大评估(可含体脂) |
| 周期末 | 整体回顾，调整下一周期 |

### 6.3 补剂精简原则

**精简到核心**:
1. 蛋白质(所有阶段)
2. 肌酸(增肌期为主)
3. 咖啡因(表现期)
4. 维生素D(缺乏者)
5. 镁(恢复/睡眠)

## 7. 研究局限性

1. **周期化研究有限**: 补剂周期化的直接研究很少
2. **个体差异**: 需根据个人反应调整
3. **运动水平差异**: 休闲vs精英运动员需求不同
4. **补剂质量差异**: 天然产品批次差异
5. **长期坚持难度**: 复杂方案依从性低
