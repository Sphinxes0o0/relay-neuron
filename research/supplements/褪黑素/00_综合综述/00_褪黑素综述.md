## 2. 作用机制

### 2.1 昼夜节律系统

褪黑素是人松果体分泌的激素，其分泌遵循严格的昼夜节律：

- **分泌模式**: 黑暗刺激启动分泌，血浆浓度从深夜22:00开始上升，峰值出现在凌晨02:00-04:00（可高达200 pg/mL），日间降至<10 pg/mL
- **光照调控**: 光照（尤其是460-480nm蓝光）通过视网膜-视交叉上核(SCN)通路抑制褪黑素分泌
- **SCN整合**: 视交叉上核(SCN)作为主生物钟，整合光照信号，调控全身节律

```
光照 → 视网膜神经节细胞 → 视交叉上核(SCN) → 抑制褪黑素分泌
黑暗 → 松果体 → 分泌褪黑素 → 诱导睡眠倾向
```

### 2.2 褪黑素受体：MT1与MT2

褪黑素通过两种高亲和力G蛋白偶联受体发挥作用：

| 受体 | 分布 | 主要功能 | 信号转导 |
|------|------|----------|----------|
| **MT1 (MTNR1A)** | SCN、脉络膜、视网膜、下丘脑 | 抑制神经元活动，促进入睡 | Gi蛋白 → 抑制cAMP ↑ |
| **MT2 (MTNR1B)** | SCN、视网膜、海马、小脑 | 调节相位转换、视网膜信号 | Gi蛋白 → 抑制cAMP ↑；Gq → PLC通路 |

**核心作用**:
1. **MT1**: 介导睡眠的启动，通过抑制SCN唤醒神经元活动
2. **MT2**: 调节昼夜节律相位（相位提前/推迟），参与视网膜光信号处理
3. **MT3 (XRXR)**: 低亲和力结合酶，体外实验中具有抗氧化作用，非主流睡眠靶点

### 2.3 睡眠诱导机制

褪黑素促进睡眠的多路径模型：

```
褪黑素 ─┬─→ MT1 → 抑制SCN谷氨酸能神经元 → 降低皮层唤醒
        ├─→ MT2 → 重置SCN时钟相位 → 同步睡眠窗口
        ├─→ 下丘脑视前区 → GABA能活化 → 促睡眠
        └─→ 降低核心体温(~0.3-0.5°C) → 增强睡意
```

**关键生理效应**:
- 核心体温轻度下降（通过外周血管舒张）
- 皮质醇水平晚间降低
- 血压、心率轻度下降
- 增强夜间免疫功能

## 3. 睡眠改善效果

### 3.1 入睡潜伏期（Sleep Onset Latency, SOL）

**元分析结论**:
- 褪黑素显著缩短SOL，平均减少**7-12分钟**（Buscemi et al., 2006）
- Ferracioli-Oda等(2013)对19项RCT的Meta分析：标准化均数差(SMD) = -0.22，p<0.001 ⚠️ TODO
- 效果在原发性失眠患者中更显著

**剂量效应**:
- 极低剂量(0.1-0.3mg): 轻度效果，可能仅适用于褪黑素缺乏者
- 低剂量(0.5-1mg): 中等效果，接近生理峰值水平
- 标准剂量(1-5mg): 最常见有效剂量范围
- 高剂量(>5mg): 不增加额外收益，徒增清晨困倦风险

### 3.2 睡眠质量（Sleep Quality）

**主观睡眠质量**:
- PSQI（匹兹堡睡眠质量指数）改善约10-15%
- 主观感受到"更容易入睡"和"睡眠更深"

**客观睡眠指标**（多导睡眠图/PSG研究）⚠️ TODO:
- 睡眠效率：提升3-8%
- NREM Stage 2比例增加
- NREM Stage 3（深度睡眠）比例无显著变化或轻度减少
- REM睡眠时间无显著改变
- 觉醒次数(WASO)轻度减少

**睡眠结构特点**:
- 褪黑素不显著改变正常睡眠结构
- 与苯二氮卓类药物不同，无反跳性失眠
- 不产生耐受性（长期使用效果稳定）

### 3.3 特定人群

| 人群 | 效果评价 | 备注 |
|------|----------|------|
| 原发性失眠 | 有效，尤其入睡困难型 | 首选非药物治疗 |
| 继发性失眠（疼痛/焦虑） | 辅助效果 | 需治疗原发病因 |
| 倒班工作者 | 效果有限 | 时钟混乱难以纠正 |
| 老年人（褪黑素分泌下降） | 效果较好 | 生理性缺乏者获益最大 |
| 青少年 | 需谨慎，剂量≤1mg | 影响青春期HPA轴 ⚠️ TODO |

## 4. 时差（Jet Lag）效果

### 4.1 循证依据

**Cochrane系统评价** (Herxheimer & Petrie, 2002; 更新版) ⚠️ TODO:
- 跨≥5时区旅行：褪黑素显著减少时差症状
- 疗效量：标准化效应约0.7（中等效应量）
- 东向旅行（提前生物钟）：效果优于西向

**Honn等(2019)系统评价** ⚠️ TODO:
- 纳入15项RCT，n=738
- 褪黑素组时差症状评分显著低于安慰剂组
- 最优剂量：0.5-5mg

### 4.2 机制

时差的本质是昼夜节律与目的地时间不同步：
- 褪黑素通过MT2受体调节SCN相位
- 相位提前效应：服用褪黑素+提前光照可加速节律适应
- 典型相位提前：每日提前30分钟睡眠时间窗 + 对应时间服用褪黑素

### 4.3 应用方案

**东向旅行（跨越≥5时区）**:
```
目的地入睡时间 = 原出发地入睡时间 - 时差数 + 适应期修正
建议方案:
- 出发前1-3天：开始提前睡眠时间（每日15-30min）
- 旅行期间：按目的地夜间时间服用褪黑素0.5-3mg
- 到达后：连续使用3-5晚，或直到适应
```

**西向旅行**:
- 效果相对弱，因为延长昼夜周期比缩短更难
- 高剂量(3-5mg)可能更有效 ⚠️ TODO

**跨越极地（>8时区）**:
- 效果不确定，相当于绕道飞行 ⚠️ TODO

### 4.4 时差严重程度评估

| 时差 | 症状 | 建议 |
|------|------|------|
| ≤3时区 | 轻微疲劳，可忽略 | 可不使用褪黑素 |
| 4-6时区 | 中度症状：睡眠紊乱、日间疲劳 | 建议0.5-1mg |
| 7-9时区 | 明显症状 | 建议1-3mg |
| ≥10时区 | 严重：胃肠紊乱、认知下降 | 建议3-5mg+光照配合 |

## 5. 剂量建议

### 5.1 剂量-效应关系

基于现有文献的综合建议：

| 剂量范围 | 适用场景 | 效果预期 |
|----------|----------|----------|
| **0.3-0.5mg** | 轻度睡眠障碍、老年人 | 补充生理缺乏，微调相位 |
| **1-2mg** | 一般失眠、时差预防 | 标准起始剂量 |
| **3-5mg** | 中重度时差、昼夜节律失调 | 强效相位调节 |
| **>5mg** | 一般不推荐 | 无额外收益，翌日困倦风险↑ |

### 5.2 服用时间

**核心原则**:
- **睡前30-60分钟**服用，使血药峰值与睡眠窗口对齐
- 时差应用：根据目的地时间调整，在"目的地睡前时间"服用

**时间误差敏感性**:
- 褪黑素相位调节效果高度依赖服用时间
- 错误时间（错误相位）可能导致反效果（进一步推迟节律）
- 一般规则：比目标睡眠时间早**5-6小时**服用 = 相位提前效应

### 5.3 剂型选择

| 剂型 | 吸收速度 | 适用性 |
|------|----------|--------|
| 普通片剂 | 30-60min达峰 | 通用选择 |
| 缓释片 | 持续释放4-6小时 | 维持睡眠效率 |
| 舌下含服 | 15-30min达峰 | 紧急入睡需求 |
| 透皮贴 | 稳定释放 | 吸收不良者 ⚠️ TODO |

### 5.4 运动员剂量建议

考虑到运动表现的敏感性：
- **低剂量优先**: 0.5-1mg作为起始
- **比赛期**: 赛前2-3天使用，避免影响晨间表现
- **训练期**: 可每晚使用，改善恢复性睡眠
- **注意**: 高剂量(>3mg)可能降低夜间生长激素脉冲 ⚠️ TODO

## 6. 安全性

### 6.1 急性安全性

**短期使用（数天至数周）**:
- 耐受性良好，不良反应轻微
- 常见轻微反应：头痛(2-3%)、头晕(1-2%)、恶心(1-2%)
- 翌日困倦：剂量>3mg或服用时间过晚时可能出现

**严重不良事件**: 罕见，无明确因果关系 ⚠️ TODO

### 6.2 长期安全性

**3个月内使用**: 安全性数据充分，无显著担忧

**>6个月使用** ⚠️ TODO:
- 现有数据未显示严重安全问题
- 但长期数据仍然有限
- 建议定期评估使用必要性

### 6.3 特殊人群安全性

| 人群 | 安全性评估 | 注意事项 |
|------|------------|----------|
| 孕妇/哺乳期 | 缺乏数据 | 不推荐使用 |
| 儿童/青少年 | ⚠️ TODO | 仅在明确医学指征下使用，注意剂量≤1mg |
| 癫痫患者 | 理论风险 | 谨慎使用，可能降低阈值 |
| 免疫抑制患者 | 双向调节 | 可能增强夜间免疫，可逆性 ⚠️ TODO |
| 糖尿病患者 | ⚠️ TODO | 可能影响血糖波动，需监测 |
| 凝血障碍患者 | ⚠️ TODO | 可能增强抗凝效果 |

### 6.4 药物相互作用

| 药物类别 | 相互作用 | 机制/建议 |
|----------|----------|----------|
| **抗凝药** (华法林) | 增强抗凝效果 | 监测凝血参数 |
| **抗癫痫药** | 可能降低癫痫阈值 | 谨慎合用 |
| **避孕药** | 可能增强褪黑素效果 | 降低褪黑素剂量 |
| **咖啡因** | 轻微抵消入睡效果 | 睡前6小时避免咖啡因 |
| **褪黑素类似物** | 叠加效应 | 避免联用 |
| **苯二氮卓类** | 睡眠结构改变 | 不推荐常规联用 |

## 7. 运动员特别注意

### 7.1 运动表现影响

**理论上存在的风险** ⚠️ TODO:
1. **生长激素**: 高剂量褪黑素(>3mg)可能抑制夜间GH脉冲
   - 但低剂量(0.5-1mg)影响极小
   - 睡眠本身是GH分泌的主要时段，良好睡眠反而促进GH

2. **肌肉恢复**: 优质睡眠对肌肉蛋白合成至关重要，褪黑素间接有利于恢复

3. **次晨表现**: 高剂量可能导致晨起残留睡意，影响早训

**实践建议**:
- 训练日：训练后低剂量(0.5-1mg)
- 比赛日：仅在必要时使用，避免影响当日表现
- 跨时区比赛：提前3-5天开始调节

### 7.2 兴奋剂检测

- 褪黑素不在WADA禁用清单上
- 运动员可合法使用
- 但需注意部分复合补充剂中可能含有禁用成分

### 7.3 免疫功能

- 褪黑素具有免疫调节作用
- 长期高强度训练者（免疫抑制风险）使用需谨慎 ⚠️ TODO
- 可能增强夜间免疫监视（胸腺肽分泌峰值）

### 7.4 应激激素

- 褪黑素可能降低基础皮质醇水平
- 对HPA轴的影响尚不明确 ⚠️ TODO
- 高皮质醇状态的运动员可能获益

## 8. 与睡眠卫生的比较

### 8.1 睡眠卫生定义

睡眠卫生(Sleep Hygiene)是非药物干预的基础，包括：
- 规律作息（固定起床/就寝时间）
- 卧室环境优化（温度18-20°C、黑暗、安静）
- 限制咖啡因（睡前6小时）
- 限制酒精
- 避免睡前屏幕蓝光
- 规律运动（但避免睡前3小时剧烈运动）

### 8.2 效果对比

| 维度 | 睡眠卫生 | 褪黑素 | 联合干预 |
|------|----------|--------|----------|
| **入睡潜伏期** | 改善10-15分钟 | 改善7-12分钟 | 最佳效果 |
| **睡眠质量** | 显著改善（长期） | 中度改善 | 显著改善 |
| **睡眠效率** | 改善3-5% | 改善3-8% | 改善8-12% |
| **相位调节** | 轻度 | 强效 | 强效 |
| **时差** | 有限 | 显著 | 最佳 |
| **依赖性** | 无 | 无 | 无 |
| **副作用** | 无 | 轻微 | 轻微 |
| **起效速度** | 数周 | 30-60分钟 | 最快 |

### 8.3 推荐实践

**一线方案**: 睡眠卫生（所有患者/运动员的基础）

**二线方案**: 睡眠卫生 + 低剂量褪黑素(0.5-1mg)

**三线方案**: 睡眠卫生 + 褪黑素 + 认知行为疗法(CBT-I)

**不建议**:
- 单独使用褪黑素而不改善睡眠卫生
- 高剂量褪黑素作为长期解决方案
- 将褪黑素作为"助眠剂"滥用

### 8.4 睡眠卫生具体建议（运动员版）

1. **作息规律**: 每周7天固定就寝/起床时间（±30分钟内）
2. **午睡管理**: 午后1-3点小睡≤30分钟，避免影响夜间睡眠压力
3. **训练-睡眠协调**: 高强度训练日提前30分钟就寝
4. **营养配合**: 睡前避免高碳水（血糖波动影响睡眠）
5. **光照管理**: 晨间光照30分钟（促进相位提前）
6. **环境控制**: 运动员村/酒店使用遮光眼罩+耳塞
7. **电子设备**: 睡前1小时关闭电子设备
