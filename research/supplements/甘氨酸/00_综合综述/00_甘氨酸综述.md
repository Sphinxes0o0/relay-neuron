**研究日期**: 2026-05-04

**研究类型**: 综合文献综述

## 摘要

甘氨酸（Glycine，氨基乙酸，化学式C₂H₅NO₂，分子量75.07 g/mol）是非必需氨基酸，在人体中具有多重生理功能。作为抑制性神经递质，甘氨酸参与睡眠调节并通过GABAergic机制改善睡眠质量；作为胶原蛋白的核心氨基酸组分，甘氨酸占胶原蛋白氨基酸总量的约33%；作为谷胱甘肽（GSH）的前体，甘氨酸在抗氧化防御中发挥重要作用。本综述综合分析甘氨酸在睡眠改善、胶原蛋白合成支持及运动表现方面的循证医学证据。

## 1. 甘氨酸基础特性

### 1.1 化学与生化特性

| 属性 | 数值/描述 |
|-----|---------|
| **化学式** | C₂H₅NO₂ |
| **分子量** | 75.07 g/mol |
| **CAS号** | 56-40-6 |
| **等电点** | pH 5.97 |
| **味道** | 甜味（甘氨酸名称来源） |
| **分类** | 非必需氨基酸、最简单的氨基酸 |

### 1.2 生理分类

- **非蛋白质氨基酸**: 甘氨酸是胶原蛋白中唯一的非α-氨基酸
- **神经递质**: 抑制性神经递质，作用于甘氨酸受体（GlyR，配体门控Cl⁻通道）
- **NMDA受体共激动剂**: 甘氨酸位点是NMDA受体的必需共激动剂（结合GluN1亚基）

### 1.3 体内代谢与分布

```
膳食摄入甘氨酸 → 肠道吸收 → 门静脉 → 肝脏代谢
     ↓
主要去路:
1. 胶原蛋白合成（结缔组织、皮肤、骨骼）— 占比最高
2. 谷胱甘肽合成（抗氧化）— GSH由甘氨酸、半胱氨酸、谷氨酸构成
3. 肌酸合成（能量代谢）— 甘氨酸提供胍基
4. 神经递质（中枢神经系统）
5. 嘌呤合成（DNA/RNA前体）
```

## 2. 甘氨酸生理功能

### 2.1 胶原蛋白合成

#### 2.1.1 胶原蛋白氨基酸组成

甘氨酸在胶原蛋白中占据独特且不可替代的地位:

| 氨基酸 | 胶原蛋白中占比 | 生理功能 |
|-------|-------------|---------|
| **甘氨酸 (Glycine)** | **33%** | 三螺旋结构核心，每三个氨基酸一个 |
| 脯氨酸 (Proline) | 12-13% | 稳定螺旋结构 |
| 羟脯氨酸 (Hydroxyproline) | 10% | 分子间交联必需 |
| 丙氨酸 (Alanine) | 8-9% | 结构支持 |
| 谷氨酸 (Glutamate) | 7-8% | 结构支持 |

#### 2.1.2 胶原蛋白三螺旋结构

```
胶原三螺旋（Triple Helix）结构:

甘氨酸-X-Y 重复序列（Gly-X-Y）

其中:
- X 常为脯氨酸（Pro）
- Y 常为羟脯氨酸（Hyp）

甘氨酸的小体积（最小氨基酸，仅H原子侧链）使三股螺旋能够紧密缠绕
```

**关键要点**:
- 甘氨酸是每个第三个氨基酸位置的**绝对必需**
- 无甘氨酸则无法形成稳定的三螺旋
- 甘氨酸的最小侧链（仅一个氢原子）是三螺旋紧密缠绕的结构基础

### 2.2 神经递质功能

#### 2.2.1 甘氨酸受体（GlyR）

甘氨酸作为抑制性神经递质作用于脑干和脊髓的甘氨酸受体:

```
甘氨酸 + GlyR → 氯离子内流 → 超极化 → 神经元抑制
```

- **配体门控氯离子通道**（类似GABA_A受体）
- 主要分布于脑干、脊髓
- 参与运动控制和睡眠调节

#### 2.2.2 NMDA受体调节

NMDA受体需要双激动剂才能激活:

| 激动剂 | 结合位点 | 作用 |
|-------|---------|-----|
| 谷氨酸 | GluN2亚基 | 主要激动剂 |
| 甘氨酸 | GluN1亚基（甘氨酸位点） | 共激动剂（必需） |

```
NMDA受体激活:
谷氨酸（结合GluN2）+ 甘氨酸（结合GluN1）→ 通道开放 → Ca²⁺内流 → 突触可塑性
```

### 2.3 谷胱甘肽（GSH）前体

谷胱甘肽由三种氨基酸构成:

| 成分 | 比例 | 功能 |
|-----|-----|-----|
| **甘氨酸** | 1/3 | 结构支持 |
| 半胱氨酸 | 1/3 | 活性硫醇，提供还原性 |
| 谷氨酸 | 1/3 | γ-肽键 |

```
γ-谷氨酰半胱氨酸 + 甘氨酸 → 谷胱甘肽（GSH）
```

GSH是细胞内主要的抗氧化剂，参与:
- 自由基清除
- 维生素C/E再生
- 解毒作用
- 免疫功能

## 3. 睡眠改善效果

### 3.1 GABAergic机制

甘氨酸改善睡眠的核心机制涉及多个层面:

#### 3.1.1 抑制性神经传导

```
甘氨酸 → 激活GlyR → Cl⁻内流 → 神经元超极化 → 抑制性神经传导 → 促进睡眠
```

甘氨酸与GABA系统存在功能性交互:
- 甘氨酸可增强GABAergic抑制
- 两者共同促进睡眠 onset
- 甘氨酸可能通过桥脑抑制区促进睡眠

#### 3.1.2 体温调节作用

```
甘氨酸 → 促进皮肤血管舒张 → 末梢血流量增加 → 散热增加 → 核心体温下降 → 触发睡眠
```

核心体温下降是睡眠 onset 的重要生理信号，甘氨酸通过促进散热加速这一过程。

### 3.2 关键研究证据

#### 3.2.1 Bannai et al. (2012) — PSG研究

- **设计**: 随机双盲交叉对照试验
- **样本**: 15名有睡眠问题的健康志愿者
- **干预**: 3g甘氨酸 vs 安慰剂，睡前服用
- **评估**: 多导睡眠图（PSG）

**主要发现**:
- 睡眠效率改善
- 睡眠潜伏期缩短趋势
- 无次日残留困倦

> DOI: 10.1111/j.1479-8425.2012.00558.x

- **设计**: 开放标签研究
- **样本**: 睡眠不满意者
- **干预**: 3g甘氨酸睡前

**主要发现**:
- 主观睡眠质量改善
- 入睡容易度提高
- 第二天清醒度改善

- **设计**: 多中心开放研究

- **干预**: 3g/天，持续4周

**主要发现**:
- 主观睡眠满意度改善
- 睡眠障碍评分降低

## 4. 胶原蛋白协同

### 4.1 协同因素概述

胶原蛋白合成是一个复杂过程，需要多种营养素协同:

```
胶原蛋白合成协同因素:

甘氨酸（底物33%）+ 脯氨酸 + 羟脯氨酸 → 胶原三螺旋
                ↓
        维生素C（辅因子）→ 羟脯氨酸合成必需
                ↓
        赖氨酸 → 分子间交联必需
                ↓
        锌、铜 → 交联酶辅因子
```

### 4.2 维生素C协同

⚠️ TODO: 甘氨酸与维生素C协同促进胶原蛋白合成的直接临床研究

**理论基础**:
- 维生素C是脯氨酰羟化酶和赖氨酰羟化酶的必需辅因子
- 这些酶负责将脯氨酸和赖氨酸转化为羟脯氨酸和羟赖氨酸
- 羟化反应对胶原三螺旋的稳定性和分子间交联至关重要

**实践建议**:
- 胶原蛋白补充时配合维生素C 500-1000mg
- 维生素C可通过饮食（柑橘类、浆果）或补充剂获取

### 4.3 赖氨酸协同

⚠️ TODO: 赖氨酸与甘氨酸在胶原蛋白合成中的协同作用研究

**理论基础**:
- 赖氨酸是胶原蛋白分子间交联的关键氨基酸
- 赖氨酰氧化酶催化赖氨酸转化为羟赖氨酸，进而形成交联
- 交联是胶原纤维力学强度的分子基础

### 4.4 组织特异性分布

| 组织 | 胶原蛋白类型 | 代谢特点 |
|-----|------------|---------|
| 皮肤 (真皮) | Type I, III | 更新较慢 |
| 骨骼 | Type I | 更新周期长 |
| 肌腱 | Type I | 代谢活性低 |
| 韧带 | Type I | 代谢活性低 |
| 软骨 | Type II | 无血管，修复能力差 |
| 血管 | Type III | 需要弹性 |

### 5.3 甘氨酸与肌酸

甘氨酸参与肌酸合成:

```
甘氨酸 + 精氨酸 + 甲硫氨酸 → 胍基乙酸 → 肌酸
```

**问题**:
- 膳食甘氨酸对肌酸池的实际贡献量不明确
- 肌酸主要是内源性合成，膳食影响有限
- 不建议用甘氨酸替代肌酸补充

## 7. 安全性

### 7.1 总体安全性评价

**监管状态**: 甘氨酸被FDA认定为"Generally Recognized As Safe" (GRAS)

### 7.2 急性毒性

- 动物实验显示甘氨酸毒性极低
- LD50（大鼠，口服）: 约 7,930 mg/kg
- 人体无已知致死剂量

### 7.3 长期安全性

| 剂量范围 | 安全评估 | 不良反应 |
|---------|---------|---------|
| 3-10g/天 | 安全 | 罕见 |
| 10-30g/天 | 可能安全 | 轻微胃肠不适 |
| >30g/天 | 数据有限 | 可能腹泻、恶心 |

### 7.4 已知不良反应

| 不良反应 | 发生率 | 处理方式 |
|---------|-------|---------|
| 胃部不适 | 罕见 | 随餐服用 |
| 恶心 | 罕见 | 减量 |
| 腹泻 | 高剂量时可能出现 | 减量或分次服用 |
| 头痛 | 极罕见 | 停药观察 |

### 7.5 特殊人群

| 人群 | 建议 |
|-----|-----|
| 孕妇/哺乳期 | 食品级剂量（<10g/天）通常安全，缺乏专门研究 |
| 儿童 | 食品级剂量通常安全 |
| 肾功能不全 | 咨询医生（蛋白质代谢负担） |
| 肝脏疾病 | 咨询医生 |

## 10. 结论

### 10.1 睡眠改善

1. **证据现状**: 甘氨酸改善睡眠的证据为中等强度，主要来源于2000年代的小型研究
2. **推荐剂量**: 3-5g睡前服用
3. **机制**: 甘氨酸作为抑制性神经递质和体温调节剂，通过GABAergic机制促进睡眠 onset
4. **注意**: 需要2024-2026年更高质量研究验证

### 10.2 胶原合成支持

1. **理论支持**: 甘氨酸占胶原蛋白氨基酸的33%，是三螺旋结构的核心
2. **协同因素**: 维生素C（辅因子）、赖氨酸（交联）共同支持胶原合成
3. **实际应用**: 胶原蛋白肽（内含甘氨酸）是更直接的结缔组织支持方式

### 10.3 运动表现

⚠️ **证据极其有限**，不建议将甘氨酸作为运动表现增强剂

### 10.4 安全性

1. 甘氨酸安全性良好，GRAS认证
2. 3-10g/天剂量无已知严重不良反应
3. 高剂量（>20g/天）可能有轻微胃肠不适

### 10.5 实践建议

| 目的 | 建议 |
|-----|-----|
| 改善睡眠 | 3-5g甘氨酸睡前服用（可尝试） |
| 结缔组织支持 | 胶原蛋白肽 + 维生素C（优于单独甘氨酸）|
| 运动表现 | 不推荐 |
| 综合健康 | 均衡蛋白质饮食，甘氨酸作为补充 |
