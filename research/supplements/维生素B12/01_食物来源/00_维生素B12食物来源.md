## 摘要

维生素B12（钴胺素，Cobalamin）是人体必需的水溶性维生素，在DNA合成、红细胞生成、神经系统维护和能量代谢中发挥核心作用。由于维生素B12仅存在于动物性食品中，素食者和严格素食者面临较高的缺乏风险；老年人因胃酸分泌减少和内因子缺乏，吸收效率也显著下降。本综述系统梳理维生素B12的食物来源、吸收机制、每日摄入量建议、缺乏风险群体及运动员需求，为运动营养实践提供循证参考。

### 1.2 食物来源特点分析

#### 1.2.1 动物性食物是主要天然来源

维生素B12的唯一天然来源是动物性食物：

- **肉类和内脏**：含量最高，尤其是草饲动物的肝脏
- **海鲜类**：蛤蜊等贝类含量特别丰富
- **蛋奶类**：含量相对较低，但可作为日常来源

**生理机制**：维生素B12由微生物合成，动物通过食用含有维生素B12的微生物或互相捕食而在组织中积累B12。反刍动物（牛、羊）胃肠道中的细菌可以合成维生素B12。

#### 1.2.2 植物性食物几乎不含B12

- 未强化的植物性食物（蔬菜、水果、谷物、坚果）天然不含维生素B12
- 发酵食品（如纳豆、泡菜）可能含有微量B12，但含量不稳定
- 海藻类（如紫菜、螺旋藻）含有B12类似物（伪B12），无生物活性

## 2. 吸收机制

### 2.1 维生素B12吸收的生理过程

维生素B12的吸收是一个复杂的、多阶段的过程，需要多种蛋白质协同作用：

```
摄入（B12结合蛋白质）→ 胃酸分离 → 与R蛋白结合 → 胰蛋白酶释放
→ 内因子结合 → 回肠末端吸收 → 与转钴胺素结合 → 全身运输
```

### 2.2 内因子（Intrinsic Factor, IF）

**内因子是由胃壁细胞分泌的一种糖蛋白**，是维生素B12吸收的关键因子。

#### 2.2.1 内因子结构与功能

| 特性 | 描述 |
|-----|------|
| **来源** | 胃底和胃体部的壁细胞（parietal cells） |
| **分子量** | 约44-48 kDa |
| **分泌形式** | 与B12结合的复合物 |
| **结合能力** | 每分子内因子结合1分子B12 |
| **受体位置** | 回肠末端（ileum）的特异性受体） |

#### 2.2.2 内因子-B12复合物吸收过程

```
1. 胃酸（pH < 2）分离食物中的B12与蛋白质
2. 口腔和胃中的R蛋白与游离B12结合（保护B12）
3. 胰蛋白酶在十二指肠降解R蛋白
4. 内因子与B12结合形成IF-B12复合物
5. 回肠末端肠上皮细胞通过受体介导内吞吸收IF-B12
6. 内因子被降解，B12与转钴胺素II（TCII）结合进入血液
```

### 2.3 影响吸收效率的因素

#### 2.3.1 增强吸收的因素

| 因素 | 机制 | 吸收效率提升 |
|-----|------|-------------|
| **钙离子** | 稳定IF-B12复合物 | 约10-30% |
| **胃酸** | 分离B12与食物蛋白质 | 对蛋白结合B12至关重要 |
| **年轻年龄** | 胃功能正常 | 吸收效率约50-60%（生理剂量） |

#### 2.3.2 抑制吸收的因素

| 因素 | 机制 | 吸收效率下降 |
|-----|------|-------------|
| **胃酸分泌减少**（老年人/质子泵抑制剂） | 无法从蛋白质分离B12 | 下降30-70% |
| **内因子缺乏**（萎缩性胃炎/胃切除） | 无法形成IF-B12复合物 | 下降>95% |
| **回肠疾病/切除** | 吸收部位缺失 | 下降>95% |
| **胰腺功能不全** | R蛋白无法被降解 | 下降30-50% |
| **幽门螺杆菌感染** | 损伤壁细胞 | 间接影响 |
| **长期使用二甲双胍** | 钙离子通道干扰 | 下降约10-30% |
| **过量饮酒** | 胃肠道损伤 | 下降10-20% |

⚠️ **TODO**: 二甲双胍对B12吸收影响的分子机制需要更多研究。

### 2.4 吸收效率与剂量关系

维生素B12的吸收具有**剂量依赖性饱和**特性：

| 单次摄入剂量 | 吸收率 | 实际吸收量 |
|------------|-------|-----------|
| 1 μg | ~50% | 0.5 μg |
| 5 μg | ~20% | 1.0 μg |
| 10 μg | ~10% | 1.0 μg |
| 25 μg | ~5% | 1.25 μg |
| 100 μg | ~1% | 1.0 μg |
| 500 μg | ~0.2% | 1.0 μg |

**结论**：小剂量（1-5 μg）多次摄入比大剂量单次摄入更能有效提高B12水平。

## 4. 缺乏风险群体

### 4.1 素食者和严格素食者

#### 4.1.1 缺乏机制

```
植物性食物几乎不含B12 → 依赖强化食品/补充剂
↓
摄入不足 + 体内储存有限（2-5mg） → 3-5年后耗竭
```

#### 4.1.2 缺乏风险等级

| 素食类型 | B12缺乏风险 | 原因 |
|---------|------------|------|
| **严格素食** | 极高 | 无天然食物来源 |
| **奶素** | 中-高 | 奶制品B12含量有限 |
| **蛋素** | 中 | 蛋黄B12生物利用度约50% |
| **鱼素** | 低-中 | 鱼类提供一定B12 |

#### 4.1.3 缺乏表现

- 巨幼红细胞性贫血（megaloblastic anemia）
- 疲劳、虚弱
- 神经系统症状（麻木、刺痛、步态异常）
- 认知功能下降（记忆力减退、情绪变化）
- 胃肠道症状（食欲减退、消化不良）

### 4.4 恶性贫血（Addisonian anemia）

**定义**：由于自身免疫反应导致内因子抗体生成或壁细胞破坏，引起的维生素B12吸收障碍。

**特点**：
- 发病与自身免疫性胃炎相关
- 需要终身维生素B12注射或大剂量口服治疗
- 可伴发其他自身免疫疾病（甲状腺疾病、1型糖尿病）

## 5. 运动员需求

### 5.1 维生素B12在运动中的生理作用

#### 5.1.1 红细胞生成与氧气运输

维生素B12是DNA合成的关键辅因子，直接影响红细胞生成：

```
B12 → 参与DNA合成 → 红细胞成熟 → 血红蛋白合成 → 氧气运输能力
```

**运动意义**：充足的红细胞生成支持有氧运动能力（VO2max）。

#### 5.1.2 能量代谢

维生素B12参与以下代谢过程：

- 脂肪酸β-氧化（支链脂肪酸代谢）
- 氨基酸代谢（尤其是甲基丙二酰辅酶A途径）
- 糖异生支持

## 8. 结论

1. **维生素B12是能量代谢、神经系统功能和红细胞生成的关键营养素**

2. **动物性食物是唯一天然可靠来源**：内脏、贝类、肉类含量最高；素食者必须依赖强化食品或补充剂

3. **内因子介导的回肠吸收是B12吸收的核心途径**：任何影响内因子或回肠功能的因素都会显著降低吸收效率

4. **老年人和素食者是B12缺乏的高危群体**：需要特别关注摄入量和定期监测

5. **运动员B12需求与普通人群无显著差异**：但素食运动员必须主动管理B12摄入
