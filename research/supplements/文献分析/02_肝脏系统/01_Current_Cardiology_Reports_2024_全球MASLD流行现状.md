## 摘要核心内容

### 核心信息

**目的**：系统梳理代谢功能障碍相关脂肪性肝病（MASLD）的流行病学数据、命名演变（NAFLD→MASLD），及其与心血管疾病的关联。

**关键发现**：
1. 全球MASLD患病率高达**32.4%**，是最常见的慢性肝病
2. 命名更新（NAFLD→MASLD）减少了与酒精摄入的污名化关联，更强调代谢病因
3. MASLD是心血管疾病的**强力风险因素**，且独立于传统风险因素
4. 随着代谢综合征患病率上升，MASLD发病率预计将继续增加
5. 需要多学科团队协作、以整体方法应对

---

## 核心内容详解

### 1. 命名演变：从NAFLD到MASLD

| 旧名称 | 新名称 | 变化要点 |
|--------|--------|---------|
| NAFLD (Non-Alcoholic Fatty Liver Disease) | MASLD (Metabolic dysfunction-Associated Steatotic Liver Disease) | 强调代谢致病因素 |
| NASH | MASH | 同上 |
| — | MetALD（MASLD+酒精摄入） | 新增过渡类别 |

**为何更名？**
- NAFLD中的"non-alcoholic"标签让患者和临床医生误解为"排除性诊断"而非"代谢性疾病"
- 新命名直接指出代谢功能障碍是核心病因
- 减少与酒精相关污名的关联

**诊断标准变化**：
- **NAFLD**：排除过量饮酒和其他肝病原因 + 肝脏脂肪变性
- **MASLD**：存在肝脏脂肪变性 **+ 至少满足5个代谢风险因素之一**（超重/肥胖、2型糖尿病、代谢功能障碍、高血压、低HDL等）

### 2. 流行病学数据

| 指标 | 数据 |
|------|------|
| 全球MASLD患病率 | **32.4%**（成人） |
| 进展为MASH比例 | 约20-30%的MASLD患者 |
| 肝纤维化进展率 | 约7-10年进展一个阶段 |
| 死因构成 | 心血管疾病是MASLD患者的首要死因，**超过肝脏疾病本身** |

### 3. MASLD与心血管疾病（CVD）的关联

**核心结论**：MASLD是CVD的独立风险因素

机制：
```
MASLD → 促动脉硬化脂质谱改变（LDL↑、HDL↓、TG↑）
MASLD → 系统性炎症（IL-6、TNF-α↑）
MASLD → 胰岛素抵抗 → 糖尿病风险↑
MASLD → 脂肪因子失衡
         ↓
    心血管事件风险↑（独立于传统风险因素）
```

### 4. 疾病进展谱系

```
         代谢风险因素（肥胖/糖尿病）
                      │
                   肝脏脂肪变性
                      │
              单纯性脂肪肝（MASL）
                      │
              MASH（代谢相关脂肪性肝炎）
                      │
                 肝纤维化（F1→F4）
                      │
                  肝硬化
                      │
                 肝细胞癌（HCC）
```

**重要临床启示**：从MASLD到肝硬化的进展是渐进的，但心血管事件可在任何阶段发生——这意味着即使"只是"单纯性脂肪肝，心血管风险已经升高。

### 5. 未来趋势

- 全球代谢综合征患病率持续上升 → MASLD将继续增加
- 非侵入性纤维化评估手段（如ELF、FIB-4）在筛查中的价值日益重要
- 针对MASLD的新药研发活跃（PPAR激动剂、FXR激动剂、GLP-1等）

## 局限性

1. 患病率数据存在地区异质性（亚洲vs西方人群患病率差异显著）
2. MASLD诊断标准尚未完全统一（影像学vs活检 vs 生物标志物）
3. 命名更新后的真实世界研究数据仍然有限
4. 亚洲人群，特别是中国人群的大规模数据相对缺乏

---

## 对本综述项目的价值

**核心地位**：作为肝脏系统章节的**总论性文献**，提供：
- MASLD命名演变的权威解释
- 全球患病率的**最权威数字**（32.4%）
- 肝脏疾病命名体系（MASLD/MASH/MetALD）的标准化描述
- MASLD与心血管疾病的关联证据

## 个人学习笔记

**我学到的关键点：**
1. **MASLD患者的首要死因是心血管疾病**，而非肝脏本身——这彻底改变了对"脂肪肝只是肝的问题"的传统认知
2. 命名从NAFLD→MASLD不只是名词更换，而是**对疾病本质认知的范式转变**
3. 全球32.4%的成人患病率意味着**几乎每3个人就有1个脂肪肝**——这不是小众问题
4. MASLD患者即使还没有发展到肝硬化，心血管风险已经升高

- ELF、FIB-4等非侵入性纤维化评估在临床实践中的应用现状？
