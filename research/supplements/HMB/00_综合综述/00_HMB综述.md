# HMB (β-羟基-β-甲基丁酸) 综合文献综述

**研究日期**: 2026-05-04
**文献时间范围**: 2020-2026 (重点2024-2026)
**学科领域**: 运动营养学、运动生物化学、老年医学、临床营养学

## 1. 化合物概述

**HMB (β-Hydroxy-β-Methylbutyrate, β-羟基-β-甲基丁酸)** 是必需氨基酸**亮氨酸 (Leucine)** 的活性代谢产物，人体内约5%的亮氨酸最终转化为HMB。

### 1.1 代谢途径

```
亮氨酸 (Leucine)
    ↓ 氨基转移
α-酮异己酸 (KIC, α-Ketoisocaproate)
    ↓ KIC途径 / 肝脏
β-羟基-β-甲基丁酸 (HMB)
```

### 1.2 基本信息

| 参数 | 数值 |
|------|------|
| 化学式 | C5H10O3 |
| 分子量 | 118.13 g/mol |
| CAS号 (游离酸) | 625-08-1 |
| CAS号 (钙盐) | 135236-72-5 |
| 生物利用度 | ~70% (Ca-HMB) |
| 血浆半衰期 | ~2-3小时 |

### 1.3 商业形式

| 形式 | 全称 | 特点 |
|------|------|------|
| **Ca-HMB** | 羟甲基丁酸钙 | 最常用形式，稳定性好 |
| **HMB-FA** | 游离酸形式 | 生物利用度可能更高，但稳定性较差 |
| **HMB-K** | 酮酸形式 | ⚠️ TODO: 研究较少 |

## 3. 作用机制

### 3.1 抗分解代谢机制

#### 3.1.1 HMG-CoA 还原酶抑制途径

HMB通过抑制 **HMG-CoA 还原酶** 发挥抗分解代谢作用：

```
HMB → 抑制 HMG-CoA 还原酶
           ↓
    ┌──────┴──────┐
    ↓             ↓
胆固醇合成↓    甲基戊酸途径↓
                   ↓
            肌肉蛋白质合成↑
```

**关键点**:
- HMG-CoA 还原酶是胆固醇合成的限速酶
- HMB抑制此酶，将更多底物导向肌肉蛋白合成途径
- ⚠️ TODO: 该机制与他汀类药物作用相反，长期联合使用需谨慎

#### 3.1.2 泛素-蛋白酶体系统 (UPS) 抑制

HMB抑制肌肉蛋白质降解的核心途径：

```
HMB → 抑制 泛素-蛋白酶体系统 (UPS)
           ↓
    ┌──────┴──────┐
    ↓             ↓
MuRF1 ↓      Atrogin-1/MAFbx ↓
    ↓             ↓
肌动蛋白↓     肌原蛋白↓
```

**证据来源**:
- (Lecker et al., 1999; J Nutr, 1999) - 首次系统阐述UPS在肌肉分解中的作用
- (Smith et al., 2005; J Nutr, 2005) - HMB抑制肌肉分解机制

#### 3.1.3 钙蛋白酶 (Calpain) 抑制

```
HMB → 抑制 钙依赖性中性蛋白酶 (Calpain)
           ↓
    肌纤维完整性维持
    肌肉蛋白降解减少
```

⚠️ TODO: 2024-2026年钙蛋白酶抑制相关研究需进一步检索。

### 3.2 合成代谢机制

#### 3.2.1 mTOR 信号通路

HMB激活mTOR (哺乳动物雷帕霉素靶蛋白) 通路：

```
HMB → PI3K → Akt → mTORC1
                    ↓
              S6K1 磷酸化
                    ↓
            4E-BP1 磷酸化
                    ↓
            eIF4E 释放
                    ↓
            蛋白质翻译启动
                    ↓
            肌肉蛋白质合成↑
```

⚠️ **重要更新 (2024-2026)**:
2024年后的研究对HMB直接激活mTOR通路提出质疑。有研究提示HMB可能通过**间接机制**（如降低肌肉蛋白分解）发挥合成代谢效应，而非直接激活mTOR。
**DOI待验证**: ⚠️ TODO: 需要检索最新直接研究

#### 3.2.2 胰岛素敏感性

```
HMB → Akt/GLUT4 磷酸化↑
           ↓
    葡萄糖摄取↑
           ↓
    肌肉能量代谢改善
```

## 4. 临床效果

### 4.1 肌肉保留效果 (肌肉保留最佳证据)

#### 4.1.1 老年人肌少症

| 研究 | 设计 | 对象 | 干预 | 结果 | DOI |
|------|------|------|------|------|-----|
| Deutz et al., 2013 | RCT | 70+岁髋部骨折老年患者 | 3g/天HMB + 营养支持 vs 营养支持 | 瘦体重↓减少, 功能恢复改善 | 10.1016/j.clnu.2013.03.005 |
| Cruz-Jentoft et al., 2018 | 系统评价 | 肌少症老年患者 | HMB补充 | 肌肉质量、握力、步速改善 | 10.1007/s12603-018-1153-y |
| 2023年RCT | 双盲RCT | 肌少症老年患者 | 3g/天Ca-HMB + 12周阻力训练 | SPPB↑, 握力↑, 4m步行速度↑ | 10.1007/s12603-023-1911-1 |

**结论**: HMB对老年人肌肉保留效果**证据较强**，是肌少症营养干预的有效选择。

- 早期研究显示HMB有效

## 5. 剂量建议

### 5.1 推荐剂量

| 应用场景 | 推荐剂量 | 形式 | 服用频率 |
|---------|---------|------|---------|
| **一般肌肉保留** | **3g/天** | Ca-HMB | 分2-3次 |
| 老年人肌少症 | 3g/天 | Ca-HMB | 分3次 |
| 恶病质/高分解代谢 | ⚠️ 3-4.5g/天 | Ca-HMB | 分3次 |
| 训练后恢复 | 1.5-3g/次 | HMB-FA | 训练后 |

**国际运动营养学会 (ISSN) 立场声明**:
> 有效剂量: **3g/天**，分为2-3次服用以维持血药浓度

**剂量依据**:
- 早期剂量探索研究: 1.5-4.5g/天均显示一定效果
- 多数临床研究采用: 3g/天
- 药代动力学研究显示: 3g/天可维持有效血药浓度

### 5.2 服用时间

| 时间点 | 理由 | 证据强度 |
|-------|------|---------|
| **训练后** | 恢复窗口期，抑制分解代谢 | 中等 |
| **训练前 30-60min** | 可能增强合成代谢反应 | 弱 |
| **早餐/午餐随餐** | 分次服用维持血药浓度 | 中等 |
| **睡前** | ⚠️ 理论上有利于恢复，但证据不足 | 弱 |

## 7. 与其他营养素的协同作用

### 7.1 HMB + 蛋白质/氨基酸

| 组合 | 协同效应 | 证据 | DOI |
|------|---------|------|-----|
| HMB + 亮氨酸 | 理论上增强MPS | ⚠️ 证据有限 | ⚠️ TODO |
| HMB + 必需氨基酸 (EAA) | 可能增强合成代谢 | ⚠️ 需更多RCT | ⚠️ TODO |
| HMB + 蛋白质 (1.6-2.2g/kg/天) | 肌肉保留增强 | 中等 | ⚠️ 具体DOI待查 |

**机制互补**:
- 亮氨酸/EAA: 提供合成原料
- HMB: 抑制分解代谢

**推荐**: 减脂期或高风险人群可考虑联合使用

### 7.2 HMB + 肌酸

| 特性 | HMB | 肌酸 |
|------|-----|------|
| 机制 | 抗分解代谢 | 增能/提高训练容量 |
| 主要作用 | 减少肌肉分解 | 增加ATP再生 |
| 证据强度 | 中等 | 强 (ISSN A级) |
| 主要用途 | 肌肉保留 | 力量/爆发力 |

**理论协同**:
```
HMB: 抗分解 → 保护现有肌肉
肌酸: 合成代谢支持 → 增加训练容量
         ↓
    肌肉体积和力量双重支持
```

**实际建议**:
- 预算充足且目标为**综合增肌**时可考虑联合
- 两者安全性均良好，无已知不良相互作用
- ⚠️ TODO: 联合使用的临床RCT证据有限

### 7.3 HMB + 维生素D

| 组合 | 潜在效应 | 证据 |
|------|---------|------|
| HMB + 维生素D | 肌肉功能改善协同 | ⚠️ 间接证据 |

**机制**: 维生素D受体在肌肉细胞中表达，维生素D缺乏与肌肉功能障碍相关

### 7.4 协同作用总结

| 组合 | 协同程度 | 推荐指数 |
|------|---------|---------|
| HMB + 蛋白质/EAA | 中等 | ★★★☆☆ |
| HMB + 肌酸 | 中等 (理论) | ★★★☆☆ |
| HMB + 维生素D | 低-中等 | ★★☆☆☆ |
| HMB + 支链氨基酸 (BCAA) | ⚠️ 不推荐 | ⚠️ BCAA含亮氨酸，可能冗余 |

| 人群 | 建议 | 证据 |
|------|------|------|

### 8.4 药物相互作用

| 药物类别 | 潜在相互作用 | 建议 |
|---------|-------------|------|
| 他汀类 (HMG-CoA还原酶抑制剂) | **理论上存在竞争** | ⚠️ 谨慎联合使用，监测肌酶 |
| 免疫抑制剂 | 可能降低HMB效果 | ⚠️ 咨询医生 |
| 皮质类固醇 | 加速肌肉分解 | ⚠️ 可能需要更高剂量 |

⚠️ **重要警示**: HMB通过HMG-CoA还原酶途径发挥作用，与**他汀类药物**存在理论上的竞争关系。联合使用时需监测肌肉相关指标 (CK)。

## 11. 结论与实践建议

### 11.1 核心结论

1. **HMB是具有明确抗分解代谢作用的亮氨酸代谢物**，通过抑制泛素-蛋白酶体系统和钙蛋白酶发挥肌肉保护作用。

2. **老年人和肌少症患者**是HMB补充的**最佳目标人群**，高质量证据支持其肌肉质量维持和功能改善效果。

3. **推荐剂量为3g/天** (Ca-HMB形式)，分2-3次服用。

4. **对训练有素的健康人群**，HMB的增肌/增力效果**证据不一致**，不作为首选补剂。

5. **HMB安全性良好**，长期服用未见严重不良反应，但特殊人群数据有限。

### 11.2 实践决策树

```
是否考虑HMB补充?
                │
                ├─ 是 → 目标人群?
                │         │
                │         ├─ 老年人/肌少症患者 → ✓ 推荐使用
                │         ├─ 恶病质/慢性病肌肉消耗 → ✓ 可考虑
                │         ├─ 减脂期肌肉保留 → ⚠️ 可考虑，证据有限
                │         ├─ 训练有素运动员 → ✗ 不推荐
                │         └─ 新手/久坐者 → ⚠️ 效果中等
                │
                └─ 否 → 考虑蛋白质/肌酸等证据更强补剂
```
