# HMB（β-羟基-β-甲基丁酸酯）肌肉保护机制研究

## 摘要

HMB（β-Hydroxy β-Methylbutyrate，β-羟基-β-甲基丁酸酯）是亮氨酸的代谢产物，已被研究证明具有抑制肌肉蛋白分解和促进肌肉蛋白合成的作用。本文件综述HMB的代谢途径、分子作用机制、对不同训练水平人群的效果差异，以及最佳服用策略。

**关键词**: HMB, β-羟基-β-甲基丁酸酯, 肌肉蛋白合成, mTOR, ubiquitin-proteasome, 亮氨酸代谢

---

## 1. HMB代谢途径与作用机制

### 1.1 HMB的来源与代谢

HMB是必需氨基酸亮氨酸（Leucine）的天然代谢产物，约占亮氨酸代谢的5%。

```
亮氨酸 → α-酮异己酸(KIC) → HMB
         └→ α-酮甲基戊酸 → HMG-CoA → 胆固醇
```

**代谢分支点**:
- 约5%的亮氨酸经KIC途径转化为HMB
- HMB在体内以两种形式存在：游离HMB（HMBf）和HMB钙盐（Ca-HMB）

### 1.2 HMB的生理浓度

⚠️ TODO: 需要2024-2026年文献确认具体数值

- 基础血浆HMB浓度：约2-5 μmol/L
- 口服3g HMB后峰值：约150-200 μmol/L
- 半衰期：约2-3小时

### 1.3 HMB作用的双重途径

HMB通过两条主要途径发挥肌肉保护作用：

1. **抑制肌肉蛋白分解途径**（主要通过ubiquitin-proteasome系统）
2. **促进肌肉蛋白合成途径**（主要通过mTOR信号通路）

## 2. 抑制肌肉蛋白分解途径（Ubiquitin-Proteasome系统）

### 2.1 Ubiquitin-Proteasome系统概述

泛素-蛋白酶体系统（UPS）是骨骼肌蛋白分解的主要途径，负责分解：
- 肌原纤维蛋白（肌动蛋白、肌球蛋白）
- 结构蛋白
- 受损或错误折叠的蛋白质

### 2.2 HMB对UPS的抑制机制

⚠️ TODO: 需要2024-2026年文献验证以下分子机制

**已报道的抑制机制**:

1. **下调肌肉特异性E3泛素连接酶**
   - Atrogin-1/MAFbx表达下降
   - MuRF-1表达下降

2. **抑制UPS活性**
   - 蛋白酶体活性降低
   - 泛素结合蛋白减少

3. **稳定肌细胞膜**
   - 减少肌细胞损伤标志物释放
   - 维护肌纤维完整性

### 2.3 关键分子证据

| 分子标志物 | HMB处理效果 | 证据强度 |
|-----------|-----------|---------|
| Atrogin-1 mRNA | ↓ 30-50% | 中等 ⚠️ TODO |
| MuRF-1 mRNA | ↓ 20-40% | 中等 ⚠️ TODO |
| 20S蛋白酶体活性 | ↓ 15-25% | 较低 ⚠️ TODO |
| 泛素结合量 | ↓ 20-35% | 较低 ⚠️ TODO |

### 2.4 抗分解效应的生理意义

- 减少力量训练后的肌肉损伤
- 延缓老年性肌肉衰减（ sarcopenia）
- 防止长期卧床或制动期间的肌肉丢失

## 3. 促进肌肉蛋白合成途径（mTOR）

### 3.1 mTOR信号通路概述

mTOR（哺乳动物雷帕霉素靶蛋白）是肌肉蛋白合成的核心调控因子，整合营养、生长因子和机械刺激信号。

### 3.2 HMB对mTOR的激活机制

⚠️ TODO: 需要2024-2026年文献确认以下机制细节

**已报道的激活机制**:

```
HMB → 上调氨基酸转运体表达 → 增强亮氨酸摄取
     ↓
mTORC1磷酸化增强
     ↓
下游效应因子激活
     ├── p70S6K1磷酸化 → 蛋白翻译启动
     └── 4E-BP1失活 → eIF4E释放 → 翻译启动
```

### 3.3 HMB与亮氨酸的比较

| 指标 | HMB | 亮氨酸 |
|------|-----|--------|
| mTOR激活强度 | 较弱 ⚠️ TODO | 强 |
| 肌肉蛋白合成刺激 | 中等 | 强 |
| 抗分解作用 | 显著 ⚠️ TODO | 较弱 |
| 最佳剂量 | 3g/天 | 2-3g/天 |

### 3.4 促进蛋白合成的证据

⚠️ TODO: 需要2024-2026年高质量RCT验证

- **短期研究**：HMB补充后2-3小时肌肉蛋白合成率增加约20-30%
- **长期研究**：力量训练配合HMB，肌肉横截面积增加约5-10%
- **敏感人群**：老年人效果优于年轻人（年龄×训练交互作用）

## 4. 不同训练水平人群的效果差异

### 4.1 未经训练人群（初学者）

**效果预期：较明显**

- 新手在力量训练初期，肌肉蛋白合成响应较强
- HMB可能加速适应过程
- ⚠️ TODO: 2024-2026年研究证据有限

**推荐策略**：训练初期（0-3个月）可考虑补充

### 4.2 业余训练者（中等训练水平）

**效果预期：中等**

- 肌肉对HMB的响应可能下降
- 抗分解作用可能优于合成作用
- ⚠️ TODO: 需要更多交叉研究验证

**推荐策略**：重点放在训练质量和蛋白质摄入

### 4.3 高级训练者（高水平/竞技运动员）

**效果预期：较弱或无显著效果**

- 肌肉已高度适应，合成阈值提高
- ⚠️ TODO: 2024-2026年研究结论存在争议

**研究发现的不一致性**:
- 部分研究显示无额外增肌效果
- 部分研究显示训练适应性改善

**推荐策略**：
- 优先保证充足蛋白质（1.6-2.2g/kg/天）
- HMB作为可选补充，非必需

### 4.4 特殊人群

#### 4.4.1 老年人（肌少症预防）

**效果预期：较显著**

- 老年人肌肉蛋白合成能力下降（HMR - Muscle anabolic resistance）
- HMB可能部分逆转合成抵抗
- ⚠️ TODO: 需要2024-2026年大规模RCT确认

**证据等级**：中等（多项研究支持，但异质性较大）

#### 4.4.2 恶病质/消耗性疾病患者

**效果预期：临床意义显著**

- 用于艾滋病、癌症、COPD等消耗综合征
- ⚠️ TODO: 需个体化评估

#### 4.4.3 长期制动/卧床患者

**效果预期：保护性**

## 5. 最佳服用剂量与时机

### 5.1 推荐剂量

**标准剂量**：3g/天（HMB游离酸或钙盐均可）

**剂量范围**：2-3g/天

⚠️ TODO: 2024-2026年研究是否支持剂量调整（体重依赖）

| 体重范围 | 推荐剂量 |
|---------|---------|
| <70kg | 2.5g/天 |
| 70-90kg | 3g/天 |
| >90kg | 3.5g/天 ⚠️ TODO待验证 |

### 5.2 服用时机

#### 方案一：训练后服用（推荐）

```
训练结束 → 30分钟内 → HMB 3g + 蛋白质
```

**理由**：
- 利用训练后蛋白质合成窗口
- 协同刺激mTOR通路
- 抑制训练后分解代谢

#### 方案二：早晚分次服用

```
早餐后 → 1.5g
晚餐后 → 1.5g
```

**理由**：
- 维持血HMB浓度稳定
- ⚠️ TODO: 证据支持有限

#### 方案三：固定时间服用

```
晨起空腹 或 睡前
```

**理由**：⚠️ TODO: 证据支持有限

## 7. 结论与研究展望

### 7.1 主要结论

1. **分子机制**：HMB通过抑制UPS活性和激活mTOR通路发挥双重肌肉保护作用
2. **效果人群**：老年人和未经训练者效果较明显，高级训练者获益有限
3. **推荐剂量**：3g/天，训练后服用为主
4. **证据等级**：整体证据等级为低-中等，⚠️ TODO多项声明需2024-2026年文献验证
