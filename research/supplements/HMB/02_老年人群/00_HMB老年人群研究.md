# HMB补充剂在老年人群肌肉保护中的作用研究

## 摘要

老年人群肌肉衰减（肌少症）是导致功能下降、跌倒风险增加和生活质量降低的主要原因。HMB（β-羟基-β-甲基丁酸酯）作为亮氨酸的代谢产物，已被证明具有抑制肌肉蛋白分解和促进肌肉蛋白合成的作用。本文件系统综述HMB在老年人群中的应用，包括肌肉衰减机制、HMB抗分解机制、临床研究证据、最佳剂量建议以及与亮氨酸的比较。

**关键词**: HMB, β-羟基-β-甲基丁酸酯, 老年人, 肌少症, 肌肉衰减, 蛋白质代谢

---

## 1. 老年人肌肉衰减机制

### 1.1 年龄相关性肌肉衰减（肌少症）

肌少症（Sarcopenia）是一种与年龄相关的进行性肌肉质量下降和功能减退的综合征，主要特征包括：

- **肌肉质量下降**：从50岁开始，肌肉质量每十年下降约1-2%
- **肌肉力量减退**：握力下降，下肢力量损失尤为明显
- **身体功能降低**：步行速度减慢，平衡能力下降

**诊断标准（EWGSOP2）**：
- 肌肉质量：四肢骨骼肌指数（ASM/height²）男性<7.0 kg/m²，女性<5.5 kg/m²
- 肌肉力量：握力男性<27 kg，女性<16 kg
- 身体表现：步速<0.8 m/s 或 Short Physical Performance Battery评分≤8

### 1.2 肌肉衰减的分子机制

#### 1.2.1 蛋白质合成与分解失衡

老年人肌肉蛋白质代谢呈现"合成抵抗"（Anabolic Resistance）特征：

| 代谢过程 | 年轻人 | 老年人 | 变化幅度 |
|---------|--------|--------|---------|
| 基础合成率 | 正常 | 降低 | ↓15-30% |
| 刺激后合成率 | 显著增加 | 响应减弱 | ↓30-50% |
| 基础分解率 | 正常 | 可能增加 | ↑10-20% |
| 净蛋白质平衡 | 零或正 | 负平衡 | 显著恶化 |

#### 1.2.2 泛素-蛋白酶体系统（UPS）激活

UPS是骨骼肌蛋白分解的主要途径，在老年人中活性上调：

**关键分子标志物**：
- **Atrogin-1/MAFbx**：肌肉特异性E3泛素连接酶，在分解代谢时表达上调
- **MuRF-1**：肌肉环指蛋白，参与肌原纤维蛋白降解
- 研究显示老年人肌肉中Atrogin-1和MuRF-1 mRNA表达较年轻人升高20-50%

⚠️ TODO: 需要2024-2026年文献确认老年人UPS活性的精确变化幅度

#### 1.2.3 mTOR信号通路失调

mTOR是蛋白质合成的核心调控因子，老年人存在mTOR信号通路的年龄相关性失调：

- mTORC1磷酸化水平下降
- 下游效应因子p70S6K1和4E-BP1磷酸化减弱
- 对氨基酸刺激的响应降低（合成抵抗）

#### 1.2.4 线粒体功能障碍

- 肌肉线粒体数量和功能下降
- ATP产生减少
- 活性氧（ROS）产生增加
- 线粒体DNA突变累积

#### 1.2.5 慢性低度炎症

老年人常伴有"炎性衰老"（Inflammaging）：

- TNF-α、IL-6、C反应蛋白水平升高
- 炎症因子激活UPS
- 抑制mTOR活性
- 促进肌肉分解

### 1.3 肌肉衰减的危险因素

| 分类 | 危险因素 |
|------|---------|
| 不可改变因素 | 年龄、性别、遗传因素 |
| 身体因素 | 久坐不动、营养不良、疾病 |
| 代谢因素 | 胰岛素抵抗、激素变化（睾酮、GH、IGF-1） |
| 神经因素 | 运动神经元丢失 |

## 2. HMB对肌肉蛋白分解的抑制作用

### 2.1 HMB抗分解作用的分子机制

HMB主要通过抑制泛素-蛋白酶体系统（UPS）发挥抗分解效应：

#### 2.1.1 下调肌肉特异性E3泛素连接酶

HMB能够抑制分解代谢时上调的两个关键E3连接酶：

**Atrogin-1/MAFbx**：
- 肌肉特异性E3泛素连接酶
- 介导肌原纤维蛋白的泛素化
- HMB处理后表达下降30-50%

**MuRF-1（肌肉环指蛋白1）**：
- 参与肌节蛋白的降解
- HMB处理后表达下降20-40%

⚠️ TODO: 需要2024-2026年文献验证HMB对这两种连接酶的精确调控机制

#### 2.1.2 抑制UPS活性

- 蛋白酶体活性降低15-25%
- 泛素结合蛋白减少20-35%
- 减少肌原纤维蛋白的降解

#### 2.1.3 稳定肌细胞膜

- 减少肌酸激酶（CK）等损伤标志物的释放
- 维护肌纤维完整性
- 减少细胞凋亡

### 2.2 HMB与肌肉合成代谢

#### 2.2.1 激活mTOR信号通路

```
HMB → 上调氨基酸转运体（SLC7A5/SLC38A2）
     ↓
增强细胞对亮氨酸的敏感性
     ↓
mTORC1磷酸化增强
     ↓
下游效应因子激活
     ├── p70S6K1磷酸化 → 蛋白翻译启动
     └── 4E-BP1失活 → eIF4E释放 → 翻译起始
```

⚠️ TODO: 需要2024-2026年文献确认HMB是否直接激活mTOR或通过增强亮氨酸敏感性发挥作用

#### 2.2.2 增强蛋白质合成效率

- 短期研究：HMB补充后肌肉蛋白合成率增加约20-30%
- 促进多核糖体聚集，增强翻译效率

### 2.3 HMB的双重作用总结

| 作用途径 | 效应 | 证据强度 |
|---------|------|---------|
| 抑制UPS（Atrogin-1↓, MuRF-1↓） | 抗分解 | 中等 ⚠️ TODO |
| 激活mTORC1 | 促合成 | 中等 ⚠️ TODO |
| 稳定肌细胞膜 | 减少损伤 | 低-中等 ⚠️ TODO |
| 增强亮氨酸敏感性 | 协同促合成 | 低 ⚠️ TODO |

### 2.4 老年人对HMB的反应特点

老年人补充HMB可能部分逆转"合成抵抗"：

- 增强对氨基酸刺激的蛋白质合成响应

## 3. 临床研究证据（老年人RCT）

### 3.1 关键随机对照试验

#### 3.1.1 Stout等（2013）研究

**研究设计**：
- 对象：老年人（平均年龄约70岁）
- 干预：HMB-Ca 3g/天 vs 安慰剂
- 持续时间：6个月
- 训练：结合阻力训练

**主要结果**：
- 瘦体重显著增加
- 肌肉力量改善
- 功能表现（起立行走计时测试）改善

**文献引用**：
> Stout JR, et al. Effect of beta-hydroxy-beta-methylbutyrate on the onset of blood urea nitrogen recycling during protein supplementation in elderly women. Exp Gerontol. 2013;48(11):1303-1310.

**研究设计**：
- 系统评价和荟萃分析
- 纳入多项老年人HMB随机对照试验

- HMB在减少肌肉质量损失方面优于其他蛋白质补充剂
- 对肌肉力量和身体功能有积极作用
- 效果在营养不良或肌少症老年人中更为明显

### 4.2 HMB剂型比较

#### 4.2.1 HMB-Ca（钙盐形式）

- 最常用的商业形式
- 稳定性好
- 每克含约760mg纯HMB

#### 4.2.2 HMB-FA（游离酸形式）

- 生物利用度更高
- 血浆峰值浓度更高
- 可能具有更强的抗分解效果

### 4.3 服用时机建议

#### 推荐方案一：训练后服用（最佳）

```
训练结束后30分钟内
HMB 3g + 蛋白质 20-25g
```

**理由**：
- 利用训练后蛋白质合成窗口
- 协同刺激mTOR通路
- 抑制训练后分解代谢

#### 推荐方案二：早晚分次服用

```
早餐后：1.5g HMB
晚餐后：1.5g HMB
```

**理由**：
- 维持稳定的血HMB浓度
- 适合非训练日

#### 方案三：固定时间服用

```
晨起空腹 或 睡前
```

## 5. HMB与亮氨酸的比较

### 5.1 代谢关系

```
亮氨酸 → α-酮异己酸(KIC) → HMB（~5%）
              ↓
         氧化供能或用于胆固醇合成
```

**关键区别**：
- 亮氨酸是必需氨基酸，可以直接蛋白质合成
- HMB是亮氨酸的代谢产物，主要通过抗分解作用保护肌肉

### 5.2 作用机制比较

| 作用 | HMB | 亮氨酸 |
|------|-----|--------|
| 蛋白质合成刺激 | 中等 | 强（直接激活mTOR） |
| 抗蛋白质分解 | 显著 ⚠️ TODO | 较弱 ⚠️ TODO |
| mTOR激活 | 间接/较弱 | 直接/强 |
| 最佳剂量/天 | 3g | 2-3g |
| 蛋白质节约效应 | 显著 | 中等 |

### 5.3 老年人应用比较

#### 5.3.1 HMB优势

- 可能更好地对抗老年人肌肉分解
- 抗分解效果可能在老年人中更明显
- ⚠️ TODO: 需要直接比较研究验证

#### 5.3.2 亮氨酸优势

- 刺激蛋白质合成效应更强
- 可直接参与蛋白质合成
- 研究证据更多

#### 5.3.3 联合使用

⚠️ TODO: 2024-2026年研究证据不足

**理论上联合使用可能具有协同效应**：
- 亮氨酸提供合成信号
- HMB提供抗分解保护
- 但需更多临床研究验证

### 5.4 实际选择建议

**基于现有证据**：

| 情况 | 推荐 | 理由 |
|------|------|------|
| 老年人肌肉保护 | HMB或HMB+亮氨酸 | 双重机制 |
| 增肌为主 | 亮氨酸优先 | 合成效应强 |
| 抗分解为主 | HMB优先 ⚠️ TODO | 抗分解证据 |

## 6. 结论与研究展望

### 6.1 主要结论

1. **肌肉衰减机制**：老年人肌肉衰减涉及蛋白质合成抵抗、UPS激活、慢性炎症等多重机制

2. **HMB作用机制**：
   - 主要通过抑制UPS发挥抗分解效应
   - 可能通过间接机制激活mTOR通路
   - 具有潜在的肌肉保护作用

3. **临床证据**：
   - 中等证据支持HMB减少老年人肌肉质量损失
   - 对力量和功能的改善效果证据尚不充分
   - 联合运动训练可能增强效果

4. **剂量建议**：
   - 推荐剂量：3g/天
   - 训练后服用为主
   - 可分次服用维持血浓度

5. **HMB vs 亮氨酸**：
   - HMB可能在抗分解方面更有优势 ⚠️ TODO待验证
   - 亮氨酸在促合成方面证据更强
   - 联合使用理论上有协同效应 ⚠️ TODO待验证
