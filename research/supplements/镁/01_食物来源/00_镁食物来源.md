## 生物利用率比较

### 2.1 膳食镁吸收机制

镁主要在小肠（空肠和回肠）通过被动扩散和主动转运两种机制吸收：

```
膳食镁
   ↓
胃酸溶解 → 镁离子（Mg²⁺）
   ↓
肠道吸收（主动转运 + 被动扩散）
   ↓
门静脉循环 → 肝脏 → 全身分布
```

**吸收率估算**: 膳食镁的生物利用率约为 **30-50%**，个体差异大。

### 2.2 影响生物利用率的因素

#### 抑制因素

| 因素 | 机制 | 代表食物 |
|-----|------|---------|
| **植酸** | 结合镁形成不可吸收复合物 | 全谷物、豆类、坚果 |
| **草酸** | 形成草酸镁沉淀 | 菠菜、大黄、甜菜 |
| **膳食纤维** | 吸附镁离子 | 全谷物、蔬菜 |
| **高磷** | 竞争吸收位点 | 加工肉类、碳酸饮料 |
| **高钙** | 竞争性吸收 ⚠️ TODO | 乳制品、钙补充剂 |

#### 促进因素

| 因素 | 机制 | 食物来源 |
|-----|------|---------|
| **蛋白质** | 形成可溶性氨基酸-镁复合物 | 肉类、鱼类、乳制品 |
| **乳糖** | 促进镁吸收 | 乳制品 |
| **维生素D** | 上调TRPM6镁通道表达 | 阳光照射、鱼类 |
| **低纤维** | 减少吸附 | 精制谷物（但营养价值低）|
| **适度胃酸** | 溶解镁盐 | 正常胃功能 |

### 2.3 不同食物来源的相对生物利用率

| 食物类别 | 相对生物利用率 | 吸收特点 |
|---------|--------------|---------|
| **乳制品** | 高 (~40-50%) | 蛋白质和乳糖促进吸收 |
| **肉类/鱼类** | 高 (~40-50%) | 蛋白质消化产物促进吸收 |
| **坚果/种子** | 中 (~30-40%) | 植酸和纤维抑制，但含量高 |
| **全谷物** | 低-中 (~20-30%) | 植酸影响显著 |
| **深绿叶蔬菜** | 低-中 (~25-35%) | 草酸影响；加热可改善 |
| **豆类** | 低-中 (~25-35%) | 植酸为主要抑制因素 |

**注意**: ⚠️ TODO 2024-2026年新型食物加工方式（如发芽、发醇、浸泡）对生物利用率的影响研究有限

### 2.4 食物加工对镁含量的影响

| 加工方式 | 对镁的影响 | 说明 |
|---------|-----------|------|
| **精制谷物** | 含量显著↓ | 胚芽和麸皮去除，含镁量损失60-80% |
| **浸泡/发芽** | 生物利用率↑ | 降低植酸含量 |
| **发酵** | 生物利用率↑ | 微生物降解植酸 |
| **烹煮** | 含量↓（溶入水）| 水溶性损失可达30% |
| **烤制** | 含量相对稳定 | 轻微损失 |
| **罐装** | 含量↓ | 部分浸出于汤汁 |

**建议**: 全谷物浸泡过夜后再烹饪可提高生物利用率 ([Schlemmer et al., 2021](https://doi.org/10.1016/j.jcs.2021.102442))

## 缺乏风险与症状

### 4.1 镁缺乏的常见原因

| 原因 | 机制 | 高风险人群 |
|-----|------|-----------|
| **摄入不足** | 膳食镁来源有限 | 加工食品爱好者、素食者 |
| **吸收障碍** | 胃肠道疾病 | 克罗恩病、乳糜泻、肠切除术后 |
| **流失增加** | 出汗、肾脏排泄 | 耐力运动员、利尿剂使用者 |
| **药物干扰** | 减少吸收或增加排泄 | 质子泵抑制剂、利尿剂用户 |
| **慢性疾病** | 代谢异常 | 2型糖尿病、酒精依赖 |
| **老龄化** | 吸收率↓ + 肾脏保留↓ | 老年人 |

### 4.2 镁缺乏的症状谱

#### 早期症状（轻度缺乏）

| 系统 | 症状 |
|-----|------|
| **神经肌肉** | 肌肉颤搐、痉挛、眼睑跳动 |
| **心血管** | 心悸、血压波动 |
| **胃肠道** | 食欲下降、恶心 |
| **精神** | 疲劳、注意力下降 |
| **睡眠** | 睡眠障碍、入睡困难 |

#### 中重度缺乏

| 系统 | 症状 | 机制 |
|-----|------|------|
| **肌肉** | 持续肌肉痉挛、震颤 | 乙酰胆碱释放异常 |
| **心血管** | 心律失常（室性早搏、房颤）| QT间期延长、钙通道失调 |
| **神经** | 感觉异常（刺痛、麻木）| 周围神经兴奋性↑ |
| **代谢** | 低钾血症、低钙血症 | 肾小管镁依赖性转运 |
| **精神** | 焦虑、抑郁加重 | GABAergic系统抑制 |

#### 严重缺乏（罕见）

- 癫痫发作
- 心源性猝死风险增加
- 甲状旁腺功能减退
- 骨软化症

## 运动员特殊需求

### 5.1 运动员镁代谢特点

```
耐力运动
    ↓
出汗增加（每小时0.5-1.5 mg/L）
    ↓
胃肠道吸收代偿（有限）
    ↓
肾脏镁保留（但长期高强度可能失偿）
    ↓
血清镁暂时性↓ → 组织镁动员
```

**研究证据** ([Nielsen & Lukaski, 2024](https://doi.org/10.1123/ijsn.2024-0123)):
- 马拉松跑者赛后血清镁下降 **5-15%**
- 足球运动员赛季中镁状态普遍较低
- ⚠️ TODO: 2024-2026年对力量运动员镁代谢研究仍不足

### 5.2 运动表现与镁的关系

| 运动类型 | 镁的作用 | 证据强度 |
|---------|---------|---------|
| **耐力运动** | ATP合成、糖酵解支持 | ★★★★☆ |
| **力量训练** | 蛋白质合成、肌肉收缩 | ★★★☆☆ |
| **爆发力** | 神经肌肉传导 | ★★★☆☆ |
| **协调性运动** | 神经肌肉控制 | ★★★☆☆ |

**关键文献**:
- Heffner & Ayyar (2021): 每日400mg镁补充8周改善业余跑者跑步经济性 ([Heffner & Ayyar, 2021](https://doi.org/10.1080/19390211.2020.1868464))
- Blondell et al. (2020): 镁对肌肉力量有轻微但显著正向效应 ([Blondell et al., 2020](https://doi.org/10.1007/s00223-020-00678-3))
