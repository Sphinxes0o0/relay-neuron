## 摘要

硒是人体必需的微量元素，在维持甲状腺功能、抗氧化防御和免疫调节中发挥核心作用。作为第21种氨基酸——硒代半胱氨酸（selenocysteine）的关键成分，硒是25种硒蛋白的生物合成基础。本综述系统阐述硒的分子机制、甲状腺激素代谢、抗氧化防御系统及免疫功能调节，并就运动员群体的特殊需求、自然食物来源及安全补充剂量提供循证建议。

## 1. 基础生物学机制

### 1.1 硒的化学形式与代谢

| 形式 | 特性 | 吸收率 | 备注 |
|------|------|--------|------|
| **硒代蛋氨酸（Selenomethionine）** | 主要有机形式 | ~90% | 取代蛋氨酸掺入蛋白质 |
| **硒代半胱氨酸（Selenocysteine）** | 第21种氨基酸，活性中心 | 不直接吸收 | 蛋白质合成后释放 |
| **亚硒酸钠（Sodium Selenite）** | 无机形式 | 40-60% | 还原为硒化物后利用 |
| **硒酸钠（Sodium Selenate）** | 无机形式 | ~85% | 代谢为亚硒酸盐 |

**吸收机制**：
- 硒代蛋氨酸通过蛋氨酸转运通道吸收
- 无机硒以简单扩散或主动运输方式吸收
- 主要吸收部位：十二指肠

### 1.2 硒代半胱氨酸与25种硒蛋白

硒的生物学功能通过**硒蛋白（Selenoproteins）**实现。人类已识别25种功能性硒蛋白：

**关键硒蛋白及其功能**：

| 硒蛋白 | 功能 | 生理意义 |
|--------|------|----------|
| **谷胱甘肽过氧化物酶1-4（GPx1-4）** | 抗氧化防御 | 清除H₂O₂和脂质过氧化物 |
| **硫氧还蛋白还原酶（TrxR1-3）** | 氧化还原调节 | 维持硫氧还蛋白还原状态 |
| **碘甲状腺原氨酸脱碘酶（DIO1, DIO2, DIO3）** | 甲状腺激素代谢 | T4→T3、T3→T2转化 |
| **硒蛋白P（SelP）** | 硒运输与抗氧化 | 血浆主要硒载体 |
| **硒蛋白K（SelK）** | 免疫功能 | 内质网应激响应 |
| **硒蛋白S（SelS）** | 炎症调节 | 内质网相关降解 |
| **硒蛋白W（SelW）** | 肌肉功能 | 肌肉代谢相关 |

### 1.3 核心分子机制

```
硒代蛋氨酸摄入
       │
       ↓（肝脏代谢）
硒代半胱氨酸（Sec）合成
       │
       ├─→ 掺入硒蛋白（GPx、TrxR、DIO等）
       │
       └─→ 释放活性硒（Se²⁻）

活性硒的生物学作用：
       │
       ├─→ 抗氧化防御（GPx催化H₂O₂还原）
       │
       ├─→ 甲状腺激素转化（DIO介导T4→T3）
       │
       ├─→ 免疫调节（硒蛋白K介导免疫细胞功能）
       │
       └─→ 蛋白质合成（Sec掺入多肽链）
```

## 2. 甲状腺功能调节

### 2.1 硒与甲状腺激素合成

甲状腺是人体硒含量最高的器官（每克组织含硒量最高）。硒在甲状腺功能中发挥多重作用：

**碘甲状腺原氨酸脱碘酶（DIO）系统**：

| 酶 | 位置 | 催化反应 | 生理作用 |
|----|------|----------|----------|
| **DIO1** | 肝脏、肾脏、甲状腺 | T4→T3；rT3→T2 | 外周T4活化 |
| **DIO2** | 骨骼肌、心脏、脑、垂体 | T4→T3 | 局部T3生成 |
| **DIO3** | 胎盘、脑、眼睛 | T3→T2；T4→rT3 | T3灭活 |

**核心反应**：
```
T4 + 2e⁻ + Se → T3 + I⁻（DIO2催化）
          ↑
    硒代半胱氨酸（Sec）位于催化活性中心
```

- 女性患病率是男性的8-10倍
- 全球患病率约2-5%
- 是导致甲状腺功能减退的最常见原因

## 3. 抗氧化防御系统

### 3.1 谷胱甘肽过氧化物酶（GPx）家族

GPx是硒最重要的抗氧化功能载体。

**GPx家族成员**：

| 成员 | 组织分布 | 底物 | 生理作用 |
|------|----------|------|----------|
| **GPx1** | 肝脏、红细胞、肾脏 | H₂O₂ | 细胞内H₂O₂清除 |
| **GPx2** | 胃肠道上皮 | H₂O₂、脂质过氧化物 | 肠道屏障保护 |
| **GPx3** | 血浆、肾脏分泌 | H₂O₂、脂质过氧化物 | 血浆抗氧化 |
| **GPx4** | 睾丸、脑、心脏 | 脂质过氧化物（PHGPx） | 膜脂质过氧化保护 |

### 3.2 核心催化机制

**GPx催化的还原反应**：
```
H₂O₂ + 2GSH → 2H₂O + GSSG
         ↑
    硒代半胱氨酸（Sec）位于催化活性中心
         ↑
    氧化型Sec（Se-OH）→ 还原型Sec（Se⁻）
         ↑
    谷胱甘肽（GSH）提供电子
```

**机制要点**：
- Sec的-SeH基团提供活性巯基
- Sec的氧化还原电位使反应速率比Cys高1000倍
- GPx4（磷脂氢过氧化物酶）对防止细胞膜脂质过氧化至关重要

## 4. 免疫功能调节

### 4.1 硒蛋白K（Selenoprotein K）

**SelK的主要功能**：
- 表达于免疫细胞（巨噬细胞、T细胞、B细胞）
- 调节内质网钙离子通道
- 影响免疫细胞活化和增殖

**免疫调节机制**：
```
SelK → ER钙离子通道调节 → 钙信号 → 免疫细胞活化
            │
            ├─→ 巨噬细胞吞噬能力
            │
            ├─→ T细胞增殖和细胞因子分泌
            │
            └─→ B细胞抗体产生
```

| **蛋白质** | 14g | 0.7g |

### 6.2 补硒效能

⚠️ **TODO**: 2024-2026年生物利用度研究待验证

**实证数据**：
- 6-8粒巴西坚果 ≈ 200-400μg硒
- 持续摄入8周可显著提高血硒水平和GPx活性
- 生物利用度与硒代蛋氨酸补充剂相当

**与补充剂比较**：

| 来源 | 优势 | 劣势 |
|------|------|------|
| **巴西坚果** | 天然、含其他营养素、低成本 | 硒含量不稳定、需注意过敏 |

⚠️ **注意**：不同产地巴西坚果硒含量差异极大（10-50倍），难以精确控制摄入量

### 7.1 推荐摄入量

### 7.2 上限与毒性

| 阈值 | 剂量 | 症状/体征 |
|------|------|----------|
| **安全上限（UL）** | 400μg/天 | 长期摄入上限 |
| **亚急性毒性** | 400-800μg/天 | 恶心、食欲减退 |
| **慢性中毒（硒中毒）** | >800μg/天 | 脱发、指甲脆裂 |
| **严重中毒** | >1000μg/天 | 神经系统症状、心脏损害 |

**硒中毒（Selenosis）症状**：
- 头发脱落
- 指甲变脆、断裂
- 胃肠道症状（恶心、腹泻）
- 皮疹
- 疲劳、情绪改变
- 大蒜样呼吸气味（特征性）

⚠️ **警告**：运动员使用含硒复合维生素时需计算总剂量，避免累积超量

## 9. 争议与局限性

### 9.1 前列腺健康争议

⚠️ **TODO**: SELECT试验后续分析（2024-2026）需关注

**历史背景**：
- 早期观察性研究显示血硒水平与前列腺癌风险呈负相关
- SELECT试验（Selenium and Vitamin E Cancer Prevention Trial）发现**补充硒（200μg/天）或维生素E对前列腺癌预防无益**
- 部分分析显示高剂量硒可能**增加**某些亚组的前列腺癌风险

**当前共识**：
- 不推荐补硒作为前列腺癌预防手段
- 已有前列腺癌患者应避免高剂量硒补充

### 9.2 证据局限性

1. **研究质量参差**：许多硒研究样本量小、随访时间短
2. **基线硒状态影响**：低硒人群补充效果好，高硒人群补充收益有限
3. **形态差异**：不同硒形态的生物效应可能不同
4. **个体差异**：遗传多态性影响硒代谢（如SEP generality polymorphisms）

## 10. 结论与建议

### 核心结论

1. **硒是必需微量元素，通过25种硒蛋白发挥生理功能**
2. **甲状腺是硒含量最高的器官，硒对甲状腺激素代谢至关重要**
3. **GPx酶系统是硒依赖性抗氧化防御的核心**
4. **运动员因高氧化应激可能存在增加的需求**
5. **巴西坚果是有效的天然硒来源（6-8粒≈200μg）**
6. **55-200μg/天是安全有效的补充范围，UL为400μg/天**
7. **硒代蛋氨酸吸收率优于亚硒酸钠**

### 分层建议

| 人群 | 建议 |
|------|------|
| **健康成人** | 优先食物补充（海产品、巴西坚果）；如需补剂，≤100μg/天 |
| **运动员** | 评估饮食；如缺乏，100-200μg/天；优先天然来源 |
| **桥本甲状腺炎患者** | 在内分泌科指导下100-200μg/天；监测抗体和甲状腺功能 |
| **孕妇/哺乳期** | RNI 60μg/天；UL 400μg/天；不建议超量补充 |
| **老年人** | 评估吸收功能；注意多重用药相互作用 |
