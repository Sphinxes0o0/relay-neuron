## 2. 豌豆蛋白氨基酸谱

### 2.1 氨基酸组成（g/100g蛋白质）

| 氨基酸类别 | 豌豆蛋白 | 乳清蛋白 | WHO成人需求模式 |
|-----------|---------|---------|----------------|
| **必需氨基酸（EAA）** | | | |
| 亮氨酸（Leu） | 8.0-8.5 | 10.5 | 6.6 |
| 异亮氨酸（Ile） | 4.5-5.0 | 6.5 | 3.0 |
| 缬氨酸（Val） | 5.0-5.5 | 6.0 | 3.9 |
| 赖氨酸（Lys） | 7.0-7.5 | 9.5 | 5.8 |
| 苏氨酸（Thr） | 4.0-4.5 | 7.0 | 2.5 |
| 色氨酸（Trp） | 1.0-1.2 | 1.8 | 0.6 |
| 苯丙氨酸（Phe） | 5.5-6.0 | 4.5 | 3.2 |
| 蛋氨酸（Met） | 1.0-1.5 | 2.8 | 1.6 |
| **非必需氨基酸（NEAA）** | | | |
| 精氨酸（Arg） | 8.5-9.0 | 2.8 | - |
| 谷氨酰胺（Gln） | 16.5-17.0 | 9.5 | - |
| 天冬酰胺（Asn） | 11.5-12.0 | 11.0 | - |
| 脯氨酸（Pro） | 4.5-5.0 | 6.0 | - |
| **支链氨基酸（BCAA）总计** | 17.5-18.5 | 23.0 | 13.5 |

**数据来源**：USDA FoodData Central、([Laguna et al., 2023](https://www.doc88.com/p-99839586778800.html)) 体外胃肠消化研究

### 2.2 氨基酸谱分析

**优势氨基酸**：
- **精氨酸（Arg）**：豌豆蛋白含量约为乳清的3倍（8.5-9.0g vs 2.8g/100g蛋白）
  - 功能：一氧化氮前体，可能改善肌肉血流灌注
  - ⚠️ TODO：精氨酸对运动表现的直接促进效果尚需更多RCT证实

- **赖氨酸（Lys）**：含量相对较高（7.0-7.5g/100g蛋白）
  - 功能：钙吸收、胶原蛋白合成、免疫功能

**限制性氨基酸**：
- **蛋氨酸+半胱氨酸（Met+Cys）**：第一限制性氨基酸
  - 豌豆蛋白评分约为0.6-0.7（相对于WHO模式）
  - 建议：与谷物蛋白搭配食用可互补

### 2.3 亮氨酸阈值与肌肉蛋白合成（MPS）

| 蛋白质类型 | 亮氨酸含量（%） | 达到2.5g亮氨酸阈值所需蛋白质量 |
|-----------|---------------|------------------------------|
| 豌豆蛋白 | 8.0-8.5% | ~29-31g |
| 乳清蛋白 | 10.5-11.0% | ~23-24g |
| 大豆蛋白 | 7.8-8.0% | ~31-32g |

**关键发现**：
- 亮氨酸是触发mTOR通路激活的关键氨基酸 ([McKendry et al., 2024](https://www.sciencedirect.com/science/article/pii/S0002916524004726))
- 豌豆蛋白需要摄入约25-30g才能达到等效的亮氨酸阈值
- ⚠️ TODO：老年人（ anabolic resistance人群）的最佳亮氨酸摄入量是否需要调整尚需研究

### 3.2 机制分析

**豌豆蛋白肌肉合成机制**：

```
氨基酸摄入 → 氨基酸吸收 → 血浆氨基酸峰值 →
→ mTORC1通路激活 → 蛋白质翻译启动 → 肌肉蛋白合成（MPS）
```

**豌豆蛋白特点**：
- 消化速率：中等（慢于乳清，但快于酪蛋白）
- 氨基酸释放曲线：适合作为"餐后"蛋白源
- 精氨酸含量高：可能通过改善血流增强氨基酸输送

### 3.3 与其他植物蛋白的比较

| 蛋白质类型 | PDCAAS | 亮氨酸（%） | BCAA（g/100g） | 限制性氨基酸 |
|-----------|--------|------------|----------------|--------------|
| 豌豆蛋白 | 0.82-0.89 | 8.0-8.5 | 17.5 | 蛋氨酸+半胱氨酸 |
| 大豆蛋白 | 0.92-0.98 | 7.8-8.0 | 16.5 | 蛋氨酸 |
| 大米蛋白 | 0.50-0.60 | 7.5-8.0 | 15.0 | 赖氨酸 |
| 小麦蛋白 | 0.40-0.50 | 2.5-3.0 | 6.0 | 赖氨酸 |
| 麻蛋白 | 0.85-0.90 | 8.5-9.0 | 19.0 | 赖氨酸 |

**结论**：豌豆蛋白在植物蛋白中具有较高的营养价值评分，与大豆蛋白接近。

## 4. 植物蛋白优势

### 4.1 营养学优势

| 优势类别 | 具体描述 | 证据等级 |
|---------|---------|---------|
| **无乳糖** | 完全适合乳糖不耐受人群（约68%全球人口） | 高 |
| **无酪蛋白** | 避免酪蛋白相关消化问题 | 高 |
| **低饱和脂肪** | 几乎不含饱和脂肪（<0.5g/100g） | 高 |
| **无胆固醇** | 植物来源，零胆固醇 | 高 |
| **膳食纤维** | 含2-4%纤维（取决于产品纯度） | 高 |
| **环境可持续性** | 碳足迹约为乳清的1/5-1/10 | 中 |
| **伦理友好** | 适合纯素食/素食饮食 | 高 |

|---------|------------|---------|------|

### 4.3 特殊人群适用性

**适合使用豌豆蛋白的人群**：
- 纯素食/素食运动员
- 乳糖不耐受者
- 牛奶蛋白过敏者（豆类过敏除外）
- 追求低饱和脂肪摄入者
- 关注环境可持续性的消费者
- 更年期前后女性（低雌激素相关肌肉流失风险）

**⚠️ TODO**：孕妇、哺乳期女性、儿童的豌豆蛋白安全性数据有限，需要更多研究。

## 附录：常见问题

**Q1：豌豆蛋白能否完全替代乳清蛋白用于增肌？**

A1：基于现有证据，在阻力训练背景下，每日摄入1.6-2.2g/kg蛋白质时，豌豆蛋白与乳清蛋白在肌肉厚度和力量增加方面无显著差异。理论上，为达到等效亮氨酸阈值，豌豆蛋白每次摄入量应略高于乳清（25-30g vs 20-25g）。

**Q2：豌豆蛋白的最佳搭配是什么？**

A2：豌豆蛋白+谷物蛋白（如大米蛋白）可实现氨基酸互补，提高整体蛋白质价值评分。也可搭配麻蛋白或大豆蛋白。

**Q3：豌豆蛋白会导致胀气吗？**

A3：部分人群可能出现胀气症状，这与α-半乳糖苷糖（棉子糖/水苏糖）含量有关。选择高纯度分离蛋白或补充α-半乳糖苷酶可减轻症状。
