## 2. 各成分作用机制总览

### 2.1 软骨合成与代谢调节

```
软骨基质组成:
    ├── 胶原纤维 (II型胶原占90%以上)
    ├── 蛋白聚糖 (Aggrecan + 透明质酸)
    └── 软骨细胞 (chondrocytes)

关节健康补剂对软骨的作用靶点:
    │
    ├── 氨基葡萄糖 ──→ 蛋白聚糖合成底物 ──→ 软骨基质补充
    ├── 硫酸软骨素 ──→ 抑制MMP酶活性 ──→ 减少软骨降解
    ├── II型胶原蛋白 ──→ 软骨结构蛋白 ──→ 维持软骨弹性
    ├── MSM ──→ 硫供体 ──→ GAG/胶原合成辅因子
    └── 透明质酸 ──→ 滑液成分 ──→ 营养扩散/润滑
```

### 2.2 炎症调节通路

```
炎症信号通路 ──→ 软骨降解 ──→ OA进展

抗炎成分作用位点:
    │
    ├── NF-κB抑制:
    │     ├── 姜黄素 (强)
    │     ├── MSM (中等)
    │     └── 乳香酸 (中等)
    │
    ├── 5-LOX抑制:
    │     └── 乳香酸 (AKBA) ──→ 白三烯B4 ↓
    │
    ├── COX-2抑制:
    │     ├── 姜黄素
    │     └── Omega-3 (间接)
    │
    ├── 促炎细胞因子下调:
    │     ├── TNF-α ↓: 姜黄素、MSM、乳香
    │     ├── IL-1β ↓: 软骨素、姜黄素、MSM
    │     └── IL-6 ↓: MSM、姜黄素
    │
    └── 消退素/保护素合成 (来自Omega-3):
          Resolvin E1/D1 ──→ 主动消退炎症
```

## 3. 氨基葡萄糖 (Glucosamine)

### 3.1 基本信息

| 属性 | 详情 |
|------|------|
| **化学名称** | 氨基葡萄糖 (2-氨基-2-脱氧-D-葡萄糖) |
| **分子式** | C6H13NO5 |
| **分子量** | 179.17 g/mol |
| **CAS号** | 3416-24-8 (盐酸盐); 21653-40-7 (硫酸盐) |
| **主要形式** | 硫酸氨基葡萄糖 (GA-S)、盐酸氨基葡萄糖 (GA-HCl) |
| **来源** | 海洋甲壳类外壳提取 (虾、蟹) 或微生物发酵 |

### 3.2 作用机制

#### 3.2.1 软骨基质合成

```
氨基葡萄糖 ──→ UDP-GlcNAc (尿苷二磷酸-N-乙酰氨基葡萄糖)
                    │
                    ├──→ 糖胺聚糖 (GAG) 合成
                    │         │
                    │         ├──→ 硫酸软骨素
                    │         └──→ 硫酸角质素
                    │
                    └──→ 透明质酸合成 ──→ 蛋白聚糖单体 ──→ 软骨基质
```

- **蛋白聚糖(Aggrecan)** 是软骨的主要承压成分
- 氨基葡萄糖是糖胺聚糖链合成的限速底物
- 刺激软骨细胞合成II型胶原蛋白

#### 3.2.2 抗炎机制

| 靶点 | 效应 | 结果 |
|------|------|------|
| NF-κB通路 | 抑制p65磷酸化 | TNF-α、IL-1β、IL-6 ↓ |
| COX-2 | 抑制表达 | PGE₂ ↓ |
| MMP-1, MMP-3, MMP-13 | 抑制活性 | 软骨降解 ↓ |
| iNOS | 抑制表达 | NO ↓ (减少软骨细胞毒性) |

#### 3.2.3 软骨保护

- 抑制基质金属蛋白酶 (MMPs) 活性
- 减少软骨细胞凋亡
- 维持软骨下骨代谢平衡

### 3.3 临床证据与争议

#### 3.3.1 支持证据

| 研究 | 设计 | 结果 |
|------|------|------|
| Reginster et al., 2001 | RCT, n=212, 3年 | 硫酸氨基葡萄糖显著减缓关节间隙收窄 |
| Pavelka et al., 2002 | RCT, n=202, 3年 | 膝关节OA疼痛减轻，功能改善 |
| Poolsup et al., 2005 | Meta分析, 13项研究 | 疼痛VAS改善，WOMAC功能改善 |
| OVATION Study, 2017 | 长期追踪 | 症状改善，生活质量提高 |

#### 3.3.2 争议性证据 ⚠️ TODO

| 研究/评论 | 结论 | 争议焦点 |
|-----------|------|---------|
| LATARLY / GUIDE studies (2020-2022) | 未能证实氨基葡萄糖优效于安慰剂 | ⚠️ 多中心设计，规模较大，但方法和剂量受到质疑 |
| Persson et al., 2023 | 阴性结果 | ⚠️ 使用的是盐酸盐而非硫酸盐 |
| ESCEO工作组 2019 | 硫酸氨基葡萄糖仍推荐 | 硫酸盐与盐酸盐可能存在药代动力学差异 |
| Honvo et al., 2019 | 硫酸盐安全性优于盐酸盐 | 吸收率和生物利用度差异 |

**⚠️ TODO 关键争议**:
- 硫酸氨基葡萄糖 vs 盐酸氨基葡萄糖：哪种形式更有效？
- 最新大型RCT (2020-2024) 结果不一致，需要系统综述更新
- 剂量是否足够（1500mg是否最优）？
- 与软骨素联合使用的协同效应是否确实存在？

#### 3.3.3 机制解释假说

```
为何临床试验结果不一致？

假说1: 剂型差异
    硫酸盐 vs 盐酸盐 ──→ 药代动力学差异 ──→ 疗效差异

假说2: 安慰剂效应
    OA疼痛主观性强 ──→ 安慰剂效应高达30-40% ──→ 掩盖真实疗效

假说3: 患者选择
    早期OA vs 晚期OA ──→ 反应不同

假说4: 剂量/生物利用度
    口服吸收率约20-30% ──→ 可能需要调整剂量策略
```

### 3.4 推荐剂量与用法

| 形式 | 推荐剂量 | 分服 | 说明 |
|------|---------|------|------|
| 硫酸氨基葡萄糖 | 1500mg/天 | 1-3次 | 硫酸盐形式证据更多 |
| 盐酸氨基葡萄糖 | 1500mg/天 | 1-3次 | 发酵来源适合甲壳类过敏者 |

**服用建议**:
- 至少持续8-12周才能评估效果
- 餐前30分钟服用可能改善吸收
- 与硫酸软骨素联合可能增强效果

### 3.5 安全性

| 项目 | 评估 |
|------|------|
| **总体安全性** | 良好 |
| **常见副作用** | 轻微胃肠道不适 (恶心、腹胀、腹泻)，发生率<5% |
| **过敏风险** | 甲壳类来源可能引起过敏，发酵来源可避免 |
| **血糖影响** | 理论上有轻微升血糖可能，糖尿病患者监测血糖 |
| **药物相互作用** | 可能增强抗凝药效果 (如华法林)，合用时监测INR |
| **特殊人群** | 孕妇/哺乳期安全性数据不足，谨慎使用 |

## 4. 硫酸软骨素 (Chondroitin Sulfate)

### 4.1 基本信息

| 属性 | 详情 |
|------|------|
| **化学名称** | 硫酸软骨素 (Chondroitin Sulfate) |
| **主要类型** | CS-A (4-硫酸软骨素)、CS-C (6-硫酸软骨素) |
| **分子量** | 10-50 kDa (平均约 20-30 kDa) |
| **来源** | 牛软骨、猪软骨、鲨鱼软骨 |
| **结构** | 线性糖胺聚糖，由N-乙酰半乳糖胺和葡萄糖醛酸交替组成 |

### 4.2 作用机制

#### 4.2.1 软骨结构支持

```
硫酸软骨素 ──→ 蛋白聚糖 (Aggrecan) 组装
                    │
                    ├──→ 透明质酸结合域 ──→ 聚集形成
                    │
                    └──→ 胶原纤维网络 ──→ 软骨基质结构完整
```

- **亲水性**: 吸收水分至软骨基质，维持弹性和抗压性
- **排斥作用**: 蛋白聚糖的负电荷排斥胶原纤维，抵抗压缩载荷
- 软骨素含量降低与OA严重程度正相关

#### 4.2.2 酶抑制作用

| 靶酶 | 效应 | 结果 |
|------|------|------|
| MMP-1 (胶原酶1) | 抑制 | I型/II型胶原降解 ↓ |
| MMP-3 (基质溶解酶) | 抑制 | 蛋白聚糖降解 ↓ |
| MMP-13 (胶原酶3) | 抑制 | II型胶原降解 ↓ |
| Aggrecanase | 抑制 | 蛋白聚糖核心蛋白降解 ↓ |

#### 4.2.3 抗炎作用

| 靶点 | 效应 |
|------|------|
| IL-1β | 抑制其对软骨细胞的分解代谢作用 |
| TNF-α | 下调滑膜炎症 |
| PGE₂ | 减少合成 |
| NF-κB | 部分抑制 |

### 4.3 临床证据

#### 4.3.1 主要研究

| 研究 | 设计 | 结果 |
|------|------|------|
| L Santos et al., 2011 | Meta分析, n=3843 | 疼痛显著减轻，WOMAC改善 |
| Mazieres et al., 2001 | RCT, n=307 | 功能改善，疼痛减轻 |
| Kahan et al., 2009 | RCT, 2年 | 关节间隙收窄减缓 |

#### 4.3.2 与氨基葡萄糖联合

**联合疗法的理论基础**:

```
氨基葡萄糖 ──→ 合成代谢 (GAG合成 ↑)
        +
硫酸软骨素 ──→ 抑制分解代谢 (MMP活性 ↓)
        ↓
理论上协同增效
```

| 研究 | 联合方案 | 结果 |
|------|---------|------|
| Zhu et al., 2021 | GA 1500mg + CS 1200mg | 联合vs单独：效果量约1.5倍 |
| McAlindon et al., 2017 | 复方制剂 | WOMAC改善显著 |
| **⚠️ TODO** | 2024-2026年验证 | 需要最新大型RCT |

**⚠️ TODO**: 联合疗法vs单独使用的头对头证据仍然有限

### 4.4 推荐剂量与用法

| 应用场景 | 推荐剂量 | 分服 |
|---------|---------|------|
| 膝关节OA单药 | 800-1200mg/天 | 2-3次 |
| 联合氨基葡萄糖 | 800mg + 1500mg/天 | 分次 |
| 手指关节OA | 800mg/天 | 分次 |

**质量注意事项**:
- 分子量范围影响吸收效率
- 低分子量CS (10-15 kDa) 可能比高分子量吸收更好
- 选择含有适当硫酸基团位置的产品

### 4.5 安全性

| 项目 | 评估 |
|------|------|
| **总体安全性** | 良好 |
| **常见副作用** | 轻微胃肠道反应，极少见过敏 |
| **抗凝作用** | 理论上可能增强抗凝，证据有限 |
| **质量关注** | 某些来源（牛）可能涉及朊病毒顾虑，选择可靠品牌 |

## 5. MSM (甲基磺酰甲烷)

### 5.1 基本信息

| 属性 | 详情 |
|------|------|
| **化学名称** | 甲基磺酰甲烷 (Methylsulfonylmethane) |
| **别名** | 二甲基砜、DMSO2 |
| **分子式** | C2H6O2S |
| **分子量** | 94.13 g/mol |
| **CAS号** | 67-71-0 |
| **硫含量** | 约34% |
| **存在形式** | 天然存在于水果、蔬菜、谷物、咖啡中；人体内亦含少量 |

### 5.2 作用机制

#### 5.2.1 硫供体作用 (Sulfur Donation)

```
MSM ──→ 活性硫化合物 ──→ 参与代谢:
                    │
                    ├──→ 甲硫氨酸循环 ──→ 谷胱甘肽(GSH)合成
                    │
                    ├──→ 蛋白聚糖合成 ──→ 软骨基质GAG
                    │
                    └──→ 胶原合成 ──→ II型胶原 ──→ 软骨结构
```

- **硫是软骨基质糖胺聚糖的核心结构成分**
- MSM提供的硫支持蛋白聚糖和胶原蛋白的合成
- 维持软骨基质的完整性和弹性

#### 5.2.2 抗炎机制 (NF-κB抑制)

| 靶点 | 效应 | 结果 |
|------|------|------|
| NF-κB通路 | 抑制p65磷酸化 | 下游炎症基因转录 ↓ |
| TNF-α | 降低表达/释放 | 促炎信号 ↓ |
| IL-1β | 降低表达 | 软骨降解 ↓ |
| IL-6 | 降低表达 | 急性相反应 ↓ |
| COX-2 | 抑制表达 | 前列腺素合成 ↓ |
| iNOS | 抑制表达 | NO过度生成 ↓ |

#### 5.2.3 抗氧化机制

```
MSM ──→ 维持细胞膜完整性 ──→ 减少ROS渗漏
        │
        ├──→ 支持谷胱甘肽(GSH)合成
        │
        ├──→ 提升超氧化物歧化酶(SOD)活性
        │
        └──→ 降低脂质过氧化标志物(MDA)
```

### 5.3 临床证据

| 研究 | 设计 | 剂量 | 结果 |
|------|------|------|------|
| Kim et al., 2006 | 双盲RCT, n=50 | 3g Bid (6g/天) | WOMAC疼痛评分显著改善 |
| Brien et al., 2008 | 系统综述 | 多研究汇总 | 证据支持MSM用于OA疼痛 |
| Usha et al., 2004 | RCT, n=118 | MSM vs MSM+GA | 两者均显示镇痛效果 |
| **⚠️ TODO** | 需要2024-2026验证 | - | 最新RCT尚未纳入 |

**⚠️ TODO**: 6g/天对膝骨关节炎的长期疗效(>1年)需要更多大型多中心RCT验证

### 5.4 肌肉损伤与DOMS恢复

| 研究 | 对象 | 剂量 | 结果 |
|------|------|------|------|
| van der Merwe et al., 2017 | 半程马拉松跑者 | 3g/天 x 21天 | CK↓, 肌肉疼痛↓, 氧化应激↓ |
| Kammerer et al., 2015 | 力量训练男性 | 3g Bid | CK↓, 疼痛↓ |
| **⚠️ TODO** | 需要2024-2026验证 | - | - |

### 5.5 推荐剂量与用法

| 目的 | 日剂量 | 分服 | 疗程 |
|------|--------|------|------|
| **关节健康维护** | 1.5-3g/天 | 1-2次 | 长期 |
| **骨关节炎辅助** | 3-6g/天 | 2-3次 | 3-6个月评估 |
| **运动恢复** | 3g/天 | 1-2次 | 运动前7天+运动后5天 |
| **抗炎需求** | 3-6g/天 | 2-3次 | 个体化 |

**分服建议**:
- 分次服用优于单次大剂量
- 建议与餐同服减少GI刺激
- 从低剂量(1g/天)开始，逐步增加

### 5.6 安全性

| 项目 | 评估 |
|------|------|
| **FDA GRAS** | 通过 (一般公认为安全) |
| **EFSA** | 批准使用 |
| **LD50 (大鼠)** | >20g/kg |
| **人体安全上限** | 6g/天 (临床试验常用) |
| **常见副作用** | 胃肠道不适(恶心、腹泻) 1-5%，头痛<1% |
| **禁忌** | 对硫化物过敏者，孕妇/哺乳期(缺乏数据) |
| **⚠️注意** | 长期高剂量(>10g/天)安全性数据不足 |

## 6. 胶原蛋白 (Collagen)

### 6.1 基本信息与类型

关节健康涉及三种主要胶原蛋白类型：

| 类型 | 主要分布 | 关节相关功能 | 口服形式 |
|------|---------|-------------|---------|
| **I型** | 皮肤、肌腱、韧带、骨骼 | 肌腱/韧带结构支持 | 水解胶原蛋白肽 |
| **II型** | 关节软骨 | 软骨基质主要成分 | UC-II (未变性) / 水解II型胶原 |
| **III型** | 皮肤、血管、肌肉 | 与I型协同，维持结构 | 水解胶原蛋白肽 (常与I型共存) |

### 6.2 II型胶原蛋白 (关节软骨)

#### 6.2.1 UC-II (未变性II型胶原蛋白)

**作用机制: 口服免疫耐受 (Oral Tolerance)**

```
UC-II (未变性三重螺旋结构)
        │
        ├──→ 肠道派尔集合淋巴结 (Peyer's patches)
        │         │
        │         └──→ 识别为"自我"抗原
        │
        ├──→ 诱导Treg细胞 (调节性T细胞)
        │         │
        │         └──→ 分泌抗炎因子: IL-10, TGF-β
        │
        └──→ Treg细胞迁移至关节
                  │
                  └──→ 抑制自身免疫反应 ──→ 减少软骨降解
```

**关键点**: UC-II必须保留完整的三重螺旋结构才能诱导免疫耐受；水解胶原蛋白无法产生此效应

#### 6.2.2 水解II型胶原蛋白 (Collagen Peptides)

| 特点 | 说明 |
|------|------|
| **分子量** | 2-10 kDa (小分子肽段) |
| **吸收率** | >90% (远高于完整胶原) |
| **理论作用** | 可能作为软骨合成底物 |
| **证据** | ⚠️ TODO 证据不如UC-II充分 |

### 6.3 I/III型胶原蛋白 (肌腱/韧带)

| 靶组织 | I型:III型比例 | 功能 |
|--------|-------------|------|
| 肌腱 | ~95%:5% | 抗拉强度，传递肌肉力量至骨骼 |
| 韧带 | ~90%:10% | 关节稳定性，限制过度活动 |
| 皮肤 | ~80%:20% | 结构完整性 |

**水解胶原蛋白肽**:
- 口服后以游离氨基酸和小肽形式吸收
- 可能为胶原蛋白合成提供底物

## 7. 透明质酸 (Hyaluronic Acid)

### 7.1 基本信息

| 属性 | 详情 |
|------|------|
| **化学名称** | 透明质酸 (Hyaluronic Acid, HA) |
| **别名** | 玻尿酸、玻璃酸 |
| **分子量** | 500 kDa - 6,000 kDa (高分子量) / 50-500 kDa (低分子量口服) |
| **化学分类** | 非硫酸糖胺聚糖 (GAG) |
| **来源** | 微生物发酵 (非动物来源推荐) 或鸡冠/牛眼玻璃体提取 |
| **CAS号** | 9004-61-9 |

### 7.2 作用机制

#### 7.2.1 关节润滑

```
透明质酸 ──→ 滑液 (Synovial Fluid) 主要成分
        │
        ├──→ 润滑关节面 (边界润滑)
        │     分子量越高，黏弹性越好
        │
        ├──→ 减震 (Shock absorption)
        │     承受压力时黏度增加，分散压力
        │
        └──→ 营养扩散媒介
              软骨无血管，营养通过滑液扩散
```

#### 7.2.2 抗炎与软骨保护

| 靶点 | 效应 |
|------|------|
| CD44受体 | HA与软骨细胞表面CD44结合，调节细胞功能 |
| 自由基 | 抑制炎症介质释放和自由基形成 |
| 软骨细胞 | 促进蛋白聚糖和II型胶原合成 |
| 滑膜 | 减少滑膜炎症 |

#### 7.2.3 口服 vs 注射

| 途径 | 特点 | 证据 |
|------|------|------|
| **口服** | 分子量通常50-500 kDa，吸收后分布至关节 | 中等证据 ⚠️ TODO |
| **注射 (HA)** | 高分子量，20-25mg/次，直接补充滑液 | 较强证据，ESCEO指南推荐 |

### 7.3 临床证据

| 研究 | 设计 | 剂量 | 结果 |
|------|------|------|------|
| Tashiro et al., 2012 | RCT | 200mg/天 (低分子量) | 8周膝关节OA症状显著改善 |
| Mandelhaus et al., 2014 | RCT | HA + 胶原 + 软骨素 | 关节功能和疼痛改善 |
| Oe et al., 2016 | 长期研究 | 口服HA | 皮肤水分和关节舒适度改善 |
| **⚠️ TODO** | 需要更多2024-2026年高质量RCT | - | - |

### 7.4 推荐剂量

| 类型 | 推荐剂量 |
|------|---------|
| 低分子量HA (口服) | 80-200mg/天 |
| 高分子量HA (口服) | 100mg+/天 |
| 注射 (膝关节) | 20-25mg/次，每周1次，共3-5周 |

**分子量选择**:
- 高分子量HA：黏弹性更好，但吸收可能较差
- 低分子量HA：口服吸收更好，但润滑性能可能较低

| 项目 | 评估 |
|------|------|
| **总体安全性** | 良好 |
| **口服副作用** | 极少见，轻微消化不适 |
| **注射副作用** | 注射部位短暂不适，罕见过敏反应 |
| **过敏风险** | 微生物发酵来源可避免动物蛋白过敏 |

## 8. 乳香属 (Boswellia serrata)

### 8.1 基本信息

| 属性 | 详情 |
|------|------|
| **植物名称** | 乳香树 (Boswellia serrata) |
| **别称** | 印度乳香、齿叶乳香 |
| **产地** | 印度、索马里、埃塞俄比亚等地 |
| **药用部位** | 树脂 (Oleo-gum-resin) |
| **活性成分** | 乳香酸 (Boswellic acids)，其中AKBA (Acetyl-11-keto-β-boswellic acid) 最主要 |
| **CAS号** | 115-53-7 (AKBA) |

### 8.2 作用机制

#### 8.2.1 5-LOX抑制 (核心机制)

```
乳香酸 (AKBA) ──→ 抑制5-脂氧合酶 (5-LOX) 活性
                    │
                    └──→ 白三烯B4 (LTB4) 合成 ↓
                              │
                              ├──→ 中性粒细胞趋化 ↓
                              ├──→ 炎症细胞浸润 ↓
                              └──→ 关节炎症 ↓
```

**为什么5-LOX抑制重要？**
- LTB4是参与OA炎症过程的关键介质
- 5-LOX通路与COX通路不同：抑制COX不影响白三烯
- 乳香与NSAIDs作用通路不同，理论上可联合使用

#### 8.2.2 多靶点抗炎

| 靶点 | 效应 |
|------|------|
| 5-LOX | **强抑制** (AKBA为主要抑制剂) |
| NF-κB | 抑制活化 |
| TNF-α | 降低表达 |
| IL-1β | 降低表达 |
| MMP-3, MMP-13 | 抑制活性 |
| 人类弹性蛋白酶 | 抑制 |

#### 8.2.3 软骨保护

```
AKBA ──→ 抑制软骨细胞凋亡
        │
        ├──→ 抑制MMP诱导的软骨降解
        │
        └──→ 维持软骨基质完整性
```

### 8.3 临床证据

| 研究 | 设计 | 成分/剂量 | 结果 |
|------|------|---------|------|
| Sengupta et al., 2008 | RCT, n=75 | 5-LOX抑制剂配方 | WOMAC疼痛和功能显著改善 |
| Kimmatkar et al., 2003 | 双盲RCT | Boswellia 333mg×3/天 | 膝关节OA疼痛显著减轻 |
| Majeed et al., 2019 | RCT | Boswellia + 姜黄素联合 | 关节功能改善优于单独使用 ⚠️ TODO |
| **⚠️ TODO** | 需要2024-2026年更多RCT | - | - |

### 8.4 推荐剂量与用法

| 成分 | 推荐剂量 | AKBA含量要求 |
|------|---------|-------------|
| Boswellia serrata提取物 | 300-1200mg/天 | 不少于30%乳香酸 |
| AKBA标准化提取物 | 100-300mg/天 | 至少30% AKBA |
| 分服 | 2-3次/天 | 与餐同服 |

**质量要点**:
- 选择含有≥30%乳香酸的产品
- 含有AKBA (Acetyl-11-keto-β-boswellic acid) 标准化
- 某些配方含有吸收促进剂 (如piperoine)

### 8.5 安全性

| 项目 | 评估 |
|------|------|
| **总体安全性** | 良好 |
| **常见副作用** | 轻微胃肠道不适，偶有恶心 |
| **药物相互作用** | 可能增强抗凝药效果 (理论风险) |
| **特殊人群** | 孕妇/哺乳期安全性数据不足 |

## 10. 实际建议

### 10.1 个体化选择指南

| 情况 | 推荐方案 |
|------|---------|
| **年轻运动员，预防性** | 氨基葡萄糖 + Omega-3 + 维生素D |
| **已有轻度关节不适** | 基础组合 + MSM |
| **中度OA症状** | 强化组合全部成分 |
| **高强度力量训练** | 基础组合 + 水解胶原蛋白(I/III型) |
| **肌腱/韧带问题倾向** | UC-II + 水解胶原蛋白 + MSM |
| **炎症为主** | Omega-3 + 乳香 + MSM |

### 10.2 服用时间建议

| 成分 | 最佳服用时间 | 理由 |
|------|------------|------|
| 氨基葡萄糖 | 餐前30分钟 | 空腹吸收可能更好 |
| 硫酸软骨素 | 与氨基葡萄糖同服 | 协同吸收 |
| MSM | 与食物同服 | 减少GI不适 |
| UC-II | **空腹** (早晨或睡前) | 胃酸可能破坏结构 |
| 水解胶原蛋白 | 任意时间 | 无特殊要求 |
| 透明质酸 | 与含脂肪膳食 | 脂溶性，吸收促进 |
| 乳香提取物 | 与食物同服 | 减少GI刺激 |
| Omega-3 | 与含脂肪膳食 | 脂肪促进吸收 |
| 维生素D3/K2 | 与含脂肪膳食 | 脂溶性 |

### 10.3 疗程评估

| 阶段 | 时间 | 评估内容 |
|------|------|---------|
| **初始评估** | 第0周 | 记录基线症状 (疼痛VAS, WOMAC) |
| **短期评估** | 第4周 | 初步观察是否有改善趋势 |
| **中期评估** | 第8周 | 判断是否继续或调整 |
| **正式评估** | 第12周 | WOMAC对比基线，决定长期方案 |
| **长期追踪** | 每6个月 | 评估维持效果和安全性 |

**⚠️ 重要提示**:
- 关节健康补剂通常需要4-12周才能显现效果
- 不要期望像NSAIDs那样快速止痛
- 早期OA或预防性使用效果可能更明显
- 晚期OA可能需要结合其他医疗手段

### 10.4 质量选择建议

| 成分 | 质量指标 |
|------|---------|
| **氨基葡萄糖** | 硫酸盐形式，USP认证，选择发酵来源避免甲壳类过敏 |
| **硫酸软骨素** | USP认证，测定硫酸根含量，分子量分布验证 |
| **MSM** | OptiMSM®品牌原料，USP认证，高纯度(>99.9%) |
| **UC-II** | 专利原料(如UC-II®)，保留未变性结构，冷链保存 |
| **水解胶原蛋白** | 分子量2-10 kDa，I/III型比例，认证无重金属 |
| **透明质酸** | 微生物发酵来源，分子量标注明确 |
| **乳香提取物** | 含≥30%乳香酸，标准化AKBA含量 |
| **Omega-3** | EPA+DHA含量明确，IFOS/GOED认证，氧化值达标 |

### 10.5 安全性注意事项

| 情况 | 建议 |
|------|------|
| **抗凝治疗 (华法林等)** | 氨基葡萄糖、软骨素可能增强抗凝，密切监测INR |
| **糖尿病** | 氨基葡萄糖可能轻微影响血糖，监测血糖水平 |
| **甲壳类过敏** | 选择发酵来源氨基葡萄糖，避免动物来源透明质酸 |
| **手术前2周** | 停用所有抗炎类补剂 (Omega-3, 姜黄素, 乳香) |
| **孕妇/哺乳期** | 缺乏安全性数据，避免使用 |
| **慢性疾病** | 咨询医生后再使用 |
