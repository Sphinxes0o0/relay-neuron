# 谷氨酰胺与免疫功能——获益分析

**研究日期**: 2026-05-26
**研究类型**: 运动营养补剂 / 免疫调节

---

## 摘要

谷氨酰胺（Glutamine）是人体内含量最丰富的游离氨基酸（占血浆游离氨基酸池的~20%，占肌肉游离氨基酸池的~60%），被归类为"条件必需氨基酸"——在严重应激、创伤、烧伤、大强度训练等分解代谢状态下，内源性合成不足以满足需求。谷氨酰胺是免疫细胞（淋巴细胞、巨噬细胞）和肠道上皮细胞的主要燃料，在维持肠道屏障完整性和免疫功能中发挥关键作用。本文综述谷氨酰胺对免疫功能、肠道健康及运动相关免疫抑制的获益。

---

## 1. 谷氨酰胺的基础生物学

### 1.1 代谢特征

| 参数 | 数值 |
|------|------|
| 血浆浓度 | 500-750 μmol/L |
| 肌肉谷氨酰胺池 | ~20 mmol/L（细胞内） |
| 每日周转量 | ~70 g/天（全身） |
| 膳食摄入 | ~5-10 g/天（标准饮食） |
| 内源性合成 | 主要在骨骼肌、肺、肝 |

### 1.2 谷氨酰胺的免疫功能

**免疫细胞燃料**：
> [Cruzat V, et al. *Nutrients*, 2018](https://doi.org/10.3390/nu10111564) 综述确定谷氨酰胺是快速分裂细胞（包括淋巴细胞、巨噬细胞和肠上皮细胞）的首选氧化燃料。

```
谷氨酰胺 → 免疫细胞代谢：
- 淋巴细胞：谷氨酰胺氧化速率 ≈ 葡萄糖
- 巨噬细胞：谷氨酰胺提供NADPH（用于ROS生成）
- 中性粒细胞：谷氨酰胺支持吞噬功能
- 肠上皮细胞：谷氨酰胺是主要能量底物
```

### 1.3 运动后的谷氨酰胺消耗

长时间高强度运动可导致血浆谷氨酰胺显著下降：

> [Castell LM, Newsholme EA. *Nutrition*, 1997](https://doi.org/10.1016/S0899-9007(97)00121-4)：马拉松跑后血浆谷氨酰胺降低约20-25%，持续数小时至数天。这种下降与运动员赛后上呼吸道感染（URTI）风险升高呈时间相关性。

## 2. 免疫功能获益

### 2.1 运动后免疫抑制防护

| 运动类型 | 谷氨酰胺下降幅度 | 感染风险 |
|---------|----------------|---------|
| 马拉松 | ↓ 20-25% | ↑ URTI 2-6倍 |
| 铁人三项 | ↓ 30-40% | ↑ URTI 3-5倍 |
| 高强度间歇训练（连续日） | ↓ 10-20% | ↑ 轻度 |
| 中等强度训练 | 无显著变化 | 无显著变化 |

> [Castell LM, et al. *Eur J Appl Physiol*, 1996](https://doi.org/10.1007/BF00376508) RCT：150名马拉松选手和赛跑选手随机接受谷氨酰胺（5 g）或安慰剂补充，7天内谷氨酰胺组自我报告感染率显著降低（谷氨酰胺组19% vs 安慰剂组51%，p<0.05）。

### 2.2 免疫细胞功能支持

谷氨酰胺在免疫激活期间的作用：

| 免疫细胞类型 | 谷氨酰胺功能 |
|-------------|-------------|
| 淋巴细胞 | 支持增殖和IL-2产生 |
| 巨噬细胞 | 提供NADPH用于吞噬作用ROI/RNS生成 |
| 中性粒细胞 | 支持趋化性和杀菌活性 |
| NK细胞 | 维持细胞毒性 |
| 树突状细胞 | 成熟和抗原呈递支持 |

### 2.3 肠道屏障功能（肠漏防护）

> [Wang B, et al. *Amino Acids*, 2015](https://doi.org/10.1007/s00726-014-1773-4)：谷氨酰胺是肠道上皮细胞的首要能量来源，维持紧密连接蛋白（Occludin、Claudin-1、ZO-1）的表达，防止"肠漏"（肠道通透性增加）。

**运动与肠漏**：
- 长时间耐力运动（>2小时）可使肠道通透性暂时增加（肠漏）
- 肠漏可能导致脂多糖（LPS/内毒素）进入血液循环，触发低度炎症
- 谷氨酰胺维持肠道屏障可能缓解运动诱导的肠漏

> [Pugh JN, et al. *Eur J Appl Physiol*, 2017](https://doi.org/10.1007/s00421-017-3613-1)：有初步证据支持谷氨酰胺减少长时间运动后的肠道通透性增加，但高质量RCT有限。

## 3. 氧化应激与谷胱甘肽

### 3.1 谷胱甘肽（GSH）合成

谷氨酰胺是谷胱甘肽合成的前体之一：

```
谷氨酰胺 → 谷氨酸（谷氨酰胺酶）
谷氨酸 + 半胱氨酸 + 甘氨酸 → 谷胱甘肽（GSH）
```

谷胱甘肽是体内最主要的**内源性抗氧化剂**，在免疫细胞和肝脏中的浓度极高。

> [Cruzat V, et al. *Nutrients*, 2018](https://doi.org/10.3390/nu10111564)：在分解代谢和氧化应激状态下，谷氨酰胺可支持谷胱甘肽合成，增强抗氧化防御能力。

### 3.2 热休克蛋白（HSP）

谷氨酰胺可诱导热休克蛋白（尤其是HSP70）表达：

> [Wischmeyer PE, et al. *Crit Care Med*, 2001](https://doi.org/10.1097/00003246-200111000-00004)：谷氨酰胺通过HSP70通路保护细胞免受氧化应激和炎症损伤。

## 4. 适用人群与场景

### 4.1 强烈推荐的场景

| 场景 | 推荐理由 |
|------|---------|
| 耐力运动员（赛后7天内） | 预防URTI（免疫开窗期） |
| 大强度训练周期（>15小时/周） | 免疫支持 |
| 烧伤/严重创伤 | 临床标准治疗（EN/PN中常规添加） |
| 大型手术后恢复 | 加速康复、减少感染 |
| ICU重症患者 | 减少感染并发症 |
| 炎症性肠病（IBD） | 肠道屏障支持 |

### 4.2 证据不一致的场景

| 场景 | 证据状况 |
|------|---------|
| 中等强度训练者（<8小时/周） | 无明显获益 |
| 一般健康人群 | 无获益（内源性合成充足） |
| 增肌人群 | 无证据支持额外的合成代谢效果 |

---

## 5. 补充方案

| 参数 | 推荐方案 |
|------|---------|
| 日剂量（运动人群） | 5-10 g（分次） |
| 日剂量（临床） | 10-30 g（分次或EN/PN） |
| 最佳时机 | 运动后即刻或睡前 |
| 补充形式 | L-谷氨酰胺粉末（游离氨基酸） |
| 与蛋白质的关系 | 乳清蛋白含谷氨酰胺约8-10 g/100 g |

---

## 6. 结论

1. **谷氨酰胺是免疫细胞（淋巴细胞、巨噬细胞）和肠道上皮细胞的关键燃料**
2. **主要获益人群：长时间耐力运动员（马拉松、铁人三项）、重症/创伤患者、大手术后恢复**
3. **马拉松赛后补充可显著减少感染率（研究显示51%→19%，p<0.05）**
4. **通过维持肠道屏障和谷胱甘肽合成，间接支持免疫功能**
5. **对一般健身人群和中等强度训练者，内源性合成充足，补充无额外获益**
6. **临床重症中作为标准营养支持成分，证据充分**
