# 肥胖导致内脏损伤的综合综述

## Comprehensive Review: Internal Organ Damage Caused by Obesity

**综述日期**：2026年4月
**文献来源**：22篇近三年（2021-2026）权威文献的系统梳理

---

## 摘要

肥胖已成为全球性公共卫生危机，目前西方国家约25%的成年人患有肥胖。肥胖不仅是体脂的简单堆积，更是一种累及全身多系统的代谢性、炎症性和内分泌性疾病。本综述基于Nature Reviews Cardiology（2023）、Nature Medicine（2025）、柳叶刀系列子刊（2023-2025）、Nature Mental Health（2025）等权威期刊发表的22篇高质量研究，系统阐述肥胖通过内脏脂肪堆积对心血管系统、肝脏系统、中枢神经系统、代谢内分泌系统及肾脏系统造成损伤的核心机制、流行病学证据及治疗进展。

## 1. 引言：重新认识内脏脂肪

### 1.1 内脏脂肪 vs 皮下脂肪

人体脂肪组织分为皮下脂肪和内脏脂肪（Visceral Adiposity Tissue, VAT）两大类。传统观念将脂肪视为储能仓库，但近十年研究已明确：**内脏脂肪是一个高度活跃的内分泌器官**。

| 脂肪类型 | 位置 | 功能 | 危害 |
|---------|------|------|------|
| **皮下脂肪** | 皮肤下层 | 储能、绝缘、激素储存 | 主要影响外观和活动能力 |
| **内脏脂肪** | 腹腔，包裹脏器 | 支撑保护脏器、活跃分泌脂肪因子 | **直接损害内脏功能** |

### 1.2 内脏脂肪的病理生理特征

当内脏脂肪过量积累时，其功能从"保护性内分泌"转变为"促炎性、促代谢紊乱性"：

```
内脏脂肪功能障碍
        │
        ├─ 抗脂解功能受损 → 游离脂肪酸（FFA）持续溢出
        │
        ├─ 促炎脂肪因子分泌↑ → IL-6、TNF-α、IL-1β等系统性释放
        │
        ├─ 保护性脂肪因子分泌↓ → 脂联素显著降低
        │
        ├─ 胰岛素抵抗加剧 → 细胞对胰岛素敏感性下降
        │
        └─ 氧化应激增加 → ROS累积，细胞损伤
                │
                ↓
        多器官系统性损伤
```

### 1.3 核心共同通路：慢性低度炎症

贯穿肥胖相关所有器官损伤的核心机制是**慢性低度炎症（Chronic Low-Grade Inflammation）**：

```
内脏脂肪过量 → 脂肪组织应激 → 炎症小体（NLRP3等）激活
        │
        ├─→ 全身炎症水平升高
        │
        ├─→ 血管内皮功能障碍
        │
        ├─→ 代谢紊乱（胰岛素抵抗、糖耐量异常）
        │
        └─→ 免疫细胞浸润（巨噬细胞M1型↑）
                │
                ↓
        [心脏] 动脉粥样硬化 → 心血管事件
        [肝脏] 脂肪变性 → MASH → 纤维化 → 肝硬化
        [大脑] 神经炎症 → 血脑屏障受损 → 认知下降
        [胰岛] β细胞功能衰竭 → T2DM
        [肾脏] 肾小球高滤过 → 肾单位损失 → CKD
```

## 2. 心血管系统

### 2.1 流行病学证据

肥胖是心血管疾病（CVD）的**独立风险因素**，其效应在校正传统风险因素（高血压、高血脂、吸烟等）后依然显著。International Journal of Obesity（2023）的研究证明，即使在无传统风险因素的人群中，肥胖仍显著增加心血管事件风险，60岁以下人群关联更强（HR=1.44）。

### 2.2 心脏结构与功能改变

#### 心肌重构

肥胖通过血流动力学高负荷导致心脏结构和功能的适应性改变：

| 改变类型 | 具体表现 | 后果 |
|---------|---------|------|
| 心肌肥厚 | 左心室壁增厚 | 舒张功能障碍 |
| 心室腔改变 | 心室腔相对缩小 | 每搏输出量受限 |
| 充盈压升高 | 左心房扩张 | 房性心律失常易感性↑ |

#### 心力衰竭

Current Heart Failure Reports（2024）指出，肥胖通过三条通路增加心衰风险：

1. **血流动力学通路**：肥胖→血容量增加→心脏前/后负荷↑→心室重构→心衰
2. **代谢通路**：脂肪毒性+线粒体功能障碍→心肌能量危机→心衰
3. **炎症通路**：系统性炎症→心肌炎症→纤维化→心衰

其中**射血分数保留的心衰（HFpEF）**是肥胖相关心衰的主要表型。

### 2.3 房颤

Obesity Reviews（2024）系统阐述了肥胖与心房颤动（AF）的关联：

| 数据 | 数值 |
|------|------|
| 美国成人肥胖患病率 | ~32.2% |
| 肥胖者房颤风险 vs 正常体重 | 增加40-90% |
| 房颤归因于肥胖的比例 | ~20% |
| 减重≥10%的效果 | 房颤负担显著降低 |

**机制**：肥胖→左心房扩张+脂肪浸润→心房纤维化→电生理稳定性下降→房颤

**临床启示**：目标导向减重是管理肥胖相关房颤的基石，每减重10%可显著减少房颤发作频率。

### 2.4 动脉粥样硬化与僵硬

Scientific Reports（2021）通过比较不同体脂参数与动脉僵硬度（baPWV）的关系，发现：

| 体脂指标 | 与baPWV的关联 |
|---------|-------------|
| BMI | 无显著关联 |
| 腰围（WC）| 在多因素校正后不显著 |
| **腰臀比（WHR）** | **显著正相关** |
| **内脏脂肪面积（VFA）** | **显著正相关** |

**结论**：WHR和VFA是反映内脏肥胖与心血管风险的更优指标，而非BMI。

### 2.5 肥胖悖论

Current Heart Failure Reports（2024）对"肥胖悖论"进行了批判性评估——即部分观察性研究发现肥胖心衰患者反而预后较好。**本文立场**：肥胖悖论主要源于队列选择偏倚和残余混杂，临床试验证据并不支持"刻意保持肥胖对心衰有益"。

### 2.6 小结

```
心血管损伤链条：
内脏肥胖 → 血流动力学改变 + 炎症 + 脂毒 → 动脉硬化 + 心肌重构 → 心衰/房颤/心梗/卒中
```

## 3. 肝脏系统

### 3.1 从NAFLD到MASLD：命名革新

非酒精性脂肪性肝病（NAFLD）已正式更名为**代谢相关脂肪性肝病（MASLD）**。Current Cardiology Reports（2024）指出这一更名的核心意义：

| 对比 | NAFLD（旧） | MASLD（新）|
|------|------------|-----------|
| 诊断逻辑 | 排除性诊断 | 包含性诊断（代谢致病）|
| 核心标准 | 排除酒精/其他肝病+脂肪变性 | 脂肪变性+≥1个代谢风险因素 |
| 疾病污名 | "non-alcoholic"暗示与酒精相关 | 直接指向代谢病因 |

**全球患病率**：约**32.4%**的成人患有MASLD，使其成为最常见的慢性肝病。

### 3.2 疾病进展谱系

MASLD代表一组从可逆到不可逆的连续肝脏病理变化：

```
肝细胞脂肪变性（单纯性脂肪肝）
        ↓ [脂肪毒性+炎症]
代谢相关脂肪性肝炎（MASH）
        ↓ [肝星状细胞激活]
肝纤维化（F1→F4）
        ↓ [假小叶形成]
肝硬化
        ↓ [基因突变累积]
肝细胞癌（HCC）
```

### 3.3 内脏脂肪参与NAFLD/MASLD的机制

Journal of Translational Medicine（2025）深入阐述了E3泛素连接酶和去泛素化酶在MASLD进展中的作用，提供了分子机制层面的理解：

**核心机制链条**：

```
内脏脂肪功能障碍
        │
        ├─→ FFA溢出 → 肝脏脂肪酸摄取↑
        │
        ├─→ 脂肪酸从头合成（DNL）↑ → 肝脏脂质蓄积
        │
        ├─→ 脂肪组织抗脂解功能受损
        │
        └─→ 炎症激活（TLR4/NLRP3通路）
                │
                ↓
        MASH进展（炎症→纤维化→硬化）
```

### 3.4 心脏-肝脏轴

Frontiers in Endocrinology（2022）的研究揭示了MASLD与左心室舒张功能障碍的关联：

| 发现 | 描述 |
|------|------|
| MASLD与舒张功能障碍 | 独立相关 |
| MASLD与左心房增大 | 相关 |
| 关联的独立性 | 校正代谢综合征后仍显著 |

**临床意义**：MASLD不只是肝脏疾病，它是**心血管风险的早期标志物**。肝脏和心脏通过炎症、脂毒性和代谢三条通路形成"**，这两篇文献共同揭示了MASLD与心血管损伤的密切关联。

### 3.5 治疗靶点

| 减重幅度 | 对MASLD的影响 |
|---------|-------------|
| ≥5% | 肝脏脂肪变性显著改善 |
| ≥7% | MASH缓解可能 |
| ≥10% | 纤维化逆转可能 |

## 4. 中枢神经系统

### 4.1 核心发现：量化损伤关系

2023年《柳叶刀》子刊发表的一项历时8年、纳入9,189名受试者的里程碑式研究，提供了最具冲击力的量化数据：

> **体脂率每增加9.2%，或内脏脂肪每增加36ml，认知功能加速衰老1年**

这一发现在校正了心血管风险因素、受教育程度和脑血管损伤等混杂变量后依然成立，证明**体脂本身是认知功能下降的独立风险因素**。

### 4.2 脑结构改变

International Journal of Obesity（2024，CHARLS中国队列）通过MRI证实：

| 改变类型 | 具体表现 |
|---------|---------|
| 全脑体积 | 腹部脂肪越多，脑体积越小 |
| 大脑萎缩 | 内脏脂肪增加与更严重的大脑萎缩相关 |
| 海马体 | 与记忆功能直接相关的区域体积缩小 |
| 额叶皮层 | 与执行功能相关的区域变薄 |

### 4.3 长期肥胖轨迹的纵向影响

Nature Mental Health（2025）的纵向研究提供了更直接的因果证据：

```
持续长期肥胖
        │
        ├─→ 神经炎症（IL-6、TNF-α↑）→ 血脑屏障功能障碍
        │
        ├─→ 大脑葡萄糖代谢受损 → 神经元能量危机
        │
        ├─→ 白质完整性↓ → 信息传递效率↓
        │
        └─→ 默认模式网络（DMN）功能异常 → 记忆/学习能力↓
                │
                ↓
        认知功能加速衰退
```

### 4.4 性别差异

《柳叶刀》子刊（2023）研究揭示，腹部内脏脂肪增加导致的大脑萎缩在**女性**身上更为明显，可能与女性在绝经后更易在腹部优先堆积脂肪有关。

### 4.5 机制：炎症介质的神经毒性

```
内脏脂肪（内分泌器官）
        │
        ↓
释放炎症介质（IL-6、TNF-α、MCP-1）
        │
        ↓
穿越血脑屏障
        │
        ↓
神经炎症 → 突触可塑性下降 → 神经细胞损伤
        │
        ↓
记忆/学习能力受损
```

### 4.6 小结

```
中枢神经损伤链条：
内脏肥胖 → 脂肪因子/炎症介质↑ → 神经炎症 → 脑结构萎缩 → 认知功能下降/痴呆风险↑
```

## 5. 代谢与内分泌系统

### 5.1 内脏脂肪与胰岛素抵抗

Nature Medicine（2025）系统综述显示，GLP-1受体激动剂和GIP/GLP-1双受体激动剂（如Tirzepatide）是最有效的减重药物：

| 药物 | 12个月减重效果 | 心血管获益 |
|------|--------------|----------|
| Semaglutide（GLP-1）| ~10-15% | MACE显著降低 |
| Tirzepatide（GIP/GLP-1）| ~15-20% | 心衰住院↓ |
| Orlistat | ~5-10% | 证据有限 |

### 5.2 内脏脂肪指数（VAI）与T2DM

Scientific Reports（2024）基于NHANES 2007-2018数据（n=11,214）发现：

> **VAI每增加1个单位，T2DM风险增加43%（OR=1.43）**

关键发现：**即使BMI正常，高VAI仍增加T2DM风险**——这解释了"代谢性肥胖"（表面瘦但内脏胖）人群的特殊风险。

### 5.3 新型脂肪因子：RBP4

Endocrine（2024）综述了视黄醇结合蛋白4（RBP4）在T2DM中的作用：

| RBP4的作用 | 结果 |
|-----------|------|
| 诱导骨骼肌胰岛素抵抗 | 葡萄糖摄取↓ |
| 促进肝脏胰岛素抵抗 | 糖异生↑ |
| 损害β细胞功能 | β细胞凋亡↑ |
| 诱导β细胞炎症 | 炎症小体激活 |

RBP4把**维生素A代谢与葡萄糖代谢**联系在一起，是未来治疗的新靶点。

### 5.4 免疫调节机制

日本京都大学等机构（2025）发表在《免疫学杂志》上的研究揭示了更深层的机制：

```
IL-7 → 维持内脏脂肪中调节性T细胞（Treg）存活
        │
        ↓
Treg数量稳定 → 脂肪组织局部抗炎
        │
        ↓
胰岛素敏感性改善 → 血糖降低
```

单次IL-7注射即可使小鼠高血糖状态得到长时间抑制，为T2DM的免疫疗法提供了全新思路。

### 5.5 小结

```
代谢损伤链条：
内脏肥胖 → 脂肪因子失衡 + 炎症 + 脂毒 → 胰岛素抵抗 → β细胞衰竭 → T2DM
```

## 6. 肾脏系统

### 6.1 肥胖与CKD的关联

BMC Nephrology（2022，REGARDS队列，n=27,550，中位随访9.4年）发现：

> **VAI每增加2倍，CKD发病风险增加12%（OR=1.12）**

Nature Reviews Endocrinology（2024）进一步指出肥胖与CKD之间存在**双向关系**：

```
传统理解：肥胖 → CKD

新理解：
肥胖 ←→ CKD
   ↑
   │ 肥胖通过多机制导致CKD（主要）
   └─ CKD通过代谢/内分泌改变促进肥胖（次要）
```

### 6.2 肥胖导致肾损伤的机制

| 机制类型 | 具体过程 |
|---------|---------|
| **血流动力学** | 肾小球高滤过→内压↑→肾小球硬化 |
| **代谢性** | 胰岛素抵抗→肾小管间质炎症 |
| **炎症性** | TNF-α/IL-6↑→肾小管损伤 |
| **脂肪因子失衡** | 脂联素↓（保护↓），瘦素↑（促硬化）|

**关键概念**：肾小球高滤过≠肾功能好，而是肾脏"过度工作"的信号，是损伤的早期表现。

### 6.3 糖尿病肾病（DKD）

DMSO（2023）探讨了不同肥胖模式与DKD的关系：

| 肥胖模式 | 代谢状态 | DKD风险 |
|---------|---------|---------|
| 肥胖 | 代谢异常（MUO）| **最高** |
| 肥胖 | 代谢健康（MHO）| 中等（但非无害）|
| 非肥胖 | 代谢异常 | 升高 |
| 非肥胖 | 代谢健康 | 最低 |

**关键结论**：在T2DM患者中，"代谢健康型肥胖"（MHO）可能并不真正"代谢健康"——T2DM本身已是代谢紊乱的标志。

### 6.4 慢性肾脏病（CKD）分期与治疗

Nature Reviews Disease Primers（2025）提供了CKD的整体框架：

| 分期 | eGFR | 重点治疗 |
|------|------|---------|
| G1-G2 | ≥60 | 控制代谢风险、减重 |
| G3a-G3b | 30-59 | SGLT-2i、GLP-1、血压控制 |
| G4-G5 | <30 | 肾脏替代治疗准备 |

### 6.5 SGLT-2抑制剂的肾脏保护

Scientific Reports（2025）的真实世界研究（n=6,946）证实：

> **SGLT-2i vs RAAS阻断剂：复合MAKEs风险降低41%（HR=0.59）**

SGLT-2i通过多重机制保护肾脏：

```
SGLT-2i
   │
   ├─→ 恢复管球反馈 → 入球小动脉收缩 → 肾小球内压↓
   ├─→ 改善线粒体功能 → 肾小管损伤↓
   ├─→ 抗炎作用 → 肾间质炎症↓
   └─→ 降压/减重 → 肾脏负荷↓
```

### 6.6 小结

```
肾脏损伤链条：
内脏肥胖 → 肾小球高滤过 + 炎症 + 脂毒 → 肾单位损失 → eGFR↓ → CKD/ESRD
```

## 7. 共同病理机制总结

### 7.1 核心共同通路

所有五大器官系统的损伤共享以下三条核心共同通路：

```
                    内脏脂肪过量积累
                          │
            ┌─────────────┼─────────────┐
            ↓             ↓             ↓
       慢性低度炎症    代谢紊乱      脂肪因子失衡
            │             │             │
     IL-6/TNF-α↑    胰岛素抵抗    脂联素↓
     NLRP3激活        糖毒性        瘦素↑
     氧化应激         脂毒性        RBP4↑
            │             │             │
            └─────────────┼─────────────┘
                          │
                   多器官系统性损伤
```

### 7.2 炎症作为核心介质

炎症是连接内脏肥胖与所有器官损伤的**共同语言**：

| 器官 | 炎症介导的损伤 |
|------|--------------|
| 心脏 | 动脉粥样硬化、心肌纤维化、心律失常 |
| 肝脏 | 脂肪变性→MASH→纤维化、肝硬化 |
| 大脑 | 神经炎症、血脑屏障破坏、突触损伤 |
| 胰岛 | β细胞凋亡、胰岛素分泌障碍 |
| 肾脏 | 肾小管间质炎症、肾小球硬化 |

### 7.3 VAI：代谢健康的综合指标

内脏脂肪指数（VAI）整合了腰围、BMI、甘油三酯和HDL信息，是比BMI更敏感的代谢健康指标：

- **VAI翻倍** → CKD风险增加12%，T2DM风险增加43%
- 即使BMI正常，高VAI仍意味着代谢风险升高

## 8. 治疗进展

### 8.1 生活方式干预

| 干预 | 效果 |
|------|------|
| 减重≥5% | 肝脏脂肪变性改善、代谢指标改善 |
| 减重≥7% | MASH可能缓解 |
| 减重≥10% | 纤维化可能逆转；房颤负担显著降低 |
| 有氧运动 | 内脏脂肪减少（独立于体重）|

### 8.2 药物治疗

| 药物类别 | 代表药物 | 核心获益 |
|---------|---------|---------|
| GLP-1激动剂 | Semaglutide | 减重+心血管保护 |
| GIP/GLP-1双靶点 | Tirzepatide | 减重更强（15-20%）|
| SGLT-2抑制剂 | Empagliflozin | 肾脏保护（降低41% MAKEs）|
| 口服降糖药 | 二甲双胍 | 代谢改善（中等减重）|

### 8.3 减重手术

减重手术（如胃旁路术、袖状胃切除术）对各器官的保护效应：

- **心脏**：心衰症状改善、左心室重构逆转
- **肝脏**：MASLD/MASH显著改善，部分纤维化逆转
- **大脑**：神经可塑性改善（BMI正常后视觉皮层可塑性恢复）
- **代谢**：T2DM长期缓解率可达50-70%
- **肾脏**：蛋白尿减少、eGFR改善

## 9. 临床启示

### 9.1 从BMI到腰围/VAI的转变

| 指标 | 局限性 | 优势 |
|------|--------|------|
| BMI | 无法区分脂肪/肌肉；无法反映脂肪分布 | 简单、标准化 |
| 腰围 | 无法区分皮下/内脏脂肪 | 简单、反映腹部肥胖 |
| **VAI** | 需要血脂数据 | **综合代谢+脂肪分布** |
| **WHR** | 需要臀围数据 | **反映脂肪分布优于BMI** |

### 9.2 多学科管理

肥胖相关多器官损伤需要**多学科协作**：

```
患者
  │
  ├─ 心血管内科：血压/血脂/心衰/房颤管理
  ├─ 内分泌科：血糖/减重药物/代谢管理
  ├─ 消化/肝病科：MASLD监测和管理
  ├─ 肾脏内科：CKD风险评估和SGLT-2i使用
  ├─ 神经内科：认知功能评估（高风险患者）
  └─ 营养/运动科：生活方式干预的核心执行者
```

### 9.3 早期筛查建议

| 高危人群 | 筛查项目 |
|---------|---------|
| 所有肥胖患者 | 血糖/HbA1c、血压、血脂、肝功能、肾功能 |
| 肥胖+代谢综合征 | + 肝脏超声（脂肪肝筛查）、心电图 |
| 肥胖+静息心悸/晕厥 | + 动态心电图（房颤筛查）|
| 肥胖+T2DM | + 尿白蛋白/肌酐比（DKD筛查）、颈动脉超声 |
| 中老年肥胖 | + 认知功能评估 |

## 10. 结论

本综述系统梳理了近三年（2021-2026）22篇高质量文献，核心结论如下：

1. **肥胖是一种 multi-organ disease**：其损害范围涵盖心血管、肝脏、中枢神经、代谢内分泌和肾脏五大系统

2. **内脏脂肪是核心**：相比皮下脂肪，内脏脂肪对健康的危害更为直接和系统，其内分泌和炎症效应是器官损伤的核心驱动

3. **慢性低度炎症是共同通路**：TNF-α、IL-6等炎症介质贯穿所有器官损伤，理解炎症机制是开发通用治疗策略的关键

4. **量化风险已有据可查**：体脂率每增9.2%=认知衰老1年；VAI每增1单位=T2DM风险增43%；VAI每翻倍=CKD风险增12%

5. **减重和代谢改善可以逆转损伤**：减重≥10%可改善房颤负担、MASH缓解、纤维化逆转——干预永远不嫌早

6. **药物治疗进入新时代**：GLP-1激动剂和SGLT-2抑制剂等心肾代谢药物的涌现，为打破"肥胖→器官损伤"的恶性循环提供了有力武器

鉴于全球肥胖率的持续上升，由肥胖驱动的多器官疾病负担将在未来数十年持续增长。早期识别内脏肥胖、及时干预代谢紊乱，是降低这一公共卫生危机的关键。
